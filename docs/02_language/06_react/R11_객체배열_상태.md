## 1️⃣ 불변성 = 원본 보존 복사 — 비유로 시작하기 [↑](#toc)

사진 편집을 생각해봅니다.

전문 사진작가는 **원본 파일을 절대 수정하지 않습니다.** 대신 원본을 복사한 뒤, 복사본을 수정합니다. 원본은 언제나 안전하게 보존됩니다.

```
원본 사진 (photo_original.jpg)
    ↓ 복사
복사본 (photo_edit.jpg) → 밝기 조절, 필터 적용...
```

React의 **불변성(Immutability)** 도 같은 원칙입니다.

- **원본 상태(state)를 직접 수정하지 않습니다**
- 대신 변경된 **새 객체/배열**을 만들어 `setState`에 전달합니다
- React는 이전 상태와 새 상태를 비교해 화면을 다시 그립니다

이 원칙을 어기면 React가 변화를 감지하지 못합니다. 화면이 바뀌지 않는 **침묵의 버그**가 생깁니다.

<a id="2"></a>
## 2️⃣ 왜 직접 수정하면 안 되는가? [↑](#toc)

### 문제: 직접 수정 시 화면이 바뀌지 않음

```jsx
import { useState } from "react";

function BuggyProfile() {
  const [user, setUser] = useState({ name: "김철수", age: 25 });

  function handleChangeName() {
    // ❌ 절대 이렇게 하지 마세요!
    user.name = "이영희"; // 원본 객체를 직접 수정
    setUser(user);         // 같은 객체를 다시 전달
    console.log(user.name); // "이영희"로 바뀌었지만...
    // 출력: "이영희" (값은 바뀌었지만 화면은 안 바뀜!)
  }

  return (
    <div className="p-4">
      <p>이름: {user.name}</p> {/* 여전히 "김철수"로 표시됩니다 */}
      <button onClick={handleChangeName} className="bg-blue-500 text-white px-4 py-2 rounded">
        이름 바꾸기
      </button>
    </div>
  );
}
// 버튼을 눌러도 "김철수"가 그대로 표시됩니다
```

**왜 바뀌지 않을까요?**

React는 상태가 바뀌었는지 확인할 때 **참조(메모리 주소)를 비교**합니다.

```
이전 user: { name: "김철수" }  ← 메모리 주소: 0x1234
이후 user: { name: "이영희" }  ← 메모리 주소: 0x1234 (같은 주소!)

React: "주소가 같네? 바뀐 게 없구나." → 리렌더 안 함
```

직접 수정하면 내용은 바뀌었지만 메모리 주소가 같아서 React가 변화를 감지하지 못합니다.

> **React는 '같은 객체네? 그럼 바뀐 게 없구나' 하고 리렌더하지 않습니다.**

```jsx
// ✅ 올바른 방법: 새 객체를 만들어서 전달
function correctUpdate() {
  const newUser = { ...user, name: "이영희" }; // 새 객체 생성!
  setUser(newUser);
  // 새 메모리 주소 → React가 변화를 감지 → 리렌더 실행
}
```

## 3️⃣ 객체 상태 업데이트 — 스프레드 연산자 [↑](#toc)

### 스프레드 연산자로 새 객체 만들기

```jsx
import { useState } from "react";

function UserProfile() {
  const [user, setUser] = useState({
    name: "김철수",
    age: 25,
    city: "서울",
    job: "개발자",
  });

  function updateName(newName) {
    setUser((prev) => ({
      ...prev,       // name, age, city, job 모두 복사
      name: newName, // name만 덮어씀
    }));
  }

  function updateAge(newAge) {
    setUser((prev) => ({
      ...prev,
      age: newAge,
    }));
  }

  function birthday() {
    setUser((prev) => ({
      ...prev,
      age: prev.age + 1, // 이전 값 기반으로 계산
    }));
  }

  return (
    <div className="p-6 space-y-4">
      <div className="bg-white rounded-xl p-4 shadow space-y-2">
        <p><span className="font-semibold">이름:</span> {user.name}</p>
        <p><span className="font-semibold">나이:</span> {user.age}세</p>
        <p><span className="font-semibold">도시:</span> {user.city}</p>
        <p><span className="font-semibold">직업:</span> {user.job}</p>
      </div>

      <div className="flex flex-wrap gap-2">
        <button
          onClick={() => updateName("이영희")}
          className="bg-blue-500 text-white px-3 py-1.5 rounded text-sm"
        >
          이름 → 이영희
        </button>
        <button
          onClick={() => updateAge(30)}
          className="bg-green-500 text-white px-3 py-1.5 rounded text-sm"
        >
          나이 → 30
        </button>
        <button
          onClick={birthday}
          className="bg-purple-500 text-white px-3 py-1.5 rounded text-sm"
        >
          생일 🎂 (+1)
        </button>
      </div>

      {/* 현재 상태 디버그 뷰 */}
      <pre className="bg-gray-900 text-green-400 rounded p-3 text-xs font-mono">
        {JSON.stringify(user, null, 2)}
      </pre>
    </div>
  );
}
// "이름 → 이영희" 클릭: name만 바뀌고 나머지는 유지됩니다
// "생일 🎂" 클릭: age가 1씩 올라갑니다
```

### 스프레드의 작동 원리 한눈에

```js
const user = { name: "김철수", age: 25, city: "서울" };

// ...user 는 모든 키-값을 펼칩니다
const newUser = { ...user, name: "이영희" };
// 결과: { name: "이영희", age: 25, city: "서울" }
//        ↑ 덮어씀      ↑ 복사됨   ↑ 복사됨
```

뒤에 오는 키가 앞의 것을 덮어씁니다. 순서가 중요합니다.

```js
{ name: "이영희", ...user } // ❌ user의 name이 이영희를 덮어씁니다
{ ...user, name: "이영희" } // ✅ 이영희가 user.name을 덮어씁니다
```

## 4️⃣ 중첩 객체 업데이트 [↑](#toc)

객체 안에 객체가 있는 경우, 각 단계마다 스프레드가 필요합니다.

```jsx
import { useState } from "react";

function UserWithAddress() {
  const [user, setUser] = useState({
    name: "김철수",
    contact: {
      email: "chulsoo@example.com",
      phone: "010-1234-5678",
    },
    address: {
      city: "서울",
      district: "강남구",
      detail: "테헤란로 123",
    },
  });

  function updateCity(newCity) {
    setUser((prev) => ({
      ...prev,              // user 레벨 복사
      address: {
        ...prev.address,    // address 레벨도 복사
        city: newCity,      // city만 변경
      },
    }));
  }

  function updateEmail(newEmail) {
    setUser((prev) => ({
      ...prev,
      contact: {
        ...prev.contact,
        email: newEmail,
      },
    }));
  }

  return (
    <div className="p-6 space-y-4">
      <div className="bg-white rounded-xl p-4 shadow">
        <h3 className="font-bold mb-2">{user.name}</h3>
        <div className="text-sm space-y-1 text-gray-600">
          <p>📧 {user.contact.email}</p>
          <p>📱 {user.contact.phone}</p>
          <p>📍 {user.address.city} {user.address.district}</p>
        </div>
      </div>

      <div className="flex gap-2 flex-wrap">
        <button
          onClick={() => updateCity("부산")}
          className="bg-blue-500 text-white px-3 py-1.5 rounded text-sm"
        >
          도시 → 부산
        </button>
        <button
          onClick={() => updateEmail("new@example.com")}
          className="bg-green-500 text-white px-3 py-1.5 rounded text-sm"
        >
          이메일 변경
        </button>
      </div>
    </div>
  );
}
// "도시 → 부산" 클릭: address.city만 바뀌고, name/contact/address.district는 유지됩니다
```

> **중첩이 깊어질수록 스프레드도 깊어집니다.** 3단계 이상 중첩되면 코드가 복잡해집니다. 그런 경우 [Immer](https://immerjs.github.io/immer/) 같은 라이브러리를 고려하세요. 하지만 입문 단계에서는 스프레드로 충분합니다.

## 5️⃣ 배열 상태: 항목 추가 [↑](#toc)

배열도 직접 수정하면 안 됩니다. `push()`, `splice()`, `sort()` 등 원본을 변경하는 메서드는 사용하지 마세요.

### 항목 추가 — 스프레드 사용

```jsx
import { useState } from "react";

function TodoAdder() {
  const [todos, setTodos] = useState([
    { id: 1, text: "React 공부하기", done: false },
    { id: 2, text: "운동하기", done: false },
  ]);
  const [inputText, setInputText] = useState("");

  function handleAdd() {
    if (!inputText.trim()) return; // 빈 값은 무시

    const newTodo = {
      id: Date.now(), // 간단한 고유 ID
      text: inputText,
      done: false,
    };

    // ❌ 하지 마세요: todos.push(newTodo)
    // ✅ 스프레드로 새 배열 생성
    setTodos((prev) => [...prev, newTodo]);
    setInputText(""); // 입력 초기화
  }

  function handleKeyDown(e) {
    if (e.key === "Enter") handleAdd();
  }

  return (
    <div className="p-6 space-y-4 max-w-sm">
      <div className="flex gap-2">
        <input
          value={inputText}
          onChange={(e) => setInputText(e.target.value)}
          onKeyDown={handleKeyDown}
          placeholder="할 일을 입력하세요"
          className="flex-1 border rounded-lg px-3 py-2 focus:outline-none focus:ring-2 focus:ring-blue-400"
        />
        <button
          onClick={handleAdd}
          className="bg-blue-500 text-white px-4 py-2 rounded-lg hover:bg-blue-600"
        >
          추가
        </button>
      </div>

      <ul className="space-y-2">
        {todos.map((todo) => (
          <li key={todo.id} className="bg-white p-3 rounded-lg shadow-sm flex items-center gap-2">
            <span>{todo.text}</span>
          </li>
        ))}
      </ul>
      <p className="text-xs text-gray-400">총 {todos.length}개</p>
    </div>
  );
}
// "React 공부하기", "운동하기" 표시
// 입력 후 추가 시 목록 맨 아래에 새 항목 추가됩니다
```

### 앞에 추가하기

```js
// 배열 맨 앞에 추가
setTodos((prev) => [newTodo, ...prev]);
// 출력: 새 항목이 목록 맨 위에 나타납니다
```

## 6️⃣ 배열 상태: 항목 삭제 [↑](#toc)

`filter()`는 조건에 맞는 항목만 남긴 새 배열을 반환합니다. 원본을 수정하지 않으므로 삭제에 안성맞춤입니다.

```jsx
import { useState } from "react";

function TodoWithDelete() {
  const [todos, setTodos] = useState([
    { id: 1, text: "React 공부하기" },
    { id: 2, text: "운동하기" },
    { id: 3, text: "장보기" },
    { id: 4, text: "독서" },
  ]);

  function handleDelete(targetId) {
    // targetId와 다른 항목만 남김 = targetId 항목을 제거
    setTodos((prev) => prev.filter((todo) => todo.id !== targetId));
    // 출력: targetId가 2이면 "운동하기"가 사라짐
  }

  return (
    <div className="p-6 max-w-sm space-y-2">
      {todos.length === 0 ? (
        <p className="text-center text-gray-400 py-8">할 일이 없습니다! 🎉</p>
      ) : (
        todos.map((todo) => (
          <div
            key={todo.id}
            className="flex items-center justify-between bg-white p-3 rounded-lg shadow-sm"
          >
            <span className="text-gray-800">{todo.text}</span>
            <button
              onClick={() => handleDelete(todo.id)}
              className="text-red-400 hover:text-red-600 hover:bg-red-50 px-2 py-1 rounded transition-colors text-sm"
            >
              삭제
            </button>
          </div>
        ))
      )}
    </div>
  );
}
// 삭제 버튼 클릭 시 해당 항목만 제거됩니다
// 모두 삭제하면 "할 일이 없습니다! 🎉" 표시
```

## 7️⃣ 배열 상태: 항목 수정 [↑](#toc)

`map()`으로 특정 항목만 바꾼 새 배열을 만듭니다.

```jsx
import { useState } from "react";

function TodoWithToggle() {
  const [todos, setTodos] = useState([
    { id: 1, text: "React 공부하기", done: false },
    { id: 2, text: "운동하기", done: true },
    { id: 3, text: "장보기", done: false },
  ]);

  // done 상태 토글
  function handleToggle(targetId) {
    setTodos((prev) =>
      prev.map((todo) =>
        todo.id === targetId
          ? { ...todo, done: !todo.done } // 해당 항목만 done을 뒤집음
          : todo                          // 나머지는 그대로
      )
    );
  }

  // 텍스트 수정
  function handleEdit(targetId, newText) {
    setTodos((prev) =>
      prev.map((todo) =>
        todo.id === targetId
          ? { ...todo, text: newText }
          : todo
      )
    );
  }

  const doneCount = todos.filter((t) => t.done).length;

  return (
    <div className="p-6 max-w-sm space-y-3">
      <p className="text-sm text-gray-500">
        {doneCount} / {todos.length} 완료
      </p>

      {/* 진행 바 */}
      <div className="w-full bg-gray-200 rounded-full h-2">
        <div
          className="bg-blue-500 h-2 rounded-full transition-all"
          style={{ width: `${(doneCount / todos.length) * 100}%` }}
        />
      </div>

      {todos.map((todo) => (
        <div
          key={todo.id}
          className="flex items-center gap-3 bg-white p-3 rounded-lg shadow-sm"
        >
          <button
            onClick={() => handleToggle(todo.id)}
            className={`w-6 h-6 rounded-full border-2 flex items-center justify-center flex-shrink-0 transition-colors ${
              todo.done
                ? "bg-blue-500 border-blue-500 text-white"
                : "border-gray-300 hover:border-blue-400"
            }`}
          >
            {todo.done && "✓"}
          </button>
          <span
            className={`flex-1 transition-all ${
              todo.done ? "line-through text-gray-400" : "text-gray-800"
            }`}
          >
            {todo.text}
          </span>
        </div>
      ))}
    </div>
  );
}
// 원형 버튼 클릭 시 완료/미완료 전환
// 완료 항목은 취소선으로 표시됩니다
// 진행 바가 실시간으로 업데이트됩니다
```

## 8️⃣ 객체 배열 종합 예제 — 학생 관리 앱 [↑](#toc)

추가, 삭제, 수정을 모두 포함한 종합 예제입니다.

```jsx
import { useState } from "react";

const initialStudents = [
  { id: 1, name: "김철수", grade: "A", present: true },
  { id: 2, name: "이영희", grade: "B+", present: false },
  { id: 3, name: "박민준", grade: "A+", present: true },
];

function StudentManager() {
  const [students, setStudents] = useState(initialStudents);
  const [newName, setNewName] = useState("");

  // 추가
  function handleAdd() {
    if (!newName.trim()) return;
    setStudents((prev) => [
      ...prev,
      { id: Date.now(), name: newName, grade: "미정", present: false },
    ]);
    setNewName("");
  }

  // 삭제
  function handleDelete(id) {
    setStudents((prev) => prev.filter((s) => s.id !== id));
  }

  // 출석 토글
  function handleTogglePresent(id) {
    setStudents((prev) =>
      prev.map((s) =>
        s.id === id ? { ...s, present: !s.present } : s
      )
    );
  }

  // 성적 업데이트
  function handleGradeChange(id, newGrade) {
    setStudents((prev) =>
      prev.map((s) =>
        s.id === id ? { ...s, grade: newGrade } : s
      )
    );
  }

  const presentCount = students.filter((s) => s.present).length;

  return (
    <div className="p-6 max-w-md space-y-4">
      <div className="flex justify-between items-center">
        <h2 className="text-xl font-bold">학생 관리</h2>
        <span className="text-sm text-gray-500">
          출석: {presentCount}/{students.length}명
        </span>
      </div>

      {/* 학생 추가 */}
      <div className="flex gap-2">
        <input
          value={newName}
          onChange={(e) => setNewName(e.target.value)}
          onKeyDown={(e) => e.key === "Enter" && handleAdd()}
          placeholder="학생 이름 추가"
          className="flex-1 border rounded-lg px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-blue-400"
        />
        <button
          onClick={handleAdd}
          className="bg-blue-500 text-white px-4 py-2 rounded-lg text-sm hover:bg-blue-600"
        >
          + 추가
        </button>
      </div>

      {/* 학생 목록 */}
      <div className="space-y-2">
        {students.map((student) => (
          <div
            key={student.id}
            className={`flex items-center gap-3 p-3 rounded-lg border transition-colors ${
              student.present ? "bg-green-50 border-green-200" : "bg-white border-gray-200"
            }`}
          >
            {/* 출석 버튼 */}
            <button
              onClick={() => handleTogglePresent(student.id)}
              className={`w-8 h-8 rounded-full text-xs font-bold flex items-center justify-center flex-shrink-0 transition-colors ${
                student.present
                  ? "bg-green-500 text-white"
                  : "bg-gray-200 text-gray-500 hover:bg-gray-300"
              }`}
            >
              {student.present ? "출" : "결"}
            </button>

            {/* 이름 */}
            <span className="flex-1 font-medium text-gray-800">
              {student.name}
            </span>

            {/* 성적 select */}
            <select
              value={student.grade}
              onChange={(e) => handleGradeChange(student.id, e.target.value)}
              className="text-sm border border-gray-200 rounded px-2 py-1 bg-white focus:outline-none"
            >
              {["A+", "A", "B+", "B", "C", "미정"].map((g) => (
                <option key={g} value={g}>{g}</option>
              ))}
            </select>

            {/* 삭제 */}
            <button
              onClick={() => handleDelete(student.id)}
              className="text-gray-300 hover:text-red-400 transition-colors"
            >
              ✕
            </button>
          </div>
        ))}
      </div>
    </div>
  );
}

export default StudentManager;
// 출석 버튼: "출"/"결" 토글 (초록/회색)
// 성적 select: 드롭다운으로 성적 변경
// ✕ 버튼: 해당 학생 삭제
// + 추가: 새 학생 추가 (Enter키도 동작)
```

## 9️⃣ 실습: 장바구니 [↑](#toc)

### 기본 과제

상품을 담고, 수량을 조절하고, 삭제할 수 있는 장바구니를 만드세요.

```jsx
// src/App.jsx
import { useState } from "react";

const PRODUCTS = [
  { id: 1, name: "아메리카노", price: 4500, emoji: "☕" },
  { id: 2, name: "카페라떼", price: 5000, emoji: "🥛" },
  { id: 3, name: "치즈케이크", price: 6500, emoji: "🍰" },
  { id: 4, name: "녹차 라떼", price: 5500, emoji: "🍵" },
];

function Cart() {
  // 장바구니: [{ id, name, price, emoji, quantity }, ...]
  const [cartItems, setCartItems] = useState([]);

  // 상품 추가 (이미 있으면 수량 증가)
  function handleAddToCart(product) {
    setCartItems((prev) => {
      const existing = prev.find((item) => item.id === product.id);
      if (existing) {
        // 이미 있는 상품 → 수량 +1
        return prev.map((item) =>
          item.id === product.id
            ? { ...item, quantity: item.quantity + 1 }
            : item
        );
      }
      // 새 상품 → 배열에 추가
      return [...prev, { ...product, quantity: 1 }];
    });
  }

  // 수량 감소 (1에서 감소하면 삭제)
  function handleDecrement(id) {
    setCartItems((prev) => {
      const item = prev.find((i) => i.id === id);
      if (item.quantity === 1) {
        return prev.filter((i) => i.id !== id); // 삭제
      }
      return prev.map((i) =>
        i.id === id ? { ...i, quantity: i.quantity - 1 } : i
      );
    });
  }

  // 수량 증가
  function handleIncrement(id) {
    setCartItems((prev) =>
      prev.map((i) =>
        i.id === id ? { ...i, quantity: i.quantity + 1 } : i
      )
    );
  }

  // 항목 삭제
  function handleRemove(id) {
    setCartItems((prev) => prev.filter((i) => i.id !== id));
  }

  // 합계 계산
  const totalPrice = cartItems.reduce(
    (sum, item) => sum + item.price * item.quantity,
    0
  );
  const totalCount = cartItems.reduce((sum, item) => sum + item.quantity, 0);

  return (
    <div className="min-h-screen bg-gray-50 p-6">
      <div className="max-w-lg mx-auto space-y-6">
        <h1 className="text-2xl font-bold">☕ 카페 주문</h1>

        {/* 상품 목록 */}
        <div className="grid grid-cols-2 gap-3">
          {PRODUCTS.map((product) => (
            <button
              key={product.id}
              onClick={() => handleAddToCart(product)}
              className="bg-white rounded-xl p-4 shadow-sm text-left hover:shadow-md hover:-translate-y-0.5 transition-all active:scale-95"
            >
              <div className="text-3xl mb-1">{product.emoji}</div>
              <div className="font-semibold text-gray-800">{product.name}</div>
              <div className="text-blue-600 font-bold text-sm">
                {product.price.toLocaleString("ko-KR")}원
              </div>
            </button>
          ))}
        </div>

        {/* 장바구니 */}
        <div className="bg-white rounded-xl shadow p-4">
          <h2 className="font-bold text-lg mb-3 flex items-center gap-2">
            🛒 장바구니
            {totalCount > 0 && (
              <span className="bg-blue-500 text-white text-xs w-5 h-5 rounded-full flex items-center justify-center">
                {totalCount}
              </span>
            )}
          </h2>

          {cartItems.length === 0 ? (
            <p className="text-center text-gray-400 py-8 text-sm">
              상품을 담아보세요
            </p>
          ) : (
            <>
              <div className="space-y-3 mb-4">
                {cartItems.map((item) => (
                  <div key={item.id} className="flex items-center gap-3">
                    <span className="text-xl">{item.emoji}</span>
                    <span className="flex-1 text-sm font-medium">{item.name}</span>

                    {/* 수량 조절 */}
                    <div className="flex items-center gap-2">
                      <button
                        onClick={() => handleDecrement(item.id)}
                        className="w-7 h-7 rounded-full bg-gray-100 hover:bg-gray-200 flex items-center justify-center font-bold"
                      >
                        −
                      </button>
                      <span className="w-4 text-center text-sm font-semibold">
                        {item.quantity}
                      </span>
                      <button
                        onClick={() => handleIncrement(item.id)}
                        className="w-7 h-7 rounded-full bg-gray-100 hover:bg-gray-200 flex items-center justify-center font-bold"
                      >
                        +
                      </button>
                    </div>

                    <span className="text-sm font-bold text-blue-600 w-16 text-right">
                      {(item.price * item.quantity).toLocaleString("ko-KR")}원
                    </span>

                    <button
                      onClick={() => handleRemove(item.id)}
                      className="text-gray-300 hover:text-red-400 text-xs"
                    >
                      ✕
                    </button>
                  </div>
                ))}
              </div>

              {/* 합계 */}
              <div className="border-t pt-3 flex justify-between items-center">
                <span className="font-semibold">합계</span>
                <span className="text-xl font-bold text-blue-600">
                  {totalPrice.toLocaleString("ko-KR")}원
                </span>
              </div>

              <button className="w-full mt-3 bg-blue-600 text-white py-3 rounded-xl font-semibold hover:bg-blue-700 transition-colors">
                주문하기
              </button>
            </>
          )}
        </div>
      </div>
    </div>
  );
}

export default Cart;
// 상품 카드 클릭 → 장바구니에 추가
// 이미 담긴 상품 클릭 → 수량 증가
// − 버튼 → 수량 감소 (수량 1에서 누르면 삭제)
// ✕ 버튼 → 즉시 삭제
// 합계 금액 실시간 계산
```

### 도전 과제

1. **전체 삭제**: "장바구니 비우기" 버튼 추가 (`setCartItems([])`)
2. **중복 방지 알림**: 이미 담긴 상품을 추가할 때 "수량이 늘어났습니다!" 토스트 메시지 표시
3. **localStorage 연동**: 새로고침해도 장바구니가 유지되도록 (힌트: `useEffect` + `localStorage`)

## 🔟 정리 + 브릿지 [↑](#toc)

### 핵심 정리

| 작업 | 방법 |
|------|------|
| 객체 필드 변경 | `{ ...prev, field: newValue }` |
| 중첩 객체 변경 | `{ ...prev, nested: { ...prev.nested, field: newValue } }` |
| 배열에 추가 | `[...prev, newItem]` |
| 배열에서 삭제 | `prev.filter(item => item.id !== targetId)` |
| 배열 항목 수정 | `prev.map(item => item.id === id ? { ...item, ... } : item)` |

### 절대 쓰지 말 것

| 금지 | 이유 |
|------|------|
| `state.field = value` | 참조 같음 → 리렌더 안 됨 |
| `array.push(item)` | 원본 변경 → 리렌더 안 됨 |
| `array.splice(i, 1)` | 원본 변경 → 리렌더 안 됨 |
| `array.sort()` | 원본 변경 → `[...array].sort()` 사용 |

### 체크리스트

- [ ] 직접 수정 vs 스프레드의 차이를 설명할 수 있다
- [ ] `{ ...prev, name: "새이름" }` 으로 객체 상태를 업데이트할 수 있다
- [ ] `[...prev, newItem]` 으로 배열에 항목을 추가할 수 있다
- [ ] `prev.filter(...)` 로 특정 항목을 삭제할 수 있다
- [ ] `prev.map(item => id 일치 ? {...item, done: true} : item)` 으로 수정할 수 있다
- [ ] 장바구니처럼 추가/수량변경/삭제가 모두 있는 앱을 만들 수 있다
