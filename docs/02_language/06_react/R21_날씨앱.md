## 1️⃣ 프로젝트 소개 [↑](#toc)

### 무엇을 만드나요?

도시 이름을 검색하면 **현재 날씨**를 보여주는 웹 앱입니다.

```
┌─────────────────────────────────────────┐
│  날씨 앱                        About   │  ← Navbar (React Router)
├─────────────────────────────────────────┤
│                                         │
│   🔍 [ 도시 이름 입력...     ] [검색]   │  ← SearchBar
│                                         │
│   ┌─────────────────────────────────┐   │
│   │   서울                     🌤️  │   │  ← WeatherCard
│   │   맑음 (Clear sky)              │   │
│   │   🌡️ 22°C    💧 45%    💨 3m/s │   │
│   └─────────────────────────────────┘   │
│                                         │
│   최근 검색: 서울  도쿄  뉴욕  런던    │  ← RecentSearches
│                                         │
└─────────────────────────────────────────┘
```

### 핵심 기능

- **도시 검색**: 영문 또는 한글 도시명으로 날씨 조회
- **날씨 표시**: 온도, 날씨 설명, 아이콘, 습도, 풍속
- **최근 검색 기록**: 최근 5개 검색어 저장 (Zustand), 클릭하면 재검색
- **에러 처리**: 존재하지 않는 도시, 네트워크 오류 안내
- **반응형 디자인**: 모바일/데스크탑 모두 지원

## 5️⃣ Step 2: useWeather Custom Hook [↑](#toc)

날씨 API 호출 로직을 Custom Hook으로 분리합니다. 컴포넌트는 이 훅을 쓰기만 하면 됩니다.

```js
// src/hooks/useWeather.js
import { useState, useCallback } from 'react';

const BASE_URL = import.meta.env.VITE_WEATHER_BASE_URL;
const API_KEY = import.meta.env.VITE_WEATHER_API_KEY;

function useWeather() {
  const [weather, setWeather] = useState(null);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState(null);

  const fetchWeather = useCallback(async (city) => {
    if (!city.trim()) return;

    setLoading(true);
    setError(null);
    setWeather(null);

    try {
      const url = `${BASE_URL}/weather?q=${encodeURIComponent(city)}&appid=${API_KEY}&units=metric&lang=kr`;
      const res = await fetch(url);

      if (res.status === 404) {
        throw new Error(`"${city}" 도시를 찾을 수 없습니다. 영문 도시명으로 시도해 보세요.`);
      }
      if (res.status === 401) {
        throw new Error('API 키가 유효하지 않습니다. .env 파일을 확인하세요.');
      }
      if (!res.ok) {
        throw new Error(`날씨 정보를 불러오지 못했습니다. (오류 ${res.status})`);
      }

      const data = await res.json();
      setWeather(data);
    } catch (err) {
      setError(err.message);
    } finally {
      setLoading(false);
    }
  }, []);

  return { weather, loading, error, fetchWeather };
}

export default useWeather;
```

### API 응답 구조 미리 보기

```json
{
  "name": "Seoul",
  "weather": [{ "description": "맑음", "icon": "01d" }],
  "main": {
    "temp": 22.3,
    "feels_like": 21.0,
    "humidity": 45
  },
  "wind": { "speed": 3.2 },
  "sys": { "country": "KR" }
}
```

이 구조를 알면 컴포넌트에서 `weather.main.temp`, `weather.weather[0].description` 처럼 접근할 수 있습니다.

## 6️⃣ Step 3: SearchBar 컴포넌트 [↑](#toc)

검색어 입력과 제출을 담당합니다. `useRef`로 마운트 시 자동 포커스를 구현합니다.

```jsx
// src/components/SearchBar.jsx
import { useState, useRef, useEffect } from 'react';

function SearchBar({ onSearch, isLoading }) {
  const [city, setCity] = useState('');
  const inputRef = useRef(null);

  // 마운트 시 자동 포커스
  useEffect(() => {
    inputRef.current?.focus();
  }, []);

  const handleSubmit = (e) => {
    e.preventDefault();
    if (!city.trim() || isLoading) return;
    onSearch(city.trim());
    setCity('');
  };

  return (
    <form onSubmit={handleSubmit} className="flex gap-2 w-full max-w-lg mx-auto">
      <input
        ref={inputRef}
        type="text"
        value={city}
        onChange={(e) => setCity(e.target.value)}
        placeholder="도시 이름을 입력하세요 (예: Seoul, Tokyo)"
        disabled={isLoading}
        className="flex-1 px-4 py-3 rounded-xl border border-gray-300 shadow-sm
                   focus:outline-none focus:ring-2 focus:ring-blue-400 focus:border-transparent
                   disabled:bg-gray-50 disabled:text-gray-400 transition text-gray-800"
      />
      <button
        type="submit"
        disabled={!city.trim() || isLoading}
        className="px-6 py-3 bg-blue-600 text-white rounded-xl font-semibold
                   hover:bg-blue-700 active:scale-95 disabled:opacity-50
                   disabled:cursor-not-allowed transition"
      >
        {isLoading ? '검색 중...' : '검색'}
      </button>
    </form>
  );
}

export default SearchBar;
```

## 7️⃣ Step 4: WeatherCard 컴포넌트 [↑](#toc)

```jsx
// src/components/WeatherCard.jsx

// 날씨 아이콘 URL 생성 헬퍼
const getIconUrl = (iconCode) =>
  `https://openweathermap.org/img/wn/${iconCode}@2x.png`;

// 로딩 스켈레톤
function LoadingSkeleton() {
  return (
    <div className="bg-white rounded-2xl shadow-md p-6 w-full max-w-sm mx-auto animate-pulse">
      <div className="h-6 bg-gray-200 rounded w-1/2 mb-4" />
      <div className="h-16 bg-gray-200 rounded w-1/3 mx-auto mb-4" />
      <div className="h-4 bg-gray-200 rounded w-2/3 mx-auto mb-6" />
      <div className="grid grid-cols-3 gap-3">
        {[0, 1, 2].map((i) => (
          <div key={i} className="h-16 bg-gray-100 rounded-xl" />
        ))}
      </div>
    </div>
  );
}

function WeatherCard({ weather, loading, error }) {
  if (loading) return <LoadingSkeleton />;

  if (error) {
    return (
      <div className="bg-red-50 border border-red-200 rounded-2xl p-6 w-full max-w-sm mx-auto text-center">
        <p className="text-3xl mb-3">😵</p>
        <p className="font-semibold text-red-700 mb-1">오류가 발생했습니다</p>
        <p className="text-sm text-red-500">{error}</p>
      </div>
    );
  }

  if (!weather) {
    return (
      <div className="bg-blue-50 border-2 border-dashed border-blue-200 rounded-2xl p-10 w-full max-w-sm mx-auto text-center text-blue-300">
        <p className="text-5xl mb-3">🌤️</p>
        <p className="text-sm">도시 이름을 검색해 보세요</p>
      </div>
    );
  }

  const { name, weather: conditions, main, wind, sys } = weather;
  const condition = conditions[0];
  const temp = Math.round(main.temp);
  const feelsLike = Math.round(main.feels_like);

  return (
    <div className="bg-gradient-to-br from-blue-400 to-blue-600 text-white rounded-2xl shadow-lg p-6 w-full max-w-sm mx-auto">
      {/* 도시 및 국가 */}
      <div className="flex justify-between items-start mb-2">
        <div>
          <h2 className="text-2xl font-bold">{name}</h2>
          <p className="text-blue-100 text-sm">{sys.country}</p>
        </div>
        <img
          src={getIconUrl(condition.icon)}
          alt={condition.description}
          className="w-16 h-16 drop-shadow"
        />
      </div>

      {/* 온도 */}
      <div className="text-center my-4">
        <p className="text-7xl font-bold tabular-nums">{temp}°</p>
        <p className="text-blue-100 capitalize mt-1">{condition.description}</p>
        <p className="text-blue-200 text-sm mt-1">체감 {feelsLike}°C</p>
      </div>

      {/* 세부 정보 */}
      <div className="grid grid-cols-3 gap-2 mt-4">
        <div className="bg-white/20 rounded-xl p-3 text-center backdrop-blur-sm">
          <p className="text-xl">💧</p>
          <p className="font-semibold text-sm mt-1">{main.humidity}%</p>
          <p className="text-blue-100 text-xs">습도</p>
        </div>
        <div className="bg-white/20 rounded-xl p-3 text-center backdrop-blur-sm">
          <p className="text-xl">💨</p>
          <p className="font-semibold text-sm mt-1">{wind.speed}m/s</p>
          <p className="text-blue-100 text-xs">풍속</p>
        </div>
        <div className="bg-white/20 rounded-xl p-3 text-center backdrop-blur-sm">
          <p className="text-xl">🌡️</p>
          <p className="font-semibold text-sm mt-1">{Math.round(main.temp_max)}°</p>
          <p className="text-blue-100 text-xs">최고</p>
        </div>
      </div>
    </div>
  );
}

export default WeatherCard;
```

## 8️⃣ Step 5: Zustand 검색 기록 스토어 [↑](#toc)

최근 검색어를 전역으로 관리합니다. 최대 5개, 중복 방지 로직을 포함합니다.

```js
// src/stores/useSearchStore.js
import { create } from 'zustand';

const useSearchStore = create((set, get) => ({
  history: [],  // 최근 검색어 배열 (최신이 앞에)

  addSearch: (city) => {
    const trimmed = city.trim();
    if (!trimmed) return;

    set((state) => {
      // 이미 있으면 앞으로 이동
      const filtered = state.history.filter(
        (h) => h.toLowerCase() !== trimmed.toLowerCase()
      );
      // 최대 5개 유지
      const newHistory = [trimmed, ...filtered].slice(0, 5);
      return { history: newHistory };
    });
  },

  clearHistory: () => set({ history: [] }),

  getHistory: () => get().history,
}));

export default useSearchStore;
```

## 9️⃣ Step 6: RecentSearches 컴포넌트 [↑](#toc)

최근 검색어 목록을 표시하고, 클릭하면 재검색합니다.

```jsx
// src/components/RecentSearches.jsx
import useSearchStore from '../stores/useSearchStore';

function RecentSearches({ onSearch }) {
  const history = useSearchStore((state) => state.history);
  const clearHistory = useSearchStore((state) => state.clearHistory);

  if (history.length === 0) return null;

  return (
    <div className="w-full max-w-lg mx-auto mt-4">
      <div className="flex justify-between items-center mb-2">
        <p className="text-sm text-gray-400 font-medium">최근 검색</p>
        <button
          onClick={clearHistory}
          className="text-xs text-gray-300 hover:text-red-400 transition"
        >
          전체 삭제
        </button>
      </div>
      <div className="flex flex-wrap gap-2">
        {history.map((city) => (
          <button
            key={city}
            onClick={() => onSearch(city)}
            className="px-3 py-1.5 bg-white border border-gray-200 rounded-full text-sm
                       text-gray-600 hover:bg-blue-50 hover:border-blue-300 hover:text-blue-700
                       transition shadow-sm"
          >
            {city}
          </button>
        ))}
      </div>
    </div>
  );
}

export default RecentSearches;
```

### Home.jsx

```jsx
// src/pages/Home.jsx
import useWeather from '../hooks/useWeather';
import useSearchStore from '../stores/useSearchStore';
import SearchBar from '../components/SearchBar';
import WeatherCard from '../components/WeatherCard';
import RecentSearches from '../components/RecentSearches';

function Home() {
  const { weather, loading, error, fetchWeather } = useWeather();
  const addSearch = useSearchStore((state) => state.addSearch);

  const handleSearch = (city) => {
    addSearch(city);
    fetchWeather(city);
  };

  return (
    <div className="flex flex-col items-center gap-6">
      <div className="text-center mb-2">
        <h1 className="text-3xl font-bold text-gray-800">날씨 검색</h1>
        <p className="text-gray-400 text-sm mt-1">전 세계 도시의 실시간 날씨를 확인하세요</p>
      </div>

      <SearchBar onSearch={handleSearch} isLoading={loading} />
      <RecentSearches onSearch={handleSearch} />
      <WeatherCard weather={weather} loading={loading} error={error} />
    </div>
  );
}

export default Home;
```

## 1️⃣3️⃣ 과정을 마치며 [↑](#toc)

### 20장 전의 나 vs 지금의 나

| 이전 | 지금 |
|------|------|
| React가 뭔지 몰랐다 | 함수 컴포넌트를 직접 만든다 |

| API가 무섭게 느껴졌다 | OpenWeatherMap API를 연동했다 |

| 페이지 이동을 몰랐다 | React Router로 SPA를 만든다 |
| Tailwind가 낯설었다 | 클래스 이름만으로 반응형 UI를 만든다 |

### 다음으로 배우면 좋은 것

이제 React의 기초를 완성했습니다. 다음 단계로 나아가기 위한 로드맵입니다.

#### 1. TypeScript — 타입 안전성

JavaScript에 타입을 추가합니다. 프로젝트가 커질수록 오류를 미리 잡아줍니다.

```
배울 내용: 기본 타입, 인터페이스, 제네릭, React with TypeScript
참고: typescriptlang.org/docs (공식 문서)
```

#### 2. Next.js — 풀스택 React 프레임워크

서버 사이드 렌더링(SSR), 파일 기반 라우팅, API 라우트를 제공합니다. 취업 시장에서 가장 수요가 많습니다.

```
배울 내용: App Router, Server Components, 데이터 페칭, 배포
참고: nextjs.org/docs (공식 문서)
```

#### 3. React Query (TanStack Query) — 서버 상태 관리

fetch + loading + error + 캐싱을 모두 자동으로 처리해 줍니다. 이 과정에서 배운 `useFetch`보다 훨씬 강력합니다.

```
배울 내용: useQuery, useMutation, 캐싱 전략, 낙관적 업데이트
참고: tanstack.com/query/latest
```

#### 4. Vitest — 테스트

코드가 의도대로 동작하는지 자동으로 확인합니다. 실무에서 필수입니다.

```
배울 내용: 단위 테스트, 컴포넌트 테스트, React Testing Library
참고: vitest.dev (공식 문서)
```

#### 5. 프로젝트 배포 — Vercel

만든 앱을 인터넷에 올립니다. GitHub 연동 시 push만 하면 자동 배포됩니다.

```
방법: GitHub에 프로젝트 올리기 → vercel.com에서 import
참고: vercel.com/docs
```
