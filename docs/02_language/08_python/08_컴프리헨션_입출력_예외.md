## 1️⃣ 리스트 컴프리헨션 [↑](#toc)

> **for 루프 + append 를 한 줄로** 줄이는 Python 특유의 문법.

### 기본 변환

```python
# 루프 버전
result = []
for x in range(5):
    result.append(x * 2)
print(result)    # [0, 2, 4, 6, 8]

# 컴프리헨션 버전 (같은 결과)
result = [x * 2 for x in range(5)]
print(result)    # [0, 2, 4, 6, 8]
```

### 일반 형식

```
[ 표현식  for 변수 in 반복가능  if 조건 ]
       ↑             ↑              ↑
     변환식         원본          (선택) 필터
```

### 예제 4종

```python
nums = [1, 2, 3, 4, 5]

# 1. 변환만
squares = [n * n for n in nums]
# [1, 4, 9, 16, 25]

# 2. 조건 필터 (짝수만)
evens = [n for n in nums if n % 2 == 0]
# [2, 4]

# 3. 변환 + 조건
even_squares = [n * n for n in nums if n % 2 == 0]
# [4, 16]

# 4. 중첩 (2차원 펼치기)
matrix = [[1, 2], [3, 4], [5, 6]]
flat = [x for row in matrix for x in row]
# [1, 2, 3, 4, 5, 6]
```

### 흔한 활용 패턴

```python
# 문자열 길이 리스트
words = ["apple", "fig", "banana"]
lengths = [len(w) for w in words]
# [5, 3, 6]

# 대소문자 변환
upper_words = [w.upper() for w in words]

# DataFrame 컬럼 변환 (ML 챕터에서 자주)
log_prices = [round(math.log(p), 2) for p in prices]
```

### ⚠️ 가독성 한계

```python
# ❌ 너무 복잡한 컴프리헨션 — 그냥 루프로 풀기
result = [(x, y) for x in range(10) if x % 2 == 0
                  for y in range(10) if y > x and y % 3 == 0]

# ✅ 읽기 쉽게 루프로
result = []
for x in range(10):
    if x % 2 == 0:
        for y in range(10):
            if y > x and y % 3 == 0:
                result.append((x, y))
```

> 💡 **외워두기**: 컴프리헨션은 **한 줄이 가독성 향상**. 두 줄 넘어가면 그냥 루프.

## 2️⃣ 딕셔너리·세트 컴프리헨션 [↑](#toc)

### dict comprehension

```python
nums = [1, 2, 3, 4]

# 숫자: 제곱
squares = {n: n ** 2 for n in nums}
# {1: 1, 2: 4, 3: 9, 4: 16}

# 두 리스트 → dict
keys = ["name", "age", "city"]
values = ["Alice", 30, "Seoul"]
d = {k: v for k, v in zip(keys, values)}
# {'name': 'Alice', 'age': 30, 'city': 'Seoul'}
# (사실은 dict(zip(keys, values)) 한 줄로도 가능)

# 조건 필터
people = {"Alice": 30, "Bob": 17, "Carol": 25}
adults = {name: age for name, age in people.items() if age >= 18}
# {'Alice': 30, 'Carol': 25}

# dict 뒤집기
inv = {v: k for k, v in people.items()}
```

### set comprehension

```python
words = ["apple", "banana", "apple", "cherry"]
first_chars = {w[0] for w in words}
# {'a', 'b', 'c'}

# 컴프리헨션 + set 변환
nums = [1, 2, 2, 3, 3, 4]
unique_squares = {n ** 2 for n in nums}
# {1, 4, 9, 16}
```

## 3️⃣ 파일 입출력 — `with open` [↑](#toc)

> **`with` 블록을 벗어나면 자동으로 파일 닫힘**. 가장 안전한 패턴.

### 읽기

```python
# 전체 한 번에
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)

# 한 줄씩 (대용량 파일 권장)
with open("data.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())    # 줄 끝 \n 제거

# 줄을 리스트로
with open("data.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()       # ['line1\n', 'line2\n', ...]
```

### 쓰기

```python
# "w" — 새로 쓰기 (기존 내용 덮어씀)
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("첫 줄\n")
    f.write("둘째 줄\n")

# "a" — 이어쓰기
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("추가 줄\n")
```

### 모드 표

| 모드 | 의미 |
|---|---|
| `"r"` | 읽기 (기본값) |
| `"w"` | 쓰기 (덮어씀!) |
| `"a"` | 이어쓰기 |
| `"x"` | 새 파일 (이미 있으면 에러) |
| `"b"` | 바이너리 (ex: 이미지는 `"rb"`) |
| `"t"` | 텍스트 (기본값) |

### ⚠️ 흔한 실수

```python
# ❌ with 없이 — 파일 안 닫힘 위험
f = open("data.txt")
content = f.read()
# 에러 나면 close() 호출 안 됨
f.close()

# ✅ with 사용
with open("data.txt") as f:
    content = f.read()
# 자동 close
```

```python
# ❌ 한글 파일 — encoding 누락
with open("한글.txt") as f:    # Windows에서 UnicodeDecodeError 가능
    ...

# ✅
with open("한글.txt", encoding="utf-8") as f:
    ...
```

> 💡 **외워두기**: 파일 열기는 **무조건 `with open(..., encoding="utf-8")`**.

## 4️⃣ 예외 처리 기본 [↑](#toc)

> 에러가 나면 프로그램이 죽지 않게 **잡아서 대응**.

### 기본 형태

```python
try:
    n = int(input("숫자: "))
    print(100 / n)
except ValueError:
    print("숫자를 입력해야 합니다")
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다")
```

### 4단계 형태 — `try / except / else / finally`

```python
try:
    f = open("data.txt")
    data = f.read()
except FileNotFoundError:
    print("파일 없음")
else:
    # try 가 성공했을 때만 실행
    print("읽기 성공!", data[:50])
finally:
    # 성공이든 실패든 무조건 실행
    print("정리 작업")
```

| 블록 | 언제 실행 |
|---|---|
| `try` | 가장 먼저 시도 |
| `except` | try 에서 에러가 나면 |
| `else` | try 가 **에러 없이** 끝나면 |
| `finally` | **무조건** 마지막에 실행 |

### 여러 예외를 한 번에

```python
try:
    ...
except (ValueError, TypeError) as e:
    print(f"입력 오류: {e}")
```

### 모든 예외 잡기 (권장 X)

```python
# ❌ 너무 광범위 — 디버깅 어려움
try:
    ...
except:
    pass    # 모든 에러를 조용히 무시

# 🤔 그래도 쓰려면 — 적어도 메시지는 출력
try:
    ...
except Exception as e:
    print(f"예상치 못한 오류: {e}")
```

### 직접 예외 발생 — `raise`

```python
def divide(a, b):
    if b == 0:
        raise ValueError("0으로 나눌 수 없습니다")
    return a / b

try:
    divide(10, 0)
except ValueError as e:
    print(e)    # 0으로 나눌 수 없습니다
```

## 5️⃣ 흔한 예외 5종 [↑](#toc)

| 예외 | 의미 | 예 |
|---|---|---|
| `ValueError` | 값이 부적절 | `int("hello")` |
| `TypeError` | 타입이 부적절 | `"a" + 1` |
| `KeyError` | dict에 key 없음 | `d["unknown"]` |
| `IndexError` | 리스트 인덱스 초과 | `[1,2][5]` |
| `FileNotFoundError` | 파일 없음 | `open("nofile.txt")` |
| `ZeroDivisionError` | 0으로 나눔 | `1 / 0` |
| `AttributeError` | 메서드/속성 없음 | `"str".unknown()` |
| `NameError` | 변수 정의 안 됨 | `print(unknown_var)` |

### 에러 메시지 읽는 법

```
Traceback (most recent call last):
  File "main.py", line 5, in <module>
    result = data["age"]
KeyError: 'age'
       ↑    ↑
       │    └ 어떤 key 가 문제인지
       └ 예외 종류
```

> 💡 **외워두기**: 에러 메시지의 **마지막 줄**이 가장 중요. 거기서 예외 종류와 원인을 봄.

## 6️⃣ 자기 점검 5문항 [↑](#toc)

### Q1

다음 코드를 컴프리헨션으로 줄이세요.

```python
result = []
for n in [1, 2, 3, 4, 5]:
    if n % 2 == 1:
        result.append(n * 10)
```

<details><summary>정답</summary>
```python
result = [n * 10 for n in [1, 2, 3, 4, 5] if n % 2 == 1]
# [10, 30, 50]
```
</details>

### Q2

`words = ["apple","banana","cherry"]` 를 `{단어: 길이}` dict로 만드세요.

<details><summary>정답</summary>
```python
result = {w: len(w) for w in words}
# {'apple': 5, 'banana': 6, 'cherry': 6}
```
</details>

### Q3

`output.txt` 에 줄을 이어쓰는 코드. 무엇이 잘못됐나?

```python
f = open("output.txt", "w", encoding="utf-8")
f.write("새 줄\n")
```

<details><summary>정답</summary>
두 가지:
1. `"w"` 는 **덮어쓰기**. 이어쓰려면 `"a"`.
2. `f.close()` 가 없음. `with open` 패턴 사용.

```python
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("새 줄\n")
```
</details>

### Q4

사용자가 숫자가 아닌 값을 입력해도 죽지 않는 입력 함수.

```python
def get_number(prompt):
    # ?
    pass
```

<details><summary>정답</summary>
```python
def get_number(prompt):
    while True:
        try:
            return int(input(prompt))
        except ValueError:
            print("숫자만 입력하세요.")
```
</details>

### Q5

다음 에러 메시지의 의미와 수정 방법은?

```
Traceback (most recent call last):
  File "main.py", line 3, in <module>
    print(d["age"])
KeyError: 'age'
```

<details><summary>정답</summary>
`d` 에 `"age"` 라는 key 가 없음. 두 가지 수정 방법:

1. 안전한 접근: `d.get("age", 0)`
2. 미리 확인: `if "age" in d: print(d["age"])`
3. try/except: 
```python
try:
    print(d["age"])
except KeyError:
    print("나이 정보 없음")
```
</details>

## 정리 카드

```
컴프리헨션
  [표현식 for x in 반복가능 if 조건]
  {k: v for ...}     # dict
  {x for ...}        # set

파일 입출력
  with open(path, "r", encoding="utf-8") as f:
      ...
  "r"=읽기 "w"=쓰기(덮어씀) "a"=이어쓰기

예외 처리
  try:        # 시도
  except E:   # 특정 예외
  else:       # try 성공 시
  finally:    # 무조건

raise ValueError("메시지")    # 직접 발생

흔한 예외
  ValueError / TypeError / KeyError / IndexError
  FileNotFoundError / ZeroDivisionError
```
