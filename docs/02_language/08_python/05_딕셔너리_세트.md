> ⭐ **상세 챕터** — 실무에서 가장 많이 만나지만 학생이 가장 적게 익숙한 자료구조.

## 학습 목표

- `dict` 와 `set` 의 용도를 구분하고 만들 수 있다
- **key-value 패턴**으로 데이터를 모델링할 수 있다
- `KeyError` 를 피하는 안전한 접근법(`get`, `setdefault`, `in`)을 사용한다
- 자주 쓰는 dict 관용구 5개(`.items()` 순회, `dict comprehension`, 카운팅 등) 를 활용한다
- `set` 을 이용한 **중복 제거**와 **집합 연산(합/교/차)** 을 수행한다

## 1️⃣ dict 와 set — 비유로 이해 [↑](#toc)

| | `dict` | `set` |
|---|---|---|
| 표기 | `{"name": "Alice", "age": 30}` | `{1, 2, 3}` |
| 구조 | **key → value** 쌍 | **값만**, 중복 없음 |
| 비유 | **사물함** — 라벨(key)로 짐(value) 찾기 | **회원 명단** — 중복 안 됨, 순서 무관 |
| 접근 속도 | 매우 빠름 (해시) | 매우 빠름 (해시) |
| 순서? | Python 3.7+ 부터 **삽입 순서 유지** | 순서 없음 (출력 순서 보장 X) |

> 💡 둘 다 **중괄호 `{}`** 를 씁니다. 차이는 `key: value` 가 있느냐 (dict) 없느냐 (set). 빈 `{}` 는 **dict** (set은 `set()`).

## 2️⃣ dict 기본 [↑](#toc)

### 생성

```python
# 가장 흔한 방식
person = {"name": "Alice", "age": 30, "city": "Seoul"}

# 빈 dict
empty = {}

# dict() 함수
d = dict(name="Alice", age=30)        # 키는 문자열 변수명처럼
d = dict([("name", "Alice"), ("age", 30)])    # 튜플 리스트로
```

### 접근 / 추가 / 변경 / 삭제

```python
person = {"name": "Alice", "age": 30}

# 읽기
print(person["name"])     # "Alice"

# 추가
person["city"] = "Seoul"
print(person)
# {"name": "Alice", "age": 30, "city": "Seoul"}

# 변경 (같은 key에 새 값 → 덮어쓰기)
person["age"] = 31

# 삭제
del person["city"]

# 길이
len(person)               # 2

# 모든 key, value, items
print(person.keys())      # dict_keys(['name', 'age'])
print(person.values())    # dict_values(['Alice', 31])
print(person.items())     # dict_items([('name', 'Alice'), ('age', 31)])
```

### key 로 쓸 수 있는 것

| 가능 | 불가능 |
|---|---|
| `str`, `int`, `float`, `bool`, `tuple` (불변) | `list`, `dict`, `set` (가변) |

```python
d = {"a": 1, 100: 2, (0, 0): 3}    # OK
# d = {[1, 2]: 5}                   # TypeError ← 리스트는 불가
```

> 💡 **외워두기**: dict 의 key 는 **불변 객체만** 가능 (이래서 4장의 immutable이 중요).

## 3️⃣ 안전한 접근 — `get`, `setdefault`, `in` [↑](#toc)

### ❌ 위험 — 없는 key 접근

```python
person = {"name": "Alice"}
print(person["age"])
# KeyError: 'age'
```

### ✅ 방법 A — `get(key, 기본값)`

```python
person.get("age")           # None (기본값)
person.get("age", 0)        # 0
person.get("name", "이름없음")    # "Alice"
```

> 💡 **가장 자주 쓰는 패턴**. KeyError를 피하는 1순위.

### ✅ 방법 B — `in` 으로 먼저 확인

```python
if "age" in person:
    print(person["age"])
else:
    print("나이 정보 없음")
```

### ✅ 방법 C — `setdefault(key, 기본값)`

"있으면 그 값, 없으면 기본값 넣고 그 값" 패턴:

```python
counts = {}
for word in ["apple", "banana", "apple"]:
    counts.setdefault(word, 0)
    counts[word] += 1

print(counts)    # {'apple': 2, 'banana': 1}
```

> 💡 더 깔끔한 방법은 `defaultdict` (다음 챕터에서 다룸).

## 4️⃣ dict 순회 패턴 [↑](#toc)

### key 만 (기본)

```python
person = {"name": "Alice", "age": 30, "city": "Seoul"}
for key in person:
    print(key)
# name / age / city
```

### value 만

```python
for val in person.values():
    print(val)
# Alice / 30 / Seoul
```

### key 와 value 함께 — `items()` (가장 자주 씀)

```python
for key, value in person.items():
    print(f"{key}: {value}")
# name: Alice
# age: 30
# city: Seoul
```

> 💡 **외워두기**: dict 순회는 **`.items()` 거의 항상**. f-string과 함께 보기 좋은 출력.

## 5️⃣ 흔한 관용구 5선 [↑](#toc)

### 1. dict comprehension

```python
nums = [1, 2, 3, 4]
squares = {n: n ** 2 for n in nums}
print(squares)    # {1: 1, 2: 4, 3: 9, 4: 16}

# 조건도 가능
even_squares = {n: n ** 2 for n in nums if n % 2 == 0}
print(even_squares)    # {2: 4, 4: 16}
```

### 2. 개수 세기 (카운팅) — 3가지 방법

```python
words = ["apple", "banana", "apple", "cherry", "apple", "banana"]

# 방법 A — get
counts = {}
for w in words:
    counts[w] = counts.get(w, 0) + 1
print(counts)    # {'apple': 3, 'banana': 2, 'cherry': 1}

# 방법 B — defaultdict
from collections import defaultdict
counts = defaultdict(int)
for w in words:
    counts[w] += 1

# 방법 C — Counter (가장 간단!)
from collections import Counter
counts = Counter(words)
print(counts)              # Counter({'apple': 3, 'banana': 2, 'cherry': 1})
print(counts.most_common(2))    # [('apple', 3), ('banana', 2)]
```

### 3. 두 리스트로 dict 만들기 — `zip`

```python
keys = ["name", "age", "city"]
values = ["Alice", 30, "Seoul"]
d = dict(zip(keys, values))
print(d)    # {'name': 'Alice', 'age': 30, 'city': 'Seoul'}
```

### 4. dict 합치기

```python
a = {"x": 1, "y": 2}
b = {"y": 99, "z": 3}

# Python 3.9+ (가장 간결)
merged = a | b
print(merged)    # {'x': 1, 'y': 99, 'z': 3}  ← b 가 우선

# 그 이전 버전
merged = {**a, **b}
```

### 5. value 기준 정렬

```python
scores = {"Alice": 85, "Bob": 92, "Carol": 78}

# 점수 높은 순
sorted_items = sorted(scores.items(), key=lambda x: -x[1])
print(sorted_items)
# [('Bob', 92), ('Alice', 85), ('Carol', 78)]
```

## 6️⃣ set 기본과 집합 연산 [↑](#toc)

### 생성

```python
nums = {1, 2, 3, 4}
empty = set()              # 빈 set은 set() ← {} 는 빈 dict!

# 리스트에서 변환
unique = set([1, 2, 2, 3, 3, 3])
print(unique)              # {1, 2, 3} ← 중복 제거됨
```

### 기본 연산

```python
s = {1, 2, 3}
s.add(4)                   # {1, 2, 3, 4}
s.remove(2)                # {1, 3, 4} — 없으면 KeyError
s.discard(99)              # 없어도 에러 안 남
3 in s                     # True
len(s)                     # 3
```

### 집합 연산 (수학 그대로)

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

a | b      # 합집합 {1, 2, 3, 4, 5, 6}
a & b      # 교집합 {3, 4}
a - b      # 차집합 {1, 2}
a ^ b      # 대칭차 {1, 2, 5, 6}

# 메서드 형태도 OK
a.union(b)
a.intersection(b)
a.difference(b)
```

> 💡 비유: 두 동아리의 회원 명단 → 둘 다 가입(교집합), A에만 가입(차집합) 같은 분석.

## 7️⃣ set 의 단골 용처: 중복 제거 [↑](#toc)

```python
# ❌ 비효율 — 직접 루프
unique = []
for x in [1, 2, 2, 3, 3, 3]:
    if x not in unique:
        unique.append(x)
print(unique)    # [1, 2, 3]

# ✅ Pythonic — set 한 줄
unique = list(set([1, 2, 2, 3, 3, 3]))
print(unique)    # [1, 2, 3]  (순서는 보장 X)

# ✅ 순서 유지하면서 중복 제거 (Python 3.7+)
unique = list(dict.fromkeys([1, 2, 2, 3, 3, 3]))
print(unique)    # [1, 2, 3]  (순서 유지)
```

### ⚠️ 흔한 실수

```python
# ❌ 빈 set 만들기
empty = {}            # ← 이건 dict!
print(type(empty))    # <class 'dict'>

# ✅
empty = set()
print(type(empty))    # <class 'set'>
```

words = ["apple", "banana", "apple", "cherry", "apple"]

common = set(group_a) & set(group_b)

## 정리 카드

```
dict      = {"key": "value"}    → 라벨 붙은 사물함
set       = {1, 2, 3}           → 회원 명단(중복 X)
빈 dict   = {}        빈 set = set()  ← 헷갈리지 말 것

dict 안전 접근  = d.get(key, 기본값) / if key in d
dict 순회       = for k, v in d.items()
카운팅          = Counter(words)
중복 제거       = list(set(items)) — 순서 보장 X
                  list(dict.fromkeys(items)) — 순서 유지

집합 연산  = | (합) & (교) - (차) ^ (대칭차)
```
