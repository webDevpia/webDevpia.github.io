## 1️⃣ Python의 들여쓰기 규칙 [↑](#toc)

> 다른 언어는 `{ }` 로 블록을 묶지만, **Python은 들여쓰기로 묶습니다**.

```python
# Python
if score >= 80:
    print("합격")        # 들여쓰기 = 블록 안
    print("축하해요")

# JavaScript / Java 등
if (score >= 80) {
    console.log("합격");
    console.log("축하해요");
}
```

| 규칙 | 설명 |
|---|---|
| **스페이스 4칸** | 가장 표준 (PEP 8) |
| 일관성 | 한 파일에서 섞으면 안 됨 (탭+스페이스 혼용 ❌) |
| 콜론 `:` | `if`, `for`, `def`, `class` 끝에 |

### ⚠️ 흔한 에러

```python
if x > 0:
print(x)        # IndentationError: expected an indented block
```

```python
if x > 0:
    print(x)
   print(x)     # IndentationError: 들여쓰기 불일치
```

> 💡 VS Code · Colab 모두 자동 들여쓰기 지원. 직접 4칸 치는 일은 드뭄.

## 2️⃣ 조건문 `if / elif / else` [↑](#toc)

```python
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"

print(grade)    # B
```

### 한 줄 if (삼항 연산자)

```python
status = "성인" if age >= 18 else "미성년"
result = "양수" if n > 0 else "음수" if n < 0 else "0"
```

### Pythonic 한 조건 작성법

```python
# 범위 비교 — 수학식처럼
if 0 < x < 10:        # JavaScript는 x > 0 && x < 10
    print("한자릿수 양수")

# 멤버십
if name in ["Alice", "Bob", "Carol"]:
    print("회원입니다")

# truthy / falsy
my_list = []
if my_list:                 # 빈 리스트는 False
    print("리스트에 뭔가 있음")
else:
    print("리스트 비었음")    # 이게 출력
```

### falsy 값 정리

| 값 | bool 변환 |
|---|---|
| `False`, `None`, `0`, `0.0` | `False` |
| `""`, `[]`, `{}`, `()`, `set()` | `False` |
| 그 외 모든 값 | `True` |

```python
if not my_list:           # "리스트가 비었으면" — Pythonic
    print("비었음")

# 같은 의미지만 길게
if len(my_list) == 0:
    print("비었음")
```

## 3️⃣ for 반복문 [↑](#toc)

Python의 `for` 는 **iterable 의 원소를 하나씩 꺼냅니다**. C/Java의 `for(i=0; i<n; i++)` 와 다름.

```python
# 리스트 순회
for fruit in ["apple", "banana", "cherry"]:
    print(fruit)

# 문자열 순회 (한 글자씩)
for ch in "hi":
    print(ch)
# h
# i

# dict 순회 (key)
for key in {"a": 1, "b": 2}:
    print(key)

# dict 순회 (key, value)
for k, v in {"a": 1, "b": 2}.items():
    print(k, v)
```

## 4️⃣ `range`, `enumerate`, `zip` 3대 도우미 [↑](#toc)

### `range(시작, 끝, 간격)` — 끝 불포함

```python
list(range(5))          # [0, 1, 2, 3, 4]
list(range(1, 5))       # [1, 2, 3, 4]
list(range(0, 10, 2))   # [0, 2, 4, 6, 8]
list(range(10, 0, -1))  # [10, 9, 8, ..., 1]

# 가장 흔한 패턴
for i in range(5):
    print(i)            # 0 1 2 3 4
```

> ⚠️ `range(5)` 는 0~4 (총 5개). **끝값은 불포함**.

### `enumerate(iterable)` — 인덱스 + 값 동시에

```python
fruits = ["apple", "banana", "cherry"]

# ❌ 옛 스타일 (안 권장)
for i in range(len(fruits)):
    print(i, fruits[i])

# ✅ Pythonic
for i, fruit in enumerate(fruits):
    print(i, fruit)
# 0 apple
# 1 banana
# 2 cherry

# 1부터 시작도 가능
for i, fruit in enumerate(fruits, start=1):
    print(i, fruit)
# 1 apple ...
```

> 💡 **외워두기**: "for + 인덱스" 가 필요하면 99%는 `enumerate`.

### `zip(a, b, ...)` — 두 리스트를 짝지어

```python
names = ["Alice", "Bob", "Carol"]
ages = [30, 25, 28]

for name, age in zip(names, ages):
    print(f"{name} - {age}")
# Alice - 30
# Bob - 25
# Carol - 28

# 3개 이상도 OK
for n, a, c in zip(names, ages, ["Seoul", "Busan", "Daegu"]):
    print(n, a, c)
```

> ⚠️ 길이가 다르면 **짧은 쪽 기준**으로 잘림. 긴 쪽까지 가려면 `from itertools import zip_longest`.

### 셋 다 조합

```python
names = ["Alice", "Bob"]
scores = [85, 92]

for i, (name, score) in enumerate(zip(names, scores), start=1):
    print(f"{i}등 후보: {name} ({score}점)")
```

## 5️⃣ while 반복문 [↑](#toc)

> "**조건이 참인 동안** 반복". 횟수가 정해지지 않은 상황에 사용.

```python
# 사용자 입력을 받아 종료할 때까지
while True:
    cmd = input("명령 (q=종료): ")
    if cmd == "q":
        break
    print(f"실행: {cmd}")

# 카운트다운
count = 5
while count > 0:
    print(count)
    count -= 1
print("발사!")
```

### ⚠️ 무한 루프 함정

```python
# ❌ 변수 감소를 잊음
n = 10
while n > 0:
    print(n)
    # n -= 1  ← 이게 없으면 영원히 10 출력
```

Colab/Jupyter 에서는 좌측 ■ 버튼으로 강제 중단.

### for vs while 선택 기준

| 상황 | 권장 |
|---|---|
| 횟수를 안다 (5번 / 리스트 끝까지) | `for` |
| 조건이 충족될 때까지 | `while` |
| 사용자 입력을 계속 받음 | `while` |

## 6️⃣ break / continue [↑](#toc)

| | `break` | `continue` |
|---|---|---|
| 의미 | 루프 **완전 종료** | **현재 회차만 건너뜀** |
| 비유 | 영화관에서 나감 | 이번 광고만 건너뜀 |

```python
for n in range(10):
    if n == 5:
        break              # 5를 만나면 루프 종료
    print(n, end=" ")
# 0 1 2 3 4

for n in range(10):
    if n % 2 == 0:
        continue           # 짝수는 건너뛰고 다음 회차
    print(n, end=" ")
# 1 3 5 7 9
```

### 실전 패턴: 첫 일치 찾기

```python
target = "banana"
fruits = ["apple", "banana", "cherry"]

for f in fruits:
    if f == target:
        print(f"찾음! 위치는 알 수 없지만 있음")
        break
else:
    # for-else: break 없이 끝났을 때만 실행
    print("못 찾음")
```

> 💡 `for-else` 는 잘 안 쓰지만 알아두면 유용. **`break` 없이 끝났을 때 `else` 실행**.

## 7️⃣ 자기 점검 5문항 [↑](#toc)

### Q1

```python
for i in range(1, 11, 3):
    print(i, end=" ")
```
출력은?

<details><summary>정답</summary>
`1 4 7 10`. 1부터 11 미만까지 3씩 증가.
</details>

### Q2

다음 코드를 `enumerate` 로 바꾸세요.

```python
fruits = ["apple", "banana", "cherry"]
for i in range(len(fruits)):
    print(i, fruits[i])
```

<details><summary>정답</summary>
```python
for i, fruit in enumerate(fruits):
    print(i, fruit)
```
</details>

### Q3

두 리스트 `names = ["A","B","C"]` 와 `scores = [80, 90, 70]` 을 짝지어 출력하세요.

<details><summary>정답</summary>
```python
for name, score in zip(names, scores):
    print(name, score)
```
</details>

### Q4

다음 코드의 출력은?

```python
nums = [1, 2, 3, 4, 5, 6, 7, 8]
for n in nums:
    if n % 2 == 0:
        continue
    if n > 5:
        break
    print(n, end=" ")
```

<details><summary>정답</summary>
`1 3 5`. 짝수는 건너뜀 (`continue`), 5 초과 홀수(7)에서 멈춤 (`break`).
</details>

### Q5

`numbers = [10, 25, 7, 89, 31]` 에서 30보다 큰 첫 번째 수를 찾아 출력하세요. 없으면 `"없음"`.

<details><summary>정답</summary>
```python
numbers = [10, 25, 7, 89, 31]
for n in numbers:
    if n > 30:
        print(n)
        break
else:
    print("없음")
# 출력: 89
```
</details>

## 정리 카드

```
들여쓰기   = 블록 구분. 스페이스 4칸 표준.
if-elif-else = 콜론 + 들여쓰기

for 변수 in 반복가능:    # list, str, range, dict.items()
range(시작, 끝, 간격)    # 끝 불포함
enumerate(seq)           # 인덱스+값
zip(a, b)                # 짝지어 순회

while 조건:              # 횟수 모를 때
break    = 루프 종료
continue = 이번 회차만 건너뜀
for-else = break 없이 끝났을 때 else
```
