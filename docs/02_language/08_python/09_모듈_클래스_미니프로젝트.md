## 1️⃣ 모듈과 import [↑](#toc)

> 모듈 = **`.py` 파일 하나**. 다른 파일에서 가져다 쓸 수 있음.

### 3가지 import 방식

```python
# 1. 전체 모듈 — 모듈명.이름 으로 사용
import math
print(math.pi)              # 3.14159...
print(math.sqrt(16))        # 4.0

# 2. 특정 이름만
from math import pi, sqrt
print(pi)                   # math. 안 붙임
print(sqrt(16))

# 3. 별칭 (긴 이름 줄이기)
import numpy as np          # 관행
import pandas as pd
import matplotlib.pyplot as plt
```

### ❌ 피해야 할 패턴

```python
# ❌ 와일드카드 import — 이름 충돌 위험
from math import *
print(pi)         # OK
# 하지만 다른 변수와 충돌 가능

# ✅ 명시적으로
from math import pi, sqrt
```

### 내가 만든 모듈

```
my_project/
├── main.py
└── utils.py    ← 함수 정의 파일
```

**utils.py**:
```python
def add(a, b):
    return a + b
```

**main.py**:
```python
from utils import add
print(add(3, 4))    # 7
```

## 2️⃣ 자주 쓰는 표준 라이브러리 [↑](#toc)

> Python 설치하면 함께 따라오는 모듈들. 외부 설치 불필요.

### `math` — 수학

```python
import math
math.pi              # 3.14159...
math.sqrt(2)         # 1.414...
math.log(100, 10)    # 2.0
math.ceil(1.2)       # 2
math.floor(1.8)      # 1
```

### `random` — 난수

```python
import random
random.random()                 # 0.0 ~ 1.0
random.randint(1, 6)            # 1~6 정수 (양 끝 포함)
random.choice([1, 2, 3])        # 리스트에서 하나
random.sample([1,2,3,4,5], 2)   # 2개 골라 리스트
random.shuffle(my_list)         # 제자리 섞기
```

### `datetime` — 날짜/시간

```python
from datetime import datetime, timedelta
now = datetime.now()
print(now)                      # 2026-05-13 14:30:45.123
print(now.strftime("%Y-%m-%d")) # 2026-05-13

# 시간 차이
tomorrow = now + timedelta(days=1)
```

### `os` / `pathlib` — 파일 시스템

```python
import os
os.listdir(".")              # 현재 폴더 파일 목록
os.path.exists("data.txt")    # True/False

# 더 모던한 방식 (권장)
from pathlib import Path
p = Path("data.txt")
p.exists()
p.read_text(encoding="utf-8")
```

### `json` — JSON 처리

```python
import json

# dict → JSON 문자열
d = {"name": "Alice", "age": 30}
s = json.dumps(d, ensure_ascii=False, indent=2)
print(s)

# JSON 문자열 → dict
parsed = json.loads(s)
print(parsed["name"])

# 파일 입출력
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(d, f, ensure_ascii=False, indent=2)
```

### `collections` — 자주 쓰는 자료구조

```python
from collections import Counter, defaultdict

# 가장 자주 등장
Counter(["a", "b", "a", "c", "a"])
# Counter({'a': 3, 'b': 1, 'c': 1})

# KeyError 없는 dict
dd = defaultdict(int)
dd["new"] += 1     # 자동으로 0 으로 시작
```

## 3️⃣ 클래스 — 왜 필요한가 [↑](#toc)

> 한 개념을 **데이터 + 동작**으로 묶기.

### dict 만으로도 사람을 표현

```python
# 학생 한 명 = dict
student1 = {"name": "Alice", "score": 85}
student2 = {"name": "Bob", "score": 92}

# 동작은 함수로 분리
def grade(student):
    if student["score"] >= 90:
        return "A"
    elif student["score"] >= 80:
        return "B"
    return "F"

print(grade(student1))    # B
```

### 같은 일을 클래스로

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def grade(self):
        if self.score >= 90:
            return "A"
        elif self.score >= 80:
            return "B"
        return "F"

alice = Student("Alice", 85)
print(alice.grade())    # B
```

### 차이점

| | dict + 함수 | 클래스 |
|---|---|---|
| 데이터·동작 결합 | 분리 | **결합** |
| 가독성 | 단순 데이터에 OK | 복잡한 개념에 유리 |
| 확장 | 함수 추가만 | 메서드·상속 등 다양 |

> 💡 **언제 클래스?** 데이터에 **여러 동작**이 따라붙거나, 같은 형태의 인스턴스를 **여러 개** 만들 때.

## 4️⃣ 클래스 기본 [↑](#toc)

```python
class 클래스이름:
    def __init__(self, 매개변수):
        self.속성 = 매개변수

    def 메서드이름(self, 매개변수):
        # self.속성 사용 가능
        ...
```

### 예시

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name}: 멍멍!")

    def info(self):
        return f"{self.name} ({self.age}살)"

# 인스턴스 생성
my_dog = Dog("Buddy", 3)

# 속성 접근
print(my_dog.name)      # Buddy
print(my_dog.age)       # 3

# 메서드 호출
my_dog.bark()           # Buddy: 멍멍!
print(my_dog.info())    # Buddy (3살)

# 여러 인스턴스
your_dog = Dog("Max", 5)
your_dog.bark()         # Max: 멍멍!
```

> 💡 클래스는 **틀(설계도)**, 인스턴스는 그 틀로 만든 **구체적 객체**.

## 5️⃣ `__init__` 과 `self` 깊이 이해 [↑](#toc)

### `__init__` 은 생성자

- 이름 앞뒤 더블 언더스코어(`dunder`)
- **인스턴스를 만드는 순간 자동 호출**
- 인스턴스의 초기 상태(속성)를 설정

```python
class Counter:
    def __init__(self):
        self.count = 0     # 초기값

c = Counter()              # __init__ 자동 호출
print(c.count)             # 0
```

### `self` 는 "**이 인스턴스 자신**"

```python
class Dog:
    def __init__(self, name):
        self.name = name      # self.name = 이 인스턴스의 name

    def introduce(self):
        return f"저는 {self.name}입니다"

d1 = Dog("Buddy")
d2 = Dog("Max")
print(d1.introduce())     # 저는 Buddy입니다
print(d2.introduce())     # 저는 Max입니다
```

각 인스턴스는 **자기만의 `name`** 을 가짐. `self` 가 "어느 인스턴스인지" 알려줌.

### ⚠️ 흔한 실수 — `self` 누락

```python
class Dog:
    def __init__(name):           # self 누락!
        self.name = name          # NameError 또는 이상한 동작

    def bark():                   # self 누락
        print("멍멍")
```

> 💡 **외워두기**: **메서드의 첫 매개변수는 항상 `self`**. 호출할 땐 자동 전달되므로 신경 X (`d.bark()` 처럼).

### 호출 시 `self` 가 어디서 오는가?

```python
my_dog = Dog("Buddy")
my_dog.bark()              # 사실은 Dog.bark(my_dog) 와 동일
```

`인스턴스.메서드()` 는 내부적으로 `클래스.메서드(인스턴스)` 로 변환.

## 6️⃣ 상속과 메서드 오버라이드 (개념만) [↑](#toc)

> 비슷한 클래스를 **공통 부분**으로 묶기.

```python
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print(f"{self.name}: ...")    # 기본 동작

class Dog(Animal):                     # Animal을 상속
    def speak(self):                   # 오버라이드 (재정의)
        print(f"{self.name}: 멍멍!")

class Cat(Animal):
    def speak(self):
        print(f"{self.name}: 야옹~")

dogs = [Dog("Buddy"), Cat("Kitty"), Animal("Unknown")]
for animal in dogs:
    animal.speak()
# Buddy: 멍멍!
# Kitty: 야옹~
# Unknown: ...
```

> 💡 **수업 범위는 "읽을 줄 알기"** 까지. 직접 설계는 후속 OOP 강의에서.

## 7️⃣ 미니 프로젝트: 학생 성적 관리 [↑](#toc)

8시간 학습 내용을 **모두 합쳐서** 만드는 작은 프로그램.

### 요구사항

1. 여러 학생의 이름·점수를 저장
2. 학생 추가/삭제 가능
3. 평균·최고·최저 점수 출력
4. 점수 순 정렬 출력
5. 파일에 저장/불러오기 (JSON)

### 단계별 작성

#### Step 1 — `Student` 클래스 (속성·메서드)

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def grade(self):
        if self.score >= 90:
            return "A"
        elif self.score >= 80:
            return "B"
        elif self.score >= 70:
            return "C"
        return "F"

    def __repr__(self):
        return f"Student({self.name}, {self.score}, {self.grade()})"
```

> 💡 `__repr__` 은 `print()` 했을 때 보기 좋게 만드는 메서드.

#### Step 2 — `GradeBook` 클래스 (학생 모음 관리)

```python
class GradeBook:
    def __init__(self):
        self.students = []        # 학생 리스트

    def add(self, name, score):
        self.students.append(Student(name, score))

    def remove(self, name):
        self.students = [s for s in self.students if s.name != name]

    def average(self):
        if not self.students:
            return 0
        return sum(s.score for s in self.students) / len(self.students)

    def top(self):
        if not self.students:
            return None
        return max(self.students, key=lambda s: s.score)

    def bottom(self):
        if not self.students:
            return None
        return min(self.students, key=lambda s: s.score)

    def ranking(self):
        return sorted(self.students, key=lambda s: -s.score)
```

#### Step 3 — JSON 저장/불러오기

```python
import json

class GradeBook:
    # ... 위와 동일 ...

    def save(self, path):
        data = [{"name": s.name, "score": s.score} for s in self.students]
        with open(path, "w", encoding="utf-8") as f:
            json.dump(data, f, ensure_ascii=False, indent=2)

    def load(self, path):
        try:
            with open(path, "r", encoding="utf-8") as f:
                data = json.load(f)
            self.students = [Student(d["name"], d["score"]) for d in data]
        except FileNotFoundError:
            print(f"파일 없음: {path}. 빈 채로 시작합니다.")
```

#### Step 4 — 실제 사용

```python
gb = GradeBook()
gb.add("Alice", 85)
gb.add("Bob", 92)
gb.add("Carol", 78)
gb.add("Dave", 65)

print("평균:", gb.average())          # 80.0
print("1등:", gb.top())                # Student(Bob, 92, A)
print("꼴등:", gb.bottom())            # Student(Dave, 65, F)

print("\n순위표:")
for rank, s in enumerate(gb.ranking(), start=1):
    print(f"{rank}. {s.name} ({s.score}, {s.grade()})")

# 파일에 저장
gb.save("students.json")

# 새 GradeBook에서 불러오기
gb2 = GradeBook()
gb2.load("students.json")
print("\n불러온 데이터:")
print(gb2.ranking())
```

### 학습 포인트 매핑

| 챕터 | 미니프로젝트에서 어디? |
|---|---|
| 02 변수·타입 | `score`, `name` 변수 |
| 03 문자열 | f-string |
| 04 리스트 | `self.students` |
| 05 dict | JSON 저장 시 |
| 06 조건문 | `grade()` 메서드 |
| 06 반복문 | `for rank, s in enumerate(...)` |
| 07 함수 | 메서드 모두 |
| 07 lambda | `sorted(..., key=lambda s: -s.score)` |
| 08 컴프리헨션 | `[s for s in ... if s.name != name]` |
| 08 파일 입출력 | `with open` |
| 08 예외 | `try / except FileNotFoundError` |
| 09 모듈 | `import json` |
| 09 클래스 | `Student`, `GradeBook` |

> 🎯 이걸 손으로 친 번 친다면 8시간 내용이 **하나의 그림**으로 정리됩니다.

### Q3

`Counter` 클래스를 만드세요.
- `__init__` 에서 `count=0` 초기화
- `increment()` 메서드로 1 증가
- `value()` 메서드로 현재 값 반환

<details><summary>정답</summary>
```python
class Counter:
    def __init__(self):
        self.count = 0

    def increment(self):
        self.count += 1

    def value(self):
        return self.count

c = Counter()
c.increment(); c.increment(); c.increment()
print(c.value())    # 3
```
</details>

### Q4

random.seed(42)

## 정리 카드

```
import math / from math import pi / import numpy as np
표준 라이브러리: math, random, datetime, os, pathlib, json, collections

클래스
  class Name:
      def __init__(self, ...):
          self.속성 = ...
      def 메서드(self, ...):
          ...

  __init__  = 인스턴스 생성 시 자동 호출 (생성자)
  self      = 이 인스턴스 자신 (메서드 첫 매개변수)
  인스턴스   = Name(인자들)
  속성 접근  = obj.속성
  메서드 호출 = obj.메서드(인자)

상속        class Child(Parent):
오버라이드   같은 이름의 메서드 재정의
__repr__    print() 시 보기 좋게
```
