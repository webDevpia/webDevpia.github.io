# 사전 자가진단을 Google Form으로 만들기

`create_diagnostic_form.gs` 스크립트를 한 번 실행하면 **10문항 + 이름/이메일 + 관심 챕터 조사** 가 포함된 퀴즈형 Google Form이 자동 생성됩니다.

> ⏱️ 전체 약 3~5분 소요.

---

## 🎯 1단계 — Apps Script 사이트 접속

브라우저(Chrome 권장)에서 주소창에 입력 후 이동:

```
https://script.google.com
```

> 로그인 안 돼 있으면 Google 계정 로그인 창이 뜹니다. 본인 Gmail로 로그인하세요.

---

## 🎯 2단계 — 새 프로젝트 만들기

화면 좌측 상단의 파란색 **`+ 새 프로젝트`** 버튼 클릭.

→ 코드 편집 화면이 열립니다. 좌측 코드 영역에 이런 코드가 미리 있습니다.

```javascript
function myFunction() {

}
```

---

## 🎯 3단계 — 기존 코드 모두 지우기

좌측 코드 영역 안을 클릭한 뒤:

- **Windows**: `Ctrl + A` → `Delete`
- **Mac**: `Cmd + A` → `Delete`

빈 화면이 됩니다.

---

## 🎯 4단계 — 스크립트 파일 내용 복사

지금 푸시한 파일을 GitHub에서 열어 복사하세요:

```
https://github.com/shimseonjo/shimseonjo.github.io/blob/main/docs/02_language/08_python/_setup/create_diagnostic_form.gs
```

또는 로컬 저장소에서:

```
docs/02_language/08_python/_setup/create_diagnostic_form.gs
```

파일을 열어 **전체 내용**(`/**` 부터 마지막 `}` 까지) 을 복사.

---

## 🎯 5단계 — Apps Script에 붙여넣기

방금 비운 코드 영역에 붙여넣기:

- **Windows**: `Ctrl + V`
- **Mac**: `Cmd + V`

---

## 🎯 6단계 — 프로젝트 이름 정하기 (선택)

상단 중앙의 **"제목 없는 프로젝트"** 글자 클릭 → 적당한 이름 입력:

```
Python 자가진단 폼
```

→ Enter.

---

## 🎯 7단계 — 저장

- **Windows**: `Ctrl + S`
- **Mac**: `Cmd + S`

또는 코드 영역 위쪽의 💾 **저장** 아이콘 클릭.

> 저장 후엔 상단 함수 선택 드롭다운에 `createPythonDiagnosticForm` 이 나타납니다.

---

## 🎯 8단계 — 함수 선택 + 실행

코드 영역 바로 위 가운데 줄에:

```
 ▶ 실행   디버그   함수 선택 [____________]   실행 로그
```

**[함수 선택]** 드롭다운을 클릭 → `createPythonDiagnosticForm` 선택.

그 다음 좌측의 **▶ 실행** 버튼 클릭.

---

## 🎯 9단계 — 권한 허용 (처음 1회만)

다음 창들이 차례로 뜹니다.

### 9-1. "승인 필요" 창
하단의 **[권한 검토]** 클릭.

### 9-2. "계정 선택" 창
본인 Google 계정 클릭.

### 9-3. "Google에서 확인하지 않은 앱입니다" 경고
- 이 경고는 **본인이 직접 만든 스크립트이기 때문에** 뜨는 일반 안내입니다.
- 좌측 하단의 **[고급]** 글자 클릭 *(작아서 잘 안 보입니다)*.
- 펼쳐진 부분의 **[{프로젝트 이름}(으)로 이동(안전하지 않음)]** 클릭.

### 9-4. "이 앱이 다음에 액세스하려고 합니다" 창
스크롤을 내려 우측 하단의 **[허용]** 클릭.

---

## 🎯 10단계 — 실행 로그 확인

권한 허용 후 자동으로 스크립트가 다시 실행됩니다. 잠시(약 5~10초) 후 화면 **하단**에 **[실행 로그]** 패널이 펼쳐집니다.

## 🎯 11단계 — URL 활용

| URL | 용도 |
|---|---|
| **단축 URL** (`forms.gle/...`) | 학생에게 공유. 카톡·이메일·QR로 보내기 좋음 |
| **응답 URL** (`viewform` 끝남) | 단축 못 쓸 때 사용하는 풀 URL |
| **편집 URL** (`edit` 끝남) | 강사가 문구·디자인 손보기, 응답 확인 |

---

## 🎯 12단계 — 응답을 스프레드시트로 받기 (강력 추천)

학생 결과를 한눈에 보고 정답률을 분석하려면:

1. **편집 URL** 클릭해서 폼 편집 화면 열기
2. 상단 가운데의 **[응답]** 탭 클릭
3. 우측의 **초록색 스프레드시트 아이콘** (📊) 클릭
4. **[새 스프레드시트 만들기]** 선택 → 이름 정하기 → **[만들기]**

이제 학생이 응답할 때마다 한 행씩 자동으로 추가됩니다.

## ❓ 자주 막히는 부분

### "Google에서 확인하지 않은 앱" 창에서 [고급] 글자가 안 보임
화면을 좀 더 키우거나 스크롤하세요. 회색의 작은 글씨로 좌측 하단에 있습니다.

### 실행 후 아무 일도 안 일어남
- 함수 선택이 `createPythonDiagnosticForm` 인지 확인 (드롭다운에 보여야 함)
- 만약 함수 목록에 안 보이면: 저장(Ctrl+S) 다시 → 페이지 새로고침

### 권한 화면이 자꾸 다시 뜸
**시크릿/InPrivate 모드** 가 아닌 일반 브라우저 창에서 시도하세요. 회사·학교 Google 계정이면 관리자가 차단했을 수 있으니 개인 Gmail 사용.

### 실행 로그에 URL이 안 보임
실행 로그가 닫혀 있을 수 있습니다.
- 단축키: `Ctrl + Enter` (Mac: `Cmd + Enter`)
- 또는 화면 우측 하단의 **[실행 로그]** 아이콘 클릭

### `setIsQuiz is not a function` 같은 에러
좌측 ⚙️ **[프로젝트 설정]** → "Chrome V8 런타임 사용" 체크.

### URL 단축이 안 됨
스크립트의 `form.shortenFormUrl(publishedUrl)` 라인을 주석 처리하고 publishedUrl만 사용. (드물게 지역 제한 있음)

### 폼을 다시 만들고 싶음
이전 폼은 Google Drive에 남아 있습니다. 스크립트를 한 번 더 실행하면 **새 폼이 추가로** 생성됩니다. 기존 폼은 Drive에서 수동 삭제.

## 📋 생성되는 폼 구성 요약

| # | 형식 | 자동 채점 | 비고 |
|---|---|---|---|
| 이름 | 단답 | — | 필수, 식별용 |
| Q1 변수·타입 | 객관식 | ✅ | 정답: 에러 |
| Q2 == vs is | 장문 | ❌ (수동) | 모범답안·해설 자동 표시 |
| Q3 슬라이싱 | 장문 | ❌ | 모범답안 표시 |
| Q4 mutability | 객관식 | ✅ | 정답: [1,2,3,4] |
| Q5 KeyError | 단답 | ❌ | 정답·해설 표시 |
| Q6 range | 단답 | ❌ | 정답·해설 표시 |
| Q7 return | 단답 | ❌ | 정답: None |
| Q8 가변 기본값 | 객관식 | ✅ | 정답: ["apple","banana"] |
| Q9 컴프리헨션 | 단답 | ❌ | 정답: [4,8] |
| Q10 try/except | 장문 | ❌ | 모범답안 표시 |
| 관심 챕터 | 객관식 | — | 강의 비중 조정용 |

> 💡 Google Forms 단답형의 자동 채점은 **정확 일치**만 지원합니다. "[4, 8]" vs "[4,8]" 같은 사소한 차이로 오답 처리되는 걸 피하려고 단답·장문은 **수동 채점**으로 설정했습니다. 강사 화면(응답 탭 → 개인 응답)에서 한 명씩 보며 점수 부여하시면 됩니다.
