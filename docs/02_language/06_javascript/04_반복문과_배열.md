# 04장. 반복문과 배열

<a id="part1"></a>

## 1️⃣ 반복문이 필요한 이유 [↑](#toc)

프로그래밍을 하다 보면 **같은 작업을 여러 번 반복**해야 할 때가 많습니다.
반복문이 없다면, 같은 코드를 일일이 복사해서 써야 합니다.

### 출석부 호명 비유

> 선생님이 30명의 학생에게 인사할 때,
> 한 명씩 이름을 불러도 되지만 **"1번부터 30번까지 순서대로 호명"**하면 훨씬 빠릅니다.
> 반복문은 바로 이 "자동 호명" 기능입니다.

```javascript
// 반복문 없이 — 같은 코드를 30번 써야 함
console.log("안녕하세요, 1번 학생!");
console.log("안녕하세요, 2번 학생!");
console.log("안녕하세요, 3번 학생!");
// ... 27번 더 써야 한다
```

실행 결과:
```
안녕하세요, 1번 학생!
안녕하세요, 2번 학생!
안녕하세요, 3번 학생!
```

```javascript
// 반복문 사용 — 3줄로 30번 반복
for (let i = 1; i <= 30; i++) {
    console.log(`안녕하세요, ${i}번 학생!`);
}
```

실행 결과:
```
안녕하세요, 1번 학생!
안녕하세요, 2번 학생!
...
안녕하세요, 30번 학생!
```

코드가 30줄에서 3줄로 줄었습니다. 이것이 반복문의 힘입니다.

## 2️⃣ for 반복문 [↑](#toc)

**for 반복문**은 반복 횟수가 정해져 있을 때 가장 많이 사용하는 반복문입니다.

### 기본 구조

```javascript
for (초기값; 조건; 증감) {
    // 반복할 코드
}
```

| 부분 | 역할 | 예시 |
|------|------|------|
| 초기값 | 반복 시작 전 딱 한 번 실행 | `let i = 1` |
| 조건 | 매 반복 전에 확인, false이면 종료 | `i <= 10` |
| 증감 | 매 반복 후 실행 | `i++` |

### 실행 흐름

```
① 초기값 설정 (let i = 1)
② 조건 확인 (i <= 10) → true면 실행, false면 종료
③ 코드 블록 실행
④ 증감 (i++)
⑤ ② 로 돌아가서 반복
```

```javascript
// 1부터 10까지 합계 계산
let sum = 0;

for (let i = 1; i <= 10; i++) {
    sum += i;  // sum = sum + i
}

console.log(`1부터 10까지의 합: ${sum}`);
```

실행 결과:
```
1부터 10까지의 합: 55
```

```javascript
// 구구단 3단 출력
for (let i = 1; i <= 9; i++) {
    console.log(`3 × ${i} = ${3 * i}`);
}
```

실행 결과:
```
3 × 1 = 3
3 × 2 = 6
3 × 3 = 9
3 × 4 = 12
3 × 5 = 15
3 × 6 = 18
3 × 7 = 21
3 × 8 = 24
3 × 9 = 27
```

### break — 비상 탈출

**break**는 반복문을 즉시 빠져나올 때 사용합니다.

> 비유: 엘리베이터를 타고 내려가다가 원하는 층에 도착하면 **비상 탈출** 버튼을 눌러 멈추는 것.

```javascript
// 1부터 순서대로 숫자를 찾다가 5를 만나면 즉시 중단
for (let i = 1; i <= 10; i++) {
    if (i === 5) {
        console.log("5를 찾았습니다! 반복 종료.");
        break;  // 반복문 즉시 탈출
    }
    console.log(i);
}
```

실행 결과:
```
1
2
3
4
5를 찾았습니다! 반복 종료.
```

### continue — 이번만 건너뛰기

**continue**는 현재 반복만 건너뛰고 다음 반복으로 넘어갑니다.

> 비유: 출석부를 호명하다가 결석한 학생이 나오면 **이번만 건너뛰고** 다음 학생으로 넘어가는 것.

```javascript
// 1부터 10까지 중 홀수만 출력 (짝수는 건너뛰기)
for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        continue;  // 짝수이면 이번 반복 건너뜀
    }
    console.log(i);
}
```

실행 결과:
```
1
3
5
7
9
```

## 3️⃣ while 반복문 [↑](#toc)

**while 반복문**은 **조건이 참(true)인 동안** 계속 반복합니다.
반복 횟수를 미리 알 수 없을 때 주로 사용합니다.

### 우산 비유

> 비가 그칠 때까지 우산을 쓰고 있는 것처럼,
> **while은 조건이 false가 될 때까지 계속 반복**합니다.

```javascript
while (조건) {
    // 조건이 true인 동안 반복할 코드
}
```

```javascript
// 1부터 5까지 출력
let count = 1;

while (count <= 5) {
    console.log(`카운트: ${count}`);
    count++;  // 이 줄이 없으면 무한 루프 발생!
}
```

실행 결과:
```
카운트: 1
카운트: 2
카운트: 3
카운트: 4
카운트: 5
```

### for vs while 비교

| 구분 | for | while |
|------|-----|-------|
| 사용 시기 | 반복 횟수가 정해진 경우 | 종료 조건이 불확실한 경우 |
| 카운터 관리 | 구조 안에 포함 | 직접 관리해야 함 |
| 가독성 | 반복 횟수가 명확히 보임 | 조건 로직이 더 유연함 |

### 무한 루프 주의

조건이 영원히 `true`이면 프로그램이 멈추지 않습니다.

```javascript
// 주의: 이 코드는 무한 루프입니다 (실행하지 마세요)
let x = 1;
while (x > 0) {   // x는 항상 양수라서 조건이 영원히 true
    x++;           // 종료 조건 없음 → 프로그램 강제 종료 필요
}
```

while 반복문을 쓸 때는 **반드시 조건이 언젠가 false가 되도록** 만들어야 합니다.

### do...while — 최소 1번은 실행

**do...while**은 코드를 먼저 한 번 실행한 뒤 조건을 확인합니다.

```javascript
// 조건이 처음부터 false여도 최소 1번은 실행됨
let num = 10;

do {
    console.log(`현재 값: ${num}`);
    num--;
} while (num > 10);  // 처음부터 false지만 위 코드는 한 번 실행됨
```

실행 결과:
```
현재 값: 10
```

### 실용 예시: 랜덤 숫자 맞추기 게임

```javascript
// 1~5 사이의 숫자를 맞출 때까지 반복
const answer = 3;  // 정답 (실제로는 Math.random()으로 생성)
let guess = 0;
let tries = 0;

// 실제 게임에서는 prompt()로 입력받지만, 여기서는 시뮬레이션
const guesses = [1, 4, 3];  // 시뮬레이션용 입력값

while (guess !== answer) {
    guess = guesses[tries];
    tries++;

    if (guess === answer) {
        console.log(`정답! ${tries}번 만에 맞췄습니다.`);
    } else {
        console.log(`${guess}는 오답입니다. 다시 시도하세요.`);
    }
}
```

실행 결과:
```
1는 오답입니다. 다시 시도하세요.
4는 오답입니다. 다시 시도하세요.
정답! 3번 만에 맞췄습니다.
```

## 4️⃣ 배열(Array)이란? [↑](#toc)

**배열(Array)**은 여러 개의 값을 **순서대로** 하나의 변수에 저장하는 자료구조입니다.

### 서랍장 비유

> 배열은 **여러 칸이 있는 서랍장**입니다.
> 각 칸에는 번호(인덱스)가 붙어 있고, 칸마다 물건(값)을 넣을 수 있습니다.
> 서랍 번호는 **0번부터** 시작합니다.

```
서랍장 (fruits 배열)
┌─────────┬──────────┬──────────┐
│  0번칸  │  1번칸   │  2번칸   │
│  사과   │  바나나  │  딸기    │
└─────────┴──────────┴──────────┘
```

### 배열 생성

```javascript
// 대괄호 [] 안에 값을 쉼표로 구분하여 작성
const fruits = ["사과", "바나나", "딸기"];

console.log(fruits);         // 배열 전체 출력
console.log(fruits[0]);      // 0번 인덱스 → 첫 번째 요소
console.log(fruits[1]);      // 1번 인덱스 → 두 번째 요소
console.log(fruits[2]);      // 2번 인덱스 → 세 번째 요소
console.log(fruits.length);  // 배열의 요소 개수
```

실행 결과:
```
[ '사과', '바나나', '딸기' ]
사과
바나나
딸기
3
```

### 왜 인덱스는 0부터 시작할까?

> 인덱스는 "첫 번째 칸까지의 거리"를 나타냅니다.
> 첫 번째 칸 자체는 거리가 0이기 때문에 **0번**입니다.
> 이는 컴퓨터 메모리 주소 계산 방식에서 유래한 약속입니다.

```javascript
// 다양한 타입을 섞어서 저장할 수도 있음
const mixed = [1, "안녕", true, null];
console.log(mixed[0]);  // 1
console.log(mixed[1]);  // 안녕
```

실행 결과:
```
1
안녕
```

## 5️⃣ 배열 조작 메서드 [↑](#toc)

배열에는 요소를 추가, 삭제, 검색하는 다양한 **메서드(Method, 기능)**가 내장되어 있습니다.

| 메서드 | 설명 | 반환값 |
|--------|------|--------|
| `push()` | 배열 끝에 요소 추가 | 새 배열의 길이 |
| `pop()` | 배열 끝 요소 제거 | 제거된 요소 |
| `unshift()` | 배열 앞에 요소 추가 | 새 배열의 길이 |
| `shift()` | 배열 앞 요소 제거 | 제거된 요소 |
| `includes()` | 특정 요소 포함 여부 확인 | true / false |
| `indexOf()` | 특정 요소의 인덱스 반환 | 인덱스 (없으면 -1) |
| `splice()` | 중간 요소 삽입 또는 삭제 | 제거된 요소 배열 |

### push() — 끝에 추가

```javascript
const fruits = ["사과", "바나나", "딸기"];

fruits.push("포도");
console.log(fruits);
```

실행 결과:
```
[ '사과', '바나나', '딸기', '포도' ]
```

### pop() — 끝에서 제거

```javascript
const fruits = ["사과", "바나나", "딸기"];

const removed = fruits.pop();  // 마지막 요소를 제거하고 반환
console.log(removed);  // 제거된 요소
console.log(fruits);   // 남은 배열
```

실행 결과:
```
딸기
[ '사과', '바나나' ]
```

### unshift() / shift() — 앞에 추가/제거

```javascript
const fruits = ["사과", "바나나", "딸기"];

fruits.unshift("멜론");  // 앞에 추가
console.log(fruits);

fruits.shift();          // 앞에서 제거
console.log(fruits);
```

실행 결과:
```
[ '멜론', '사과', '바나나', '딸기' ]
[ '사과', '바나나', '딸기' ]
```

### includes() / indexOf() — 검색

```javascript
const fruits = ["사과", "바나나", "딸기"];

console.log(fruits.includes("사과"));   // 포함 여부
console.log(fruits.includes("수박"));   // 없는 요소
console.log(fruits.indexOf("바나나"));  // 위치 (인덱스)
console.log(fruits.indexOf("수박"));    // 없으면 -1 반환
```

실행 결과:
```
true
false
1
-1
```

### splice() — 중간 삽입/삭제

`splice(시작인덱스, 삭제할개수, 추가할요소)` 형태로 사용합니다.

```javascript
const fruits = ["사과", "바나나", "딸기", "포도"];

// 1번 인덱스부터 1개 삭제
fruits.splice(1, 1);
console.log(fruits);  // 바나나 삭제

// 1번 인덱스에 "멜론" 삽입 (삭제 없이)
fruits.splice(1, 0, "멜론");
console.log(fruits);
```

실행 결과:
```
[ '사과', '딸기', '포도' ]
[ '사과', '멜론', '딸기', '포도' ]
```

## 6️⃣ 반복문 + 배열 활용 [↑](#toc)

반복문과 배열을 함께 사용하면 **대량의 데이터를 효율적으로 처리**할 수 있습니다.

### for 루프로 배열 순회

```javascript
const fruits = ["사과", "바나나", "딸기", "포도"];

for (let i = 0; i < fruits.length; i++) {
    console.log(`${i}번: ${fruits[i]}`);
}
```

실행 결과:
```
0번: 사과
1번: 바나나
2번: 딸기
3번: 포도
```

### for...of — 더 간결한 순회

**for...of**는 배열의 요소를 하나씩 꺼내는 더 간결한 방법입니다.
인덱스가 필요 없을 때 사용합니다.

> 비유: 서랍장에서 물건을 하나씩 꺼내 확인하는 것처럼,
> **for...of는 배열에서 요소를 하나씩 꺼내** 사용합니다.

```javascript
const fruits = ["사과", "바나나", "딸기", "포도"];

for (const fruit of fruits) {
    console.log(fruit);
}
```

실행 결과:
```
사과
바나나
딸기
포도
```

### forEach() — 배열 전용 반복 메서드

**forEach()**는 배열의 각 요소에 대해 함수를 실행합니다.
(고차 함수에 대한 자세한 내용은 5장에서 다룹니다.)

```javascript
const fruits = ["사과", "바나나", "딸기"];

fruits.forEach((fruit, index) => {
    console.log(`${index + 1}번째: ${fruit}`);
});
```

실행 결과:
```
1번째: 사과
2번째: 바나나
3번째: 딸기
```

### 실용 예시 1: 학생 점수 평균 계산

```javascript
const scores = [85, 92, 78, 95, 60, 88, 73];

let total = 0;
for (const score of scores) {
    total += score;
}

const average = total / scores.length;
console.log(`총점: ${total}점`);
console.log(`평균: ${average.toFixed(1)}점`);  // 소수점 1자리까지
```

실행 결과:
```
총점: 571점
평균: 81.6점
```

### 실용 예시 2: 조건 필터링 (80점 이상만 출력)

```javascript
const scores = [85, 92, 78, 95, 60, 88, 73];

console.log("합격자 점수 목록:");

// for...of로 순회하며 조건 확인
for (const score of scores) {
    if (score >= 80) {
        console.log(`  ${score}점 — 합격!`);
    }
}

// forEach를 사용하면 인덱스도 함께 확인 가능
console.log("\n전체 학생 점수:");
scores.forEach((score, index) => {
```

## 7️⃣ 실용 예제: 간단한 성적 관리 프로그램 [↑](#toc)

지금까지 배운 반복문과 배열을 모두 활용한 종합 예제입니다.

```javascript
// 학생 이름과 점수 배열
const students = ["김민수", "이영희", "박지성", "최수진", "정현우"];
const scores   = [85, 92, 78, 95, 60];

// ① 전체 평균 계산
let total = 0;
for (const score of scores) {
    total += score;
}
const average = total / scores.length;
console.log(`전체 평균: ${average}점`);

// ② 최고점, 최저점 찾기
let max = scores[0];  // 일단 첫 번째 값을 최댓값으로 설정
let min = scores[0];  // 일단 첫 번째 값을 최솟값으로 설정

for (const score of scores) {
    if (score > max) max = score;  // 더 큰 값이 나오면 갱신
    if (score < min) min = score;  // 더 작은 값이 나오면 갱신
}
console.log(`최고점: ${max}점, 최저점: ${min}점`);

// ③ 80점 이상 합격자 명단 출력
console.log("\n합격자 명단:");
for (let i = 0; i < students.length; i++) {
    if (scores[i] >= 80) {
        console.log(`  합격 ${students[i]} (${scores[i]}점)`);
    }
}

// ④ 불합격자 명단 출력
console.log("\n불합격자 명단:");
for (let i = 0; i < students.length; i++) {
    if (scores[i] < 80) {
        console.log(`  불합격 ${students[i]} (${scores[i]}점)`);
    }
}
```

실행 결과:
```
전체 평균: 82점
최고점: 95점, 최저점: 60점

합격자 명단:
  합격 김민수 (85점)
  합격 이영희 (92점)
  합격 최수진 (95점)

불합격자 명단:
  불합격 박지성 (78점)
  불합격 정현우 (60점)
```

## 8️⃣ 정리 [↑](#toc)

### 핵심 개념 요약

| 개념 | 설명 | 비유 |
|------|------|------|
| `for` | 정해진 횟수만큼 반복 | 출석부 1번부터 30번까지 호명 |
| `while` | 조건이 참인 동안 반복 | 비가 그칠 때까지 우산 쓰기 |
| `break` | 반복문 즉시 종료 | 원하는 층에서 엘리베이터 멈추기 |
| `continue` | 현재 반복만 건너뜀 | 결석 학생 건너뛰고 다음 호명 |
| 배열 | 여러 값을 순서대로 저장 | 번호 붙은 서랍장 |
| 인덱스 | 배열의 위치 번호 (0부터) | 서랍 번호 (0번 칸부터 시작) |
| `for...of` | 배열 요소를 하나씩 꺼내 순회 | 서랍에서 물건 하나씩 꺼내기 |

### 반복문 선택 가이드

| 반복문 | 사용 시기 | 예시 |
|--------|-----------|------|
| `for` | 반복 횟수가 정해진 경우 | 1~100까지 합계 |
| `while` | 종료 조건이 불확실한 경우 | 사용자가 "종료"를 입력할 때까지 |
| `for...of` | 배열을 순서대로 처리하는 경우 | 학생 점수를 하나씩 확인 |
