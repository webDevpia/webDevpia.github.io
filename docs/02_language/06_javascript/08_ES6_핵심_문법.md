# 08장. ES6+ 핵심 문법

<a id="part1"></a>

## 1️⃣ ES6+란? [↑](#toc)

JavaScript는 **ECMAScript(에크마스크립트)** 표준을 따릅니다.
2015년 **ES6(ES2015)** 는 언어를 대대적으로 개편했고, 이후 매년 소규모 업데이트가 이어집니다.

### 이전 장에서 이미 배운 ES6 기능

| 문법 | 예시 | 배운 장 |
|------|------|---------|
| `let` / `const` | `const name = "Alice"` | 2장 |
| 화살표 함수 | `(x) => x * 2` | 5장 |
| 템플릿 리터럴 | `` `안녕, ${name}!` `` | 2장 |
| 구조분해 할당 | `const { name, age } = user` | 7장 |
| 스프레드 연산자 | `[...arr1, ...arr2]` | 7장 |

이 장에서는 **옵셔널 체이닝(`?.`)**, **Nullish 병합(`??`)**, **단축 속성**, **모듈 시스템**을 새로 배웁니다.

## 2️⃣ 옵셔널 체이닝 (?.) [↑](#toc)

### 조심스러운 문 열기 비유

> 문이 있으면 열고, 없으면 그냥 포기(undefined 반환) — 에러 대신 조용히 넘어갑니다.

### 기존 방식 vs 옵셔널 체이닝

```javascript
const user = { name: "김철수" }; // address 속성 없음

// 기존 방식 — &&로 단계마다 확인, 코드가 길다
const city1 = user && user.address && user.address.city;

// 옵셔널 체이닝 — ?. 로 간결하게
const city2 = user?.address?.city;

console.log(city1); // undefined
console.log(city2); // undefined
```

실행 결과:
```
undefined
undefined
```

### 실용 예시: API 응답에서 안전하게 값 꺼내기

```javascript
const apiResponse = {
  data: { user: { profile: { avatar: "https://example.com/avatar.png" } } },
};

const avatar = apiResponse?.data?.user?.profile?.avatar;
const phone  = apiResponse?.data?.user?.contact?.phone; // contact 없음

console.log(avatar); // "https://example.com/avatar.png"
console.log(phone);  // undefined
```

실행 결과:
```
https://example.com/avatar.png
undefined
```

## 3️⃣ Nullish 병합 연산자 (??) [↑](#toc)

**`??`** 는 왼쪽 값이 `null` 또는 `undefined`일 때만 오른쪽 기본값을 사용합니다.
`||`와 달리 `0`, `""`, `false` 같은 값은 유효한 값으로 취급합니다.

```javascript
console.log(0  || 100);   // 100  ← 0은 falsy이므로 기본값 사용
console.log(0  ?? 100);   // 0    ← 0은 유효한 값, 기본값 사용 안 함

console.log(null || "기본"); // "기본"
console.log(null ?? "기본"); // "기본"
```

실행 결과:
```
100
0
기본
기본
```

### 실용 예시: 사용자 설정에서 기본값 적용

```javascript
const settings = { volume: 0, language: null };

// || 는 0을 falsy로 처리 — 의도하지 않은 결과
const vol1 = settings.volume || 50; // 50 ← 음소거가 50으로 바뀜!

// ?? 는 null/undefined만 처리 — 의도대로 동작
const vol2 = settings.volume   ?? 50; // 0   ← 음소거 유지
const lang = settings.language ?? "ko"; // "ko" ← 기본값 적용

console.log(vol1, vol2, lang);
```

실행 결과:
```
50 0 ko
```

## 4️⃣ 단축 속성과 계산된 속성명 [↑](#toc)

### 단축 속성 (Shorthand Property)

변수 이름과 객체 속성 이름이 같을 때 속성 이름을 생략할 수 있습니다.

```javascript
const name = "홍길동";
const age  = 25;

const userOld = { name: name, age: age }; // 기존 방식
const user    = { name, age };            // 단축 속성

console.log(user); // { name: '홍길동', age: 25 }
```

### 계산된 속성명 (Computed Property Name)

대괄호(`[]`) 안에 표현식을 써서 속성 이름을 동적으로 결정합니다.

```javascript
const field = "email";

const obj = { [field]: "hong@example.com" }; // { email: "hong@example.com" }
console.log(obj); // { email: 'hong@example.com' }
```

### 실용 예시: 폼 데이터를 객체로 변환

```javascript
function createUser(name, email, age) {
  return { name, email, age }; // 단축 속성
}

function updateField(fieldName, value) {
  return { [fieldName]: value }; // 계산된 속성명
}

console.log(createUser("홍길동", "hong@example.com", 30));
console.log(updateField("username", "hong_gildong"));
```

실행 결과:
```
{ name: '홍길동', email: 'hong@example.com', age: 30 }
{ username: 'hong_gildong' }
```

## 5️⃣ 모듈 시스템 (import/export) [↑](#toc)

### 레고 블록 비유

> 각 파일이 하나의 레고 블록입니다.
> 필요한 블록을 `import`해서 가져와 조립하면, 코드가 정리되고 재사용이 쉬워집니다.

### named export / import

여러 함수나 값을 이름을 붙여 내보낼 때 사용합니다.

```javascript
// utils.js
export function add(a, b) { return a + b; }
export function subtract(a, b) { return a - b; }
export const PI = 3.14159;
```

```javascript
// main.js
import { add, PI } from "./utils.js"; // 필요한 것만 가져옴

console.log(add(10, 5)); // 15
console.log(PI);         // 3.14159
```

### default export / import

파일의 "대표" 기능 하나를 내보낼 때 사용합니다. import 시 이름을 자유롭게 붙일 수 있습니다.

```javascript
// calculator.js
export default function multiply(a, b) { return a * b; }
```

```javascript
// main.js
import multiply from "./calculator.js"; // 중괄호 없음, 이름 자유
console.log(multiply(4, 5)); // 20
```

### Node.js 설정

`package.json`에 아래 한 줄을 추가하면 `.js` 파일에서 `import`/`export`를 쓸 수 있습니다.

```json
{ "type": "module" }
```

### 실용 예시: utils.js에 유틸 함수 모아두기

```javascript
// utils.js
export function formatCurrency(amount) {
  return amount.toLocaleString("ko-KR") + "원";
}

export function unique(arr) {
  return [...new Set(arr)];
}
```

```javascript
// main.js
import { formatCurrency, unique } from "./utils.js";

console.log(formatCurrency(1500000));
console.log(unique(["js", "es6", "js", "web"]));
```

실행 결과:
```
1,500,000원
[ 'js', 'es6', 'web' ]
```

## 6️⃣ ES2024/2025 맛보기 [↑](#toc)

> 심화 학습은 [MDN JavaScript 문서](https://developer.mozilla.org/ko/docs/Web/JavaScript)를 참고하세요.

### Object.groupBy() — 배열 그룹화 (ES2024)

```javascript
const students = [
  { name: "김철수", grade: 1 },
  { name: "이영희", grade: 2 },
  { name: "박민준", grade: 1 },
];

const grouped = Object.groupBy(students, (s) => s.grade);
console.log(grouped);
```

실행 결과:
```
{
  '1': [ { name: '김철수', grade: 1 }, { name: '박민준', grade: 1 } ],
  '2': [ { name: '이영희', grade: 2 } ]
}
```

### Set 집합 메서드 (ES2025)

```javascript
const frontend = new Set(["Alice", "Bob", "Charlie"]);
const backend  = new Set(["Bob", "Charlie", "David"]);

console.log([...frontend.union(backend)]);        // 합집합
console.log([...frontend.intersection(backend)]); // 교집합
console.log([...frontend.difference(backend)]);   // 차집합
```

실행 결과:
```
[ 'Alice', 'Bob', 'Charlie', 'David' ]
[ 'Bob', 'Charlie' ]
[ 'Alice' ]
```

## 7️⃣ 정리 [↑](#toc)

### 핵심 문법 요약

| 문법 | 설명 | 예시 |
|------|------|------|
| `?.` | 옵셔널 체이닝 — 없으면 undefined | `user?.address?.city` |
| `??` | Nullish 병합 — null/undefined일 때만 기본값 | `value ?? "기본값"` |
| 단축 속성 | 변수명=속성명일 때 생략 | `{ name, age }` |
| 계산된 속성명 | 동적 속성 이름 | `{ [key]: value }` |
| `export` | 값/함수를 내보냄 | `export function fn() {}` |
| `import` | 다른 파일에서 가져옴 | `import { fn } from "./file.js"` |
| `Object.groupBy()` | 배열을 기준별로 그룹화 | `Object.groupBy(arr, fn)` |

### `||` vs `??` 한눈에 비교

| 값 | `\|\|` 결과 | `??` 결과 |
|----|-----------|---------|
| `null` / `undefined` | 오른쪽(기본값) | 오른쪽(기본값) |
| `0` / `""` / `false` | 오른쪽(기본값) | **왼쪽(원래 값)** |

### 실습 과제

**기본** — 옵셔널 체이닝으로 중첩 객체에서 안전하게 값 꺼내기

```javascript
const users = [
  { name: "Alice", address: { city: "서울" } },
  { name: "Bob" }, // address 없음
  { name: "Charlie", address: { city: "부산" } },
];

users.forEach((user) => {
  const city = /* 옵셔널 체이닝 + ?? 로 채우세요 */;
  console.log(`${user.name}: ${city}`);
});
// Alice: 서울 / Bob: 미등록 / Charlie: 부산
```

**중급** — utils.js 모듈 만들어서 main.js에서 import하기

`utils.js`에 `capitalize(str)`, `clamp(value, min, max)`, `range(start, end)` 세 함수를 named export로 작성하고, `main.js`에서 import해서 각각 테스트하세요. `package.json`에 `"type": "module"` 추가 후 `node main.js`로 실행합니다.

**심화** — Object.groupBy()로 학생 배열을 학년별로 그룹화하고, 각 그룹의 평균 점수를 출력하세요.

```javascript
const students = [
  { name: "김철수", grade: 2, score: 85 },
  { name: "이영희", grade: 1, score: 92 },
  { name: "박민준", grade: 2, score: 78 },
  { name: "최지원", grade: 3, score: 95 },
  { name: "정수빈", grade: 1, score: 88 },
];
```
