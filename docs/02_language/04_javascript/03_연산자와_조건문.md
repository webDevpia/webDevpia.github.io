# 03장. 연산자와 조건문

<a id="part1"></a>

## 1️⃣ 산술 연산자 [↑](#toc)

**산술 연산자(Arithmetic Operator)**는 숫자를 계산할 때 사용하는 기호입니다.

| 연산자 | 설명 | 예시 | 결과 |
|--------|------|------|------|
| `+` | 더하기 | `10 + 3` | `13` |
| `-` | 빼기 | `10 - 3` | `7` |
| `*` | 곱하기 | `10 * 3` | `30` |
| `/` | 나누기 | `10 / 3` | `3.333...` |
| `%` | 나머지 | `10 % 3` | `1` |
| `**` | 거듭제곱 | `2 ** 10` | `1024` |

**증감 연산자**(`++`, `--`)와 **복합 대입 연산자**(`+=`, `-=`, `*=`, `/=`)로 코드를 단축할 수 있습니다.

```javascript
let count = 5;
count++;             // count = count + 1
console.log(count);  // 6

let score = 100;
score -= 20;         // score = score - 20
console.log(score);  // 80
```

실행 결과:
```
6
80
```

### 실용 예시: 할인가 계산과 홀짝 판별

```javascript
// 할인가 계산
const originalPrice = 50000;
const salePrice = originalPrice - originalPrice * 0.2;   // 20% 할인
console.log("최종 가격:", salePrice);

// 홀짝 판별 — % 연산자 활용 (2로 나눈 나머지가 0이면 짝수)
const number = 7;
console.log(number % 2);   // 1 → 홀수
```

실행 결과:
```
최종 가격: 40000
1
```

## 2️⃣ 비교 연산자 [↑](#toc)

**비교 연산자(Comparison Operator)**는 두 값을 비교하여 `true`(참) 또는 `false`(거짓)를 반환합니다.

| 연산자 | 설명 | 예시 | 결과 |
|--------|------|------|------|
| `===` | 값과 타입 모두 같음 | `5 === 5` | `true` |
| `!==` | 값 또는 타입이 다름 | `5 !== "5"` | `true` |
| `>` / `<` | 크다 / 작다 | `10 > 5` | `true` |
| `>=` / `<=` | 크거나 같다 / 작거나 같다 | `5 >= 5` | `true` |

### == vs === (2장 연결)

`==`는 타입이 달라도 값만 같으면 `true`를 반환하는 **느슨한 비교**,
`===`는 값과 타입 모두 같아야 `true`를 반환하는 **엄격한 비교**입니다.
항상 `===`를 사용하세요.

```javascript
console.log(5 == "5");    // true  — 타입이 달라도 같다고 판단 (비권장)
console.log(5 === "5");   // false — 값은 같지만 타입이 다름 (권장)
```

실행 결과:
```
true
false
```

### 실용 예시: 나이 확인, 비밀번호 일치 확인

```javascript
const age = 17;
console.log(age >= 18);   // false — 미성년자

const savedPw = "abc1234";
const inputPw  = "abc1234";
console.log(savedPw === inputPw);   // true — 비밀번호 일치
```

실행 결과:
```
false
true
```

## 3️⃣ 논리 연산자 [↑](#toc)

**논리 연산자(Logical Operator)**는 여러 조건을 조합할 때 사용합니다.

### 이중 잠금과 비상구 비유

> - `&&` (AND) = **이중 잠금 문**: 두 자물쇠 모두 열려야만 통과할 수 있습니다.
> - `||` (OR) = **비상구**: 여러 출구 중 하나만 열려 있어도 나갈 수 있습니다.
> - `!` (NOT) = **반전 스위치**: 켜져 있으면 끄고, 꺼져 있으면 켭니다.

| 연산자 | 이름 | 설명 | 예시 | 결과 |
|--------|------|------|------|------|
| `&&` | AND | 두 조건 모두 `true`일 때만 `true` | `true && false` | `false` |
| `\|\|` | OR | 하나라도 `true`이면 `true` | `false \|\| true` | `true` |
| `!` | NOT | `true` ↔ `false` 반전 | `!true` | `false` |

또한 논리 연산자는 **단축 평가(short-circuit)**를 수행합니다.
`&&`는 앞이 `false`면 뒤를 확인하지 않고, `||`는 앞이 `true`면 뒤를 확인하지 않습니다.

### 실용 예시: 로그인 조건 확인

```javascript
// 아이디와 비밀번호 모두 맞아야 로그인 성공 (AND 조건)
const inputId = "user01";
const inputPw = "pass123";
const savedId = "user01";
const savedPw = "pass123";

const isLoginSuccess = (inputId === savedId) && (inputPw === savedPw);
console.log("로그인 성공:", isLoginSuccess);
```

실행 결과:
```
로그인 성공: true
```

## 4️⃣ if / else if / else [↑](#toc)

**if문**은 조건이 참(`true`)일 때만 특정 코드를 실행하는 구조입니다.

### 갈림길 이정표 비유

> if/else if/else는 **갈림길에서 이정표를 보고 방향을 선택하는 것**과 같습니다.
> 첫 번째 조건이 맞으면 그 길로, 아니면 다음 이정표를 확인합니다.
> 어느 조건도 맞지 않으면 마지막 기본 길(else)로 갑니다.

```javascript
// if/else if/else — 여러 조건을 순서대로 검사
const score = 85;

if (score >= 90) {
    console.log("등급: A");
} else if (score >= 80) {
    console.log("등급: B");
} else if (score >= 70) {
    console.log("등급: C");
} else if (score >= 60) {
    console.log("등급: D");
} else {
    console.log("등급: F");
}
```

실행 결과:
```
등급: B
```

### 중첩 if문 주의사항

if문 안에 if문을 넣는 것은 가능하지만, 너무 깊어지면 읽기 어려워집니다.
2단계 이상의 중첩은 `&&` 조건 조합으로 대체하는 것을 권장합니다.

```javascript
// 중첩 대신 && 활용
const isMember = true;
const hasCoupon = true;

if (isMember && hasCoupon) {
    console.log("회원 + 쿠폰 할인 10% 적용");
} else if (isMember) {
    console.log("회원 할인 5% 적용");
} else {
    console.log("할인 없음");
}
```

실행 결과:
```
회원 + 쿠폰 할인 10% 적용
```

## 5️⃣ 삼항 연산자 [↑](#toc)

**삼항 연산자(Ternary Operator)**는 if/else를 한 줄로 줄이는 표현입니다.

```
조건 ? 참일 때 값 : 거짓일 때 값
```

```javascript
const age = 20;

// if/else 방식
let status;
if (age >= 18) { status = "성인"; } else { status = "미성년자"; }
console.log(status);

// 삼항 연산자 방식 — 위와 완전히 같은 결과
const status2 = age >= 18 ? "성인" : "미성년자";
console.log(status2);
```

실행 결과:
```
성인
성인
```

> 조건이 간단할 때만 사용하세요. 복잡한 조건에는 if/else가 더 읽기 쉽습니다.

## 6️⃣ switch문 [↑](#toc)

**switch문**은 하나의 값과 여러 경우(case)를 비교할 때 사용합니다.

### 자판기 비유

> switch문은 **자판기**와 같습니다.
> 번호(값)를 입력하면 해당 번호의 음료(case)가 나오고,
> 없는 번호면 기본 안내(default)가 표시됩니다.

```javascript
// 기본 switch 구조
const menu = 2;

switch (menu) {
    case 1:
        console.log("아메리카노가 나옵니다.");
        break;   // 여기서 switch문을 빠져나감
    case 2:
        console.log("카페라떼가 나옵니다.");
        break;
    case 3:
        console.log("녹차라떼가 나옵니다.");
        break;
    default:   // 어떤 case도 일치하지 않을 때
        console.log("해당 메뉴가 없습니다.");
}
```

실행 결과:
```
카페라떼가 나옵니다.
```

### break를 빠뜨리면? (fall-through)

`break`를 생략하면 일치한 case 이후의 모든 case가 연속으로 실행됩니다.
이를 **fall-through(폴 스루)**라고 하며, 의도하지 않은 경우 버그가 됩니다.

```javascript
const day = 2;
switch (day) {
    case 1:
        console.log("월요일");
        // break 없음 — 아래로 흘러내림
    case 2:
        console.log("화요일");
        // break 없음 — 아래로 흘러내림
    case 3:
        console.log("수요일");
        break;
}
```

실행 결과:
```
화요일
수요일
```

> `day`가 `2`인데 "수요일"까지 출력됩니다. `break`를 빠뜨리지 마세요.

### if/else vs switch — 언제 무엇을 사용?

| 상황 | 추천 | 이유 |
|------|------|------|
| 범위 비교 (`score >= 90`) | `if/else` | switch는 정확한 값 비교만 가능 |
| 하나의 값에 4개 이상의 경우 | `switch` | 가독성이 더 좋음 |
| 조건이 2~3개 | `if/else` | switch보다 간결 |

### 실용 예시: 요일별 일정

```javascript
const today = "화";

switch (today) {
    case "월":
        console.log("월요일: 팀 회의");
        break;
    case "화":
        console.log("화요일: 코드 리뷰");
        break;
    case "수":
        console.log("수요일: 스터디");
        break;
    case "토":
    case "일":
        // 두 case를 묶어서 같은 코드 실행 — 의도적 fall-through
        console.log("주말: 휴식");
        break;
    default:
        console.log("일정 없음");
}
```

실행 결과:
```
화요일: 코드 리뷰
```

## 7️⃣ 정리 [↑](#toc)

### 핵심 개념 요약

| 개념 | 설명 | 비유 |
|------|------|------|
| 산술 연산자 | `+`, `-`, `*`, `/`, `%`, `**` 로 숫자 계산 | 계산기 |
| 비교 연산자 | 두 값을 비교해 `true`/`false` 반환 | 저울 |
| `===` vs `==` | `===`은 값과 타입 모두 비교 (권장) | 엄격한 심사 vs 느슨한 심사 |
| 논리 연산자 | `&&`, `\|\|`, `!` 로 조건 조합 | 이중 잠금, 비상구, 반전 스위치 |
| if/else if/else | 조건에 따라 다른 코드 실행 | 갈림길 이정표 |
| 삼항 연산자 | `조건 ? 참 : 거짓` 형태의 한 줄 if/else | 빠른 선택 |
| switch문 | 하나의 값과 여러 case 비교, break 필수 | 자판기 |

### 실습 과제

**기본** — 두 숫자 중 큰 수 출력하기

```javascript
const a = 42;
const b = 17;

if (a > b) {
    console.log("더 큰 수:", a);
} else if (b > a) {
    console.log("더 큰 수:", b);
} else {
    console.log("두 수가 같습니다.");
}
```

실행 결과:
```
더 큰 수: 42
```

**중급** — 점수로 A~F 등급 출력하기

```javascript
const myScore = 73;

if (myScore >= 90) {
    console.log("등급: A");
} else if (myScore >= 80) {
    console.log("등급: B");
} else if (myScore >= 70) {
    console.log("등급: C");
} else if (myScore >= 60) {
    console.log("등급: D");
} else {
    console.log("등급: F");
}
```

실행 결과:
```
등급: C
```

**심화** — 간단한 계산기 만들기

```javascript
const num1 = 8;
const num2 = 4;
const operator = "*";   // "+", "-", "*", "/" 중 하나

switch (operator) {
    case "+":
        console.log(`${num1} + ${num2} = ${num1 + num2}`);
        break;
    case "-":
        console.log(`${num1} - ${num2} = ${num1 - num2}`);
        break;
    case "*":
        console.log(`${num1} * ${num2} = ${num1 * num2}`);
        break;
    case "/":
        if (num2 === 0) {
            console.log("0으로 나눌 수 없습니다.");
        } else {
            console.log(`${num1} / ${num2} = ${num1 / num2}`);
        }
        break;
    default:
        console.log("지원하지 않는 연산자입니다.");
}
```

실행 결과:
```
8 * 4 = 32
```
