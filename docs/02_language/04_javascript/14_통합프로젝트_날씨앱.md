# 14장. 통합 프로젝트 - 날씨 앱

<a id="part1"></a>

## 1️⃣ 프로젝트 개요 [↑](#toc)

이 장에서는 **도시별 날씨 검색 앱**을 만듭니다.
사용자가 도시명을 입력하면 OpenWeatherMap API에서 현재 날씨를 가져와 카드로 표시합니다.

### 완성 화면 예시

```
┌─────────────────────────────────────────┐
│  🌤 날씨 검색                            │
│  [도시 이름 입력...      ] [검색]        │
│                                         │
│  ┌───────────────────────────────────┐  │
│  │  Seoul, KR                        │  │
│  │  🌥 Clouds                        │  │
│  │  🌡 14°C  (체감 12°C)             │  │
│  │  💧 습도 60%   💨 바람 3.2m/s     │  │
│  └───────────────────────────────────┘  │
│                                         │
│  최근 검색: Seoul  Tokyo  London         │
└─────────────────────────────────────────┘
```

### 기능별 개념 매핑

| 기능 | 사용 개념 | 해당 장 |
|------|-----------|--------|
| 도시명 저장 | 변수, `const` / `let` | 2장 |
| 입력값 검증 | 조건문 (`if`) | 3장 |
| 검색 기록 관리 | 배열, `push`, `map` | 4장, 7장 |
| API 호출 함수 | 함수, 화살표 함수 | 5장 |
| API 키 관리 | 모듈, 환경 설정 | 8장 |
| 결과 카드 표시 | DOM 조작, 이벤트 | 9장 |

## 2️⃣ API 준비 [↑](#toc)

### OpenWeatherMap 무료 API 키 발급

**API(Application Programming Interface)**는 서버가 데이터를 주고받을 수 있도록 공개한 창구입니다.
OpenWeatherMap은 전 세계 날씨 데이터를 무료로 제공합니다 (하루 1,000회 호출 한도).

**발급 순서:**

1. [https://openweathermap.org/](https://openweathermap.org/) 접속
2. 우측 상단 **Sign In → Create an Account**
3. 이메일 인증 완료
4. 로그인 후 우측 상단 아이디 클릭 → **My API Keys**
5. `Default` 키가 자동 생성됨 (없으면 **Generate** 클릭)

> 발급 직후 10~60분 정도 활성화 대기 시간이 있습니다.

### API 엔드포인트 (endpoint, 요청 주소)

```
현재 날씨:
https://api.openweathermap.org/data/2.5/weather
  ?q={도시명}
  &appid={API키}
  &units=metric       ← 섭씨 온도
  &lang=kr            ← 한국어 설명
```

실제 요청 예:
```
https://api.openweathermap.org/data/2.5/weather?q=Seoul&appid=YOUR_KEY&units=metric&lang=kr
```

### 응답 JSON 구조 미리보기

```json
{
  "name": "Seoul",
  "sys": { "country": "KR" },
  "weather": [
    { "main": "Clouds", "description": "약간의 구름이 낀 하늘" }
  ],
  "main": {
    "temp": 14.2,
    "feels_like": 12.1,
    "humidity": 60
  },
  "wind": { "speed": 3.2 }
}
```

### JSONPlaceholder 대체 연습

> API 키 발급이 늦어질 경우, 5️⃣ 전체 코드의 `USE_MOCK` 상수를 `true`로 바꾸면
> 가짜 날씨 데이터로도 UI를 미리 테스트할 수 있습니다.

## 4️⃣ JavaScript 핵심 로직 [↑](#toc)

각 단계를 순서대로 구현합니다. 각 스텝에 **몇 장에서 배운 개념인지** 주석으로 표시했습니다.

### Step 1 — DOM 요소 선택 + 이벤트 리스너

```javascript
// ── Step 1: DOM 요소 선택 (9장 - DOM 조작) ──────────────────────────

const cityInput  = document.getElementById('cityInput');
const searchBtn  = document.getElementById('searchBtn');
const errorMsg   = document.getElementById('errorMsg');
const loading    = document.getElementById('loading');
const weatherCard = document.getElementById('weatherCard');
const historyBox = document.getElementById('historyBox');
const historyList = document.getElementById('historyList');

// 검색 버튼 클릭 이벤트 (9장 - addEventListener)
searchBtn.addEventListener('click', handleSearch);

// 엔터키 이벤트 (9장 - keydown 이벤트)
cityInput.addEventListener('keydown', (e) => {
  if (e.key === 'Enter') handleSearch();
});

// 검색 실행 함수 (5장 - 함수 선언)
function handleSearch() {
  const city = cityInput.value.trim(); // 앞뒤 공백 제거

  // 입력값 검증 (3장 - 조건문)
  if (!city) {
    showError('도시 이름을 입력해주세요.');
    return;
  }

  fetchWeather(city); // 날씨 가져오기
}
```

### Step 2 — fetchWeather(city) async 함수

```javascript
// ── Step 2: API 호출 (11장 - async/await, 12장 - fetch) ─────────────

const API_KEY = 'YOUR_API_KEY_HERE'; // ← 발급받은 키로 교체
const API_URL = 'https://api.openweathermap.org/data/2.5/weather';

async function fetchWeather(city) {
  showLoading(true);      // 로딩 시작
  hideError();            // 이전 에러 숨기기
  hideCard();             // 이전 결과 숨기기

  // try/catch로 에러 처리 (12장 - 에러 처리)
  try {
    // fetch로 HTTP 요청 (12장 - fetch API)
    const url = `${API_URL}?q=${encodeURIComponent(city)}&appid=${API_KEY}&units=metric&lang=kr`;
    const response = await fetch(url);

    // HTTP 상태 코드 확인 (404 = 도시를 찾지 못함)
    if (!response.ok) {
      if (response.status === 404) {
        throw new Error(`"${city}" 도시를 찾을 수 없습니다.`);
      }
      throw new Error('날씨 정보를 불러오지 못했습니다.');
    }

    // JSON 파싱 (12장 - response.json())
    const data = await response.json();

    displayWeather(data);        // 결과 표시
    addToHistory(city);          // 검색 기록 추가
    cityInput.value = '';        // 입력창 초기화

  } catch (error) {
    // 에러 메시지 표시 (3장 - 조건문으로 분기)
    showError(error.message || '알 수 없는 오류가 발생했습니다.');
  } finally {
    showLoading(false);           // 로딩 종료 (성공·실패 모두)
  }
}
```

### Step 3 — displayWeather(data) 결과 카드 표시

```javascript
// ── Step 3: 결과를 DOM 카드로 표시 (9장 - DOM 수정) ─────────────────

function displayWeather(data) {
  // 구조분해 할당으로 필요한 값만 추출 (7장 - 구조분해 할당)
  const { name, sys, weather, main, wind } = data;
  const { country } = sys;
  const { description } = weather[0];         // 배열 첫 번째 항목 (4장 - 배열)
  const { temp, feels_like, humidity } = main; // 중첩 구조분해

  // 온도 단위 변환 객체 (7장 - 객체)
  const celsius  = Math.round(temp);
  const feelsC   = Math.round(feels_like);

  // textContent로 텍스트 삽입 (9장 - DOM 수정)
  document.getElementById('cityName').textContent    = `${name}, ${country}`;
  document.getElementById('weatherDesc').textContent = description;
  document.getElementById('temp').textContent        = `🌡 ${celsius}°C`;
  document.getElementById('feelsLike').textContent   = `체감 ${feelsC}°C`;
  document.getElementById('humidity').textContent    = `💧 습도 ${humidity}%`;
  document.getElementById('wind').textContent        = `💨 바람 ${wind.speed}m/s`;

  weatherCard.classList.remove('hidden'); // 카드 보이기
}
```

### Step 4 — 에러 처리 헬퍼 함수

```javascript
// ── Step 4: 에러 처리 + 상태 제어 (5장 - 함수, 3장 - 조건문) ────────

function showError(message) {
  errorMsg.textContent = `⚠️ ${message}`;
  errorMsg.classList.remove('hidden');
}

function hideError() {
  errorMsg.textContent = '';
  errorMsg.classList.add('hidden');
}

function showLoading(isVisible) {
  // 삼항 연산자로 클래스 토글 (3장 - 삼항 연산자)
  isVisible
    ? loading.classList.remove('hidden')
    : loading.classList.add('hidden');
}

function hideCard() {
  weatherCard.classList.add('hidden');
}
```

### Step 5 — 검색 기록 관리

```javascript
// ── Step 5: 검색 기록 (4장 - 배열, 7장 - 스프레드 연산자) ───────────

const MAX_HISTORY = 5;       // 최대 기록 수 (2장 - const)
let searchHistory = [];      // 기록 배열   (2장 - let, 4장 - 배열)

function addToHistory(city) {
  // 중복 제거: 같은 도시가 이미 있으면 삭제 후 맨 앞에 추가
  searchHistory = searchHistory.filter(c => c !== city); // 7장 - filter
  searchHistory = [city, ...searchHistory];              // 7장 - 스프레드

  // 최대 개수 초과 시 마지막 항목 제거 (4장 - slice)
  if (searchHistory.length > MAX_HISTORY) {
    searchHistory = searchHistory.slice(0, MAX_HISTORY);
  }

  renderHistory(); // DOM 업데이트
}

function renderHistory() {
  if (searchHistory.length === 0) {
    historyBox.classList.add('hidden');
    return;
  }

  historyBox.classList.remove('hidden');

  // map으로 li 요소 생성 (7장 - map), join으로 HTML 조합 (4장 - join)
  historyList.innerHTML = searchHistory
    .map(city => `<li><button onclick="fetchWeather('${city}')">${city}</button></li>`)
    .join('');
}
```

### Step 6 — 로딩 상태 + 버튼 비활성화

```javascript
// ── Step 6: 로딩 중 버튼 비활성화 (9장 - DOM 속성 변경) ────────────

function showLoading(isVisible) {
  if (isVisible) {
    loading.classList.remove('hidden');
    searchBtn.disabled = true;         // 버튼 비활성화
    searchBtn.textContent = '검색 중...';
  } else {
    loading.classList.add('hidden');
    searchBtn.disabled = false;        // 버튼 재활성화
    searchBtn.textContent = '검색';
  }
}
```

> Step 4의 `showLoading`을 이 버전으로 교체하면 더 완성도 있는 UX가 됩니다.

```html
      <h3>최근 검색</h3>
      <ul id="historyList"></ul>
    </div>
  </div>

  <script>
    // ─────────────────────────────────────────────────────────────────
    // 날씨 검색 앱 - JavaScript 전체 코드
    // ─────────────────────────────────────────────────────────────────

    // ── 설정 상수 (2장 - const, 8장 - 환경 설정) ────────────────────
    const API_KEY  = 'YOUR_API_KEY_HERE'; // ← 본인 키로 교체
    const API_URL  = 'https://api.openweathermap.org/data/2.5/weather';
    const MAX_HISTORY = 5;    // 최대 검색 기록 개수
    const USE_MOCK    = false; // true로 바꾸면 가짜 데이터로 테스트

    // ── 가짜 데이터 (API 키 없을 때 테스트용) ─────────────────────
    const MOCK_DATA = {
      name: 'Seoul', sys: { country: 'KR' },
      weather: [{ main: 'Clouds', description: '약간의 구름이 낀 하늘' }],
      main: { temp: 14.2, feels_like: 12.1, humidity: 60 },
      wind: { speed: 3.2 }
    };

    // ── DOM 요소 선택 (9장 - DOM 조작) ───────────────────────────────
    const cityInput   = document.getElementById('cityInput');
    const searchBtn   = document.getElementById('searchBtn');
    const errorMsg    = document.getElementById('errorMsg');
    const loadingEl   = document.getElementById('loading');
    const weatherCard = document.getElementById('weatherCard');
    const historyBox  = document.getElementById('historyBox');
    const historyList = document.getElementById('historyList');

    // ── 검색 기록 배열 (4장 - 배열, 2장 - let) ───────────────────────
    let searchHistory = [];

    // ── 이벤트 리스너 등록 (9장 - addEventListener) ──────────────────
    searchBtn.addEventListener('click', handleSearch);
    cityInput.addEventListener('keydown', (e) => {
      if (e.key === 'Enter') handleSearch();
    });

    // ── 검색 실행 (5장 - 함수, 3장 - 조건문) ────────────────────────
    function handleSearch() {
      const city = cityInput.value.trim(); // String.trim()으로 공백 제거

      if (!city) {                         // 빈 입력 검증 (3장 - if)
        showError('도시 이름을 입력해주세요.');
        return;
      }

      fetchWeather(city);
    }

    // ── API 호출 (11장 - async/await, 12장 - fetch, try/catch) ───────
    async function fetchWeather(city) {
      setLoading(true);
      hideError();
      weatherCard.classList.add('hidden');

      try {
        let data;

        if (USE_MOCK) {
          // 가짜 데이터 반환 (테스트 모드)
          await new Promise(r => setTimeout(r, 600)); // 로딩 효과
          data = MOCK_DATA;
        } else {
          // 실제 API 호출 (12장 - fetch, encodeURIComponent)
          const url = `${API_URL}?q=${encodeURIComponent(city)}&appid=${API_KEY}&units=metric&lang=kr`;
          const response = await fetch(url);

          if (!response.ok) {
            // HTTP 오류 상태 분기 (3장 - 조건문)
            if (response.status === 404) {
              throw new Error(`"${city}" 도시를 찾을 수 없습니다.`);
            }
            if (response.status === 401) {
              throw new Error('API 키가 올바르지 않습니다.');
            }
            throw new Error('날씨 정보를 불러오지 못했습니다.');
          }

          data = await response.json(); // JSON 파싱 (12장)
        }

        displayWeather(data);  // 결과 카드 표시
        addToHistory(city);    // 검색 기록 추가
        cityInput.value = '';  // 입력창 초기화

      } catch (error) {
        // 네트워크 오류 포함 모든 예외 처리 (12장 - catch)
        showError(error.message || '알 수 없는 오류가 발생했습니다.');
      } finally {
        setLoading(false);     // 성공·실패 관계없이 로딩 종료
      }
    }

    // ── 날씨 카드 표시 (9장 - DOM 수정, 7장 - 구조분해 할당) ─────────
    function displayWeather(data) {
      // 구조분해 할당으로 필요한 값 추출 (7장)
      const { name, sys: { country }, weather, main, wind } = data;
      const { description }                   = weather[0];
      const { temp, feels_like, humidity }    = main;

      // Math.round()로 소수점 반올림 (2장 - 숫자 타입)
      document.getElementById('cityName').textContent    = `${name}, ${country}`;
      document.getElementById('weatherDesc').textContent = description;
      document.getElementById('temp').textContent        = `🌡 ${Math.round(temp)}°C`;
      document.getElementById('feelsLike').textContent   = `체감 ${Math.round(feels_like)}°C`;
      document.getElementById('humidity').textContent    = `💧 습도 ${humidity}%`;
      document.getElementById('wind').textContent        = `💨 바람 ${wind.speed}m/s`;

      weatherCard.classList.remove('hidden');
    }

    // ── 검색 기록 추가 (4장 - 배열, 7장 - filter/스프레드) ───────────
    function addToHistory(city) {
      // 중복 제거 후 맨 앞에 삽입 (7장 - filter, 스프레드 연산자)
      searchHistory = [city, ...searchHistory.filter(c => c !== city)];

      // 최대 개수 초과 시 잘라내기 (4장 - slice)
      if (searchHistory.length > MAX_HISTORY) {
        searchHistory = searchHistory.slice(0, MAX_HISTORY);
      }

      renderHistory();
    }

    // ── 검색 기록 DOM 렌더링 (9장 - innerHTML, 7장 - map/join) ───────
    function renderHistory() {
      if (searchHistory.length === 0) {
        historyBox.classList.add('hidden');
        return;
      }

      historyBox.classList.remove('hidden');

      // map + join으로 HTML 문자열 생성 (7장 - 고차 함수)
      historyList.innerHTML = searchHistory
        .map(city => `<li><button onclick="fetchWeather('${city}')">${city}</button></li>`)
        .join('');
    }

    // ── 유틸리티 함수 (5장 - 함수 재사용) ───────────────────────────

    function setLoading(isVisible) {
      if (isVisible) {
        loadingEl.classList.remove('hidden');
        searchBtn.disabled    = true;
        searchBtn.textContent = '검색 중...';
      } else {
        loadingEl.classList.add('hidden');
        searchBtn.disabled    = false;
        searchBtn.textContent = '검색';
      }
    }

    function showError(message) {
      errorMsg.textContent = `⚠️ ${message}`;
      errorMsg.classList.remove('hidden');
    }

    function hideError() {
      errorMsg.textContent = '';
      errorMsg.classList.add('hidden');
    }
  </script>
</body>
</html>
```

```javascript
function addToHistory(city) {
  searchHistory = [city, ...searchHistory.filter(c => c !== city)];
  if (searchHistory.length > MAX_HISTORY) {
    searchHistory = searchHistory.slice(0, MAX_HISTORY);

  // localStorage에 JSON 문자열로 저장
  localStorage.setItem('weatherHistory', JSON.stringify(searchHistory));
```
