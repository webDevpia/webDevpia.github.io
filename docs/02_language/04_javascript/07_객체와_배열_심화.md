# 07장. 객체와 배열 심화

<a id="part1"></a>

## 1️⃣ 객체(Object)란? [↑](#toc)

**객체(Object)**는 관련된 데이터를 **키(key)와 값(value) 쌍**으로 묶어서 저장하는 자료구조입니다.

### 신분증 비유

> 신분증에는 이름, 나이, 주소 등 한 사람에 대한 여러 정보가 **하나의 카드**에 담겨 있습니다.
> 객체는 이 신분증처럼, 한 대상에 관한 **여러 속성(property)**을 하나로 묶어 관리합니다.

```javascript
// 중괄호 {}로 객체를 만든다
const person = {
    name: "김민수",    // key: "name",  value: "김민수"
    age: 25,          // key: "age",   value: 25
    city: "서울"       // key: "city",  value: "서울"
};

console.log(person); // { name: '김민수', age: 25, city: '서울' }
```

### 속성 접근: 점 표기법 vs 대괄호 표기법

```javascript
const person = { name: "이영희", age: 30, job: "개발자" };

console.log(person.name);   // "이영희"  — 점 표기법 (가장 일반적)
console.log(person["job"]); // "개발자"  — 대괄호 표기법

// 변수를 키로 사용할 때는 대괄호 표기법만 가능
const key = "name";
console.log(person[key]);   // "이영희" — person.key 는 undefined
```

> 대괄호 표기법은 키 이름에 **공백·특수문자**가 있거나, **변수로 동적으로 키를 지정**할 때 사용합니다.

### 속성 추가, 수정, 삭제

```javascript
const car = { brand: "현대", model: "소나타" };

car.year = 2023;          // 추가
car.model = "그랜저";     // 수정
delete car.year;           // 삭제

console.log(car); // { brand: '현대', model: '그랜저' }
```

### 메서드 — 함수인 속성

```javascript
const calculator = {
    add: function(a, b) { return a + b; },  // 메서드 선언
    subtract(a, b) { return a - b; }        // 단축 문법 (ES6+)
};

console.log(calculator.add(10, 5));      // 15
console.log(calculator.subtract(10, 5)); // 5
```

## 2️⃣ 객체 활용 [↑](#toc)

### 객체 배열 — 실무에서 가장 자주 쓰는 패턴

실무에서는 **같은 구조의 객체 여러 개**를 배열로 관리합니다.
데이터베이스 목록이나 API 응답이 대부분 이 형태입니다.

```javascript
const students = [
    { name: "김민수", score: 85, grade: "2학년" },
    { name: "이영희", score: 92, grade: "3학년" },
    { name: "박철호", score: 78, grade: "1학년" }
];

console.log(students[0].name);  // "김민수"

for (const student of students) {
    console.log(`${student.name} (${student.grade}): ${student.score}점`);
}
```

실행 결과:
```
김민수
김민수 (2학년): 85점
이영희 (3학년): 92점
박철호 (1학년): 78점
```

### Object 내장 메서드 & for...in

```javascript
const product = { name: "노트북", price: 1200000, stock: 5 };

console.log(Object.keys(product));    // [ 'name', 'price', 'stock' ]
console.log(Object.values(product));  // [ '노트북', 1200000, 5 ]
console.log(Object.entries(product)); // [ ['name','노트북'], ['price',1200000], ['stock',5] ]

// for...in — 객체 키를 순서대로 순회
for (const key in product) {
    console.log(`${key}: ${product[key]}`);
}
```

실행 결과:
```
name: 노트북
price: 1200000
stock: 5
```

## 3️⃣ 구조 분해 할당 (Destructuring) [↑](#toc)

**구조 분해 할당(Destructuring)**은 객체나 배열에서 값을 꺼내 변수에 담는 문법입니다.

### 택배 개봉 비유

> 택배 상자(객체)를 받으면, 상자를 열어 내용물에 이름표(변수)를 붙입니다.
> 구조 분해 할당은 필요한 물건만 꺼내 이름표를 붙이는 과정입니다.

### 객체 구조 분해

```javascript
const person = { name: "김민수", age: 25, city: "서울" };

// 기존 방식
const name1 = person.name;
const age1  = person.age;

// 구조 분해 할당 — 한 줄로 처리
const { name, age, city } = person;
console.log(name, age, city); // 김민수 25 서울

// 기본값 지정 & 변수 이름 변경
const { username = "guest", role: permission = "viewer" } = { username: "alice" };
console.log(username);   // "alice"
console.log(permission); // "viewer"  (role이 없으므로 기본값)
```

### 배열 구조 분해

```javascript
const colors = ["빨강", "초록", "파랑", "노랑"];

const [first, second] = colors;
console.log(first, second); // 빨강 초록

// 중간 요소 건너뛰기 — 쉼표로 자리 비움
const [, , blue] = colors;
console.log(blue); // 파랑
```

### 함수 매개변수에서 구조 분해 — 실무에서 매우 빈번

```javascript
// 구조 분해 없이
function greet(person) {
    console.log(`${person.name}님 (${person.age}세)`);
}

// 구조 분해 사용 — 필요한 속성만 바로 꺼내 쓴다
function greetV2({ name, age, role = "일반회원" }) {
    console.log(`${name}님 (${age}세) — ${role}`);
}

const member = { name: "박지수", age: 22 };
greet(member);   // 박지수님 (22세)
greetV2(member); // 박지수님 (22세) — 일반회원
```

## 4️⃣ 스프레드 연산자 (...) [↑](#toc)

**스프레드 연산자(Spread Operator)** `...`는 배열이나 객체를 **펼쳐서** 복사하거나 합칩니다.

### 복사기 비유

> 복사기로 원본 문서를 새 종이에 찍어내듯, `...`는 원본의 내용을 새 배열·객체에 복사합니다.
> **원본은 변경되지 않습니다.**

### 배열 복사 & 합치기

```javascript
const fruits = ["사과", "바나나", "포도"];
const fruitsCopy = [...fruits];  // 원본과 독립된 새 배열
fruitsCopy.push("망고");

console.log(fruits);     // ["사과", "바나나", "포도"]  — 원본 유지
console.log(fruitsCopy); // ["사과", "바나나", "포도", "망고"]

// 배열 합치기
const veggies  = ["당근", "오이"];
const combined = [...fruits, ...veggies, "토마토"];
console.log(combined); // ["사과", "바나나", "포도", "당근", "오이", "토마토"]
```

### 객체 복사 & 병합 — 속성 덮어쓰기

```javascript
const defaultSettings = { theme: "light", fontSize: 14, lang: "ko" };
const userSettings    = { theme: "dark",  fontSize: 16 };

// 뒤에 오는 속성이 앞의 동일 키를 덮어쓴다
const finalSettings = { ...defaultSettings, ...userSettings };
console.log(finalSettings);
// { theme: 'dark', fontSize: 16, lang: 'ko' }
```

### 레스트 매개변수 (...args)

```javascript
// 여러 인수를 하나의 배열로 모은다
function sum(...numbers) {
    return numbers.reduce((acc, n) => acc + n, 0);
}

console.log(sum(1, 2, 3));        // 6
console.log(sum(10, 20, 30, 40)); // 100

// 첫 번째 인수는 따로, 나머지는 배열로
function introduce(name, ...hobbies) {
    console.log(`${name}의 취미: ${hobbies.join(", ")}`);
}
introduce("김민수", "독서", "코딩", "등산");
// 김민수의 취미: 독서, 코딩, 등산
```

## 5️⃣ 배열 고차 함수 [↑](#toc)

**고차 함수(Higher-Order Function)**는 **함수를 인수로 받아** 배열 요소 각각에 적용하는 메서드입니다.
`for` 반복문 없이도 데이터를 변환·필터링·집계할 수 있습니다.

### 공장 컨베이어 벨트 비유

> 원재료(배열)가 컨베이어 벨트(고차 함수)를 타고 각 공정(콜백)을 거쳐
> 가공·선별·집계된 제품(결과 배열)이 나옵니다. **원본 배열은 그대로 남습니다.**

### map() — 변환

모든 요소에 함수를 적용한 **새로운 배열** 반환 (길이 동일)

```javascript
const prices = [10000, 25000, 8000, 45000];

// 모든 가격에 10% 할인 적용
const discounted = prices.map(price => price * 0.9);
console.log(discounted); // [9000, 22500, 7200, 40500]

// 객체 배열에서 특정 속성만 추출
const users = [
    { id: 1, name: "김민수" },
    { id: 2, name: "이영희" }
];
console.log(users.map(u => u.name)); // ["김민수", "이영희"]
```

### filter() — 필터링

조건에 맞는 요소만 골라 **새로운 배열** 반환

```javascript
const scores = [88, 45, 72, 95, 60, 83];
console.log(scores.filter(s => s >= 70)); // [88, 72, 95, 83]

const products = [
    { name: "노트북", inStock: true  },
    { name: "마우스", inStock: false },
    { name: "키보드", inStock: true  }
];
console.log(products.filter(p => p.inStock).map(p => p.name));
// ["노트북", "키보드"]
```

### reduce() — 축소·집계

모든 요소를 **하나의 값으로** 합산·변환

```javascript
const numbers = [1, 2, 3, 4, 5];
// reduce(콜백(누적값, 현재요소), 초기값)
const total = numbers.reduce((acc, cur) => acc + cur, 0);
console.log(total); // 15

// 장바구니 총 금액 (price × qty)
const cart = [
    { name: "커피",     price: 5000, qty: 2 },
    { name: "샌드위치", price: 8000, qty: 1 },
    { name: "주스",     price: 4500, qty: 3 }
];
const totalPrice = cart.reduce((acc, item) => acc + item.price * item.qty, 0);
console.log(`총 결제 금액: ${totalPrice.toLocaleString()}원`);
// 총 결제 금액: 31,500원
```

### find() / some() / every()

```javascript
const members = [{ id: 101, name: "김민수" }, { id: 102, name: "이영희" }];

// find() — 첫 번째 일치 요소 반환, 없으면 undefined
console.log(members.find(m => m.id === 102)?.name); // "이영희"

const ages = [15, 22, 17, 30];
console.log(ages.some(a => a >= 18));  // true  — 하나라도 조건 만족
console.log(ages.every(a => a >= 18)); // false — 모두 조건 만족해야
```

### 메서드 체이닝 — .filter().map() 연쇄 호출

```javascript
const products = [
    { name: "노트북",  price: 1200000, inStock: true  },
    { name: "마우스",  price: 30000,   inStock: false },
    { name: "키보드",  price: 80000,   inStock: true  },
    { name: "USB허브", price: 8000,    inStock: true  },
    { name: "모니터",  price: 350000,  inStock: true  }
];

const result = products
    .filter(p => p.inStock)         // ① 재고 있는 것만
    .filter(p => p.price >= 10000)  // ② 1만원 이상만
    .map(p => p.name);              // ③ 이름만 추출

console.log(result); // ["노트북", "키보드", "모니터"]
```

## 6️⃣ 실용 예제: 학생 성적 분석기 [↑](#toc)

객체 배열, 구조 분해, 고차 함수를 종합하여 학생 성적 데이터를 분석합니다.

```javascript
const students = [
    { name: "김민수", score: 85, grade: "2학년" },
    { name: "이영희", score: 92, grade: "3학년" },
    { name: "박철호", score: 78, grade: "1학년" },
    { name: "최수진", score: 96, grade: "2학년" },
    { name: "정재원", score: 63, grade: "1학년" },
    { name: "한지민", score: 88, grade: "3학년" },
    { name: "오승현", score: 71, grade: "2학년" }
];

// ① 전체 평균
const average = students.reduce((acc, s) => acc + s.score, 0) / students.length;
console.log(`전체 평균: ${average.toFixed(1)}점`);

// ② 80점 이상 우수 학생 명단
const topStudents = students
    .filter(s => s.score >= 80)
    .map(({ name, score }) => `${name} (${score}점)`);
console.log("우수 학생:", topStudents.join(", "));

// ③ 학년별 평균 — reduce로 그룹화
const gradeMap = students.reduce((acc, { grade, score }) => {
    if (!acc[grade]) acc[grade] = { total: 0, count: 0 };
    acc[grade].total += score;
    acc[grade].count += 1;
    return acc;
}, {});

console.log("\n--- 학년별 평균 ---");
for (const [grade, data] of Object.entries(gradeMap)) {
    console.log(`${grade}: ${(data.total / data.count).toFixed(1)}점`);
}

// ④ 최고 점수 학생
const topScorer = students.reduce((best, s) => s.score > best.score ? s : best);
console.log(`\n최고 점수: ${topScorer.name} (${topScorer.score}점)`);
```

실행 결과:
```
전체 평균: 81.9점
우수 학생: 김민수 (85점), 이영희 (92점), 최수진 (96점), 한지민 (88점)

--- 학년별 평균 ---
2학년: 84.7점
3학년: 90.0점
1학년: 70.5점

최고 점수: 최수진 (96점)
```

## 7️⃣ 정리 [↑](#toc)

### 핵심 개념 요약

| 개념 | 문법 | 핵심 역할 |
|------|------|-----------|
| 객체 생성 | `{ key: value }` | 관련 데이터를 하나로 묶음 |
| 속성 접근 | `obj.key` / `obj["key"]` | 점 표기법 vs 대괄호 표기법 |
| 구조 분해 (객체) | `const { a, b } = obj` | 속성을 변수로 한번에 꺼냄 |
| 구조 분해 (배열) | `const [x, y] = arr` | 인덱스 순으로 변수에 담음 |
| 스프레드 | `{ ...obj }` / `[...arr]` | 복사·병합, 원본 불변 |
| 레스트 매개변수 | `function f(...args)` | 여러 인수를 배열로 모음 |
| map() | `arr.map(fn)` | 전체 변환 → 같은 길이 새 배열 |
| filter() | `arr.filter(fn)` | 조건 선별 → 더 짧은 새 배열 |
| reduce() | `arr.reduce(fn, init)` | 하나의 값으로 집계 |
| find() | `arr.find(fn)` | 첫 번째 일치 요소 반환 |
| some() / every() | `arr.some(fn)` | 조건 만족 여부 불리언 반환 |

### 실습 과제

#### 기본 — 자신의 정보를 객체로 만들고 구조 분해로 꺼내기

```javascript
const me = {
    name: "내 이름",
    age: 20,
    hobbies: ["독서", "코딩"]
};

// 구조 분해 할당으로 name과 hobbies를 꺼내어 출력하세요
const { name, hobbies } = me;
console.log(`이름: ${name}, 취미: ${hobbies.join(", ")}`);
```

#### 중급 — filter + map으로 1만원 이상 상품 이름 추출

```javascript
const items = [
    { name: "아메리카노", price: 4500 },
    { name: "케이크",    price: 12000 },
    { name: "샌드위치",  price: 8500 },
    { name: "스테이크",  price: 32000 },
    { name: "주스",      price: 6000 }
];
```
