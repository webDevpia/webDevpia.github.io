## ⭐ 핵심: 1️⃣ HTML이란? [↑](#toc)

### 웹페이지를 건물에 비유하기

웹페이지를 만들 때는 세 가지 언어가 함께 사용됩니다. 건물을 짓는 것에 비유하면 이해하기 쉽습니다.

| 건물 | 웹페이지 | 역할 |
|------|---------|------|
| 구조 (벽, 바닥, 기둥) | **HTML** | 무엇이 있는지 정의 |
| 인테리어 (색, 벽지, 가구) | **CSS** | 어떻게 보이는지 정의 |
| 전기 · 수도 (스위치, 엘리베이터) | **JavaScript** | 어떻게 동작하는지 정의 |

건물은 구조 없이 인테리어만 할 수 없고, 전기 없이는 엘리베이터가 안 됩니다. 웹도 마찬가지입니다. 세 가지가 각자의 역할을 합니다.

이번 장에서는 HTML(구조)을 먼저 만들고, JavaScript(동작)를 연결합니다. CSS(꾸미기)는 나중에 다룹니다.

### HTML 기본 구조

HTML(HyperText Markup Language)은 태그(tag)라고 부르는 꺽쇠괄호(`< >`) 기호를 사용합니다.

태그는 시작 태그와 끝 태그가 쌍을 이룹니다:

```html
<h1>제목입니다</h1>
 ↑            ↑
시작 태그    끝 태그 (/ 슬래시가 있음)
```

모든 웹페이지는 아래 기본 구조를 가집니다:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>나의 첫 페이지</title>
  </head>
  <body>
    <h1>안녕하세요!</h1>
    <p>나의 첫 번째 웹페이지입니다.</p>
  </body>
</html>
```

각 부분이 무엇인지 살펴봅시다:

```
<!DOCTYPE html>
→ "이 파일은 HTML5 문서야"라고 선언하는 줄. 항상 맨 위에 씁니다.

<html> ... </html>
→ 모든 내용이 이 안에 들어갑니다. 집 전체라고 생각하세요.

<head> ... </head>
→ 보이지 않는 설정들. 제목, 문자 인코딩 등.
→ 집의 전기 배전반처럼 뒤에서 동작하는 것들.

<title>나의 첫 페이지</title>
→ 브라우저 탭에 표시되는 제목.

<body> ... </body>
→ 실제로 화면에 보이는 모든 내용이 여기 들어갑니다.
→ 집에서 사람들이 생활하는 공간.

<h1>안녕하세요!</h1>
→ h는 heading(제목)의 약자. h1이 가장 크고, h6이 가장 작습니다.

<p>내용</p>
→ p는 paragraph(문단)의 약자. 일반 텍스트를 담습니다.
```

### 첫 HTML 파일 만들기

> 💡 **실습 흐름**: 강사 시연 보기 → 함께 따라하기 → 혼자 변형해 보기

강사 시연을 보면서 따라하세요. VS Code에서 작업 폴더를 열고, 새 파일을 만듭니다.

파일 이름: `index.html`

(왜 `index.html`? 웹서버는 폴더를 열었을 때 `index.html`을 기본 파일로 찾습니다. 책의 목차 페이지와 같은 역할입니다.)

아래 코드를 직접 타이핑합니다:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>나의 첫 페이지</title>
  </head>
  <body>
    <h1>안녕하세요!</h1>
    <p>나의 첫 번째 웹페이지입니다.</p>
  </body>
</html>
```

**브라우저에서 열기**

저장(`Ctrl + S`) 후 두 가지 방법으로 열 수 있습니다:

**방법 1 - 직접 열기:**
파일 탐색기에서 `index.html`을 더블클릭합니다.

**방법 2 - Live Server 사용 (권장):**
VS Code 오른쪽 하단에 "Go Live" 버튼이 있습니다. 클릭하면 브라우저가 자동으로 열립니다. 코드를 수정하면 저장 즉시 브라우저가 새로고침됩니다.

브라우저에서 "안녕하세요!"가 크게 보이면 성공입니다!

### 자주 쓰는 HTML 태그들

나중에 쓸 태그들을 미리 알아봅시다:

```html
<!-- 제목 태그 (크기가 다름) -->
<h1>가장 큰 제목</h1>
<h2>두 번째 크기</h2>
<h3>세 번째 크기</h3>

<!-- 문단 태그 -->
<p>일반 텍스트입니다.</p>

<!-- 줄바꿈 (닫는 태그 없음) -->
<br>

<!-- 링크 태그 -->
<a href="https://google.com">구글로 이동</a>

<!-- 이미지 태그 (닫는 태그 없음) -->
<img src="사진파일.jpg" alt="사진 설명">

<!-- 버튼 태그 -->
<button>클릭하세요</button>

<!-- 입력 태그 (닫는 태그 없음) -->
<input type="text" placeholder="여기에 입력하세요">

<!-- 특정 요소를 지정할 때 쓰는 id 속성 -->
<p id="myText">이 문단을 JavaScript로 바꿀 거예요</p>
```

> **중요 개념 — 주석(Comment)**: `<!-- 이 안에 있는 내용 -->`은 브라우저가 무시합니다. 코드에 메모를 남길 때 씁니다. JavaScript에서는 `//` 한 줄 주석, `/* */` 여러 줄 주석을 씁니다.

## ⭐ 핵심: 2️⃣ JavaScript 연결하기 [↑](#toc)

### `<script>` 태그

HTML에 JavaScript를 넣는 방법은 `<script>` 태그를 사용하는 것입니다.

```html
<!DOCTYPE html>
<html>
  <head>
    <title>JS 연결 테스트</title>
  </head>
  <body>
    <h1>안녕하세요!</h1>

    <!-- JavaScript는 보통 body 태그의 마지막에 씁니다 -->
    <script>
      console.log("JavaScript가 실행됐습니다!");
    </script>
  </body>
</html>
```

> **왜 `<body>` 마지막에 쓰는가?** 브라우저는 HTML을 위에서 아래로 순서대로 읽습니다. JavaScript가 `<h1>`보다 먼저 실행되면, 아직 h1이 만들어지지 않은 상태라 h1을 조작할 수 없습니다. `<body>` 끝에 두면 모든 HTML이 준비된 후에 JavaScript가 실행됩니다.

### console.log 확인하기

위 파일을 저장하고 브라우저에서 열어봅니다.

화면에는 "안녕하세요!"만 보입니다. `console.log`는 화면이 아니라 개발자 콘솔에 출력되기 때문입니다.

`F12`를 눌러 개발자 도구를 열고 "Console" 탭을 클릭하면:

```
JavaScript가 실행됐습니다!
```

이 메시지가 보입니다.

### JavaScript로 HTML 바꾸기

이제 진짜 재미있는 부분입니다. JavaScript로 HTML 내용을 바꿔봅시다.

```html
<!DOCTYPE html>
<html>
  <head>
    <title>JS 테스트</title>
  </head>
  <body>
    <h1 id="title">원래 제목</h1>
    <p id="description">원래 설명입니다.</p>

    <script>
      // id가 "title"인 요소를 찾아서 내용을 바꿉니다
      document.getElementById('title').textContent = '환영합니다!';

      // 더 짧은 방법 (querySelector 사용)
      document.querySelector('#description').textContent = '제목이 바뀌었죠?';
    </script>
  </body>
</html>
```

브라우저에서 열면 "원래 제목" 대신 "환영합니다!"가, "원래 설명입니다." 대신 "제목이 바뀌었죠?"가 보입니다.

잠깐, 이게 어떻게 동작하는 걸까요?

```javascript
document.getElementById('title').textContent = '환영합니다!';
//   ↑              ↑              ↑                ↑
//  문서(HTML)   id='title'인   텍스트 내용을    이것으로 바꿔줘
//  전체           요소를 찾아
```

`document`는 현재 웹페이지 전체를 의미합니다. `getElementById('title')`은 id가 'title'인 요소를 찾는 명령입니다. `.textContent`는 그 요소의 텍스트 내용을 말합니다. `=` 뒤에 새로운 텍스트를 써주면 바뀝니다.

이것이 JavaScript가 HTML을 조작하는 방식입니다. **DOM(Document Object Model)**이라고 부르는데, 쉽게 말하면 "JavaScript가 HTML 요소들을 레고 블록처럼 다룰 수 있다"는 것입니다.

### querySelector vs getElementById

두 가지 방법을 봤는데, 어떤 것을 쓸까요?

```javascript
// getElementById: id로만 찾을 수 있음, 빠름
document.getElementById('title')

// querySelector: CSS 선택자 방식으로 찾음, 더 유연함
document.querySelector('#title')    // id로 찾기 (# 기호 사용)
document.querySelector('.box')      // class로 찾기 (. 기호 사용)
document.querySelector('h1')        // 태그로 찾기
```

지금은 `getElementById`를 주로 사용합니다. 명확하고 이해하기 쉽기 때문입니다.

## ⭐ 핵심: 3️⃣ 버튼 만들기 [↑](#toc)

### 30분 안에 만드는 첫 결과물

> 💡 **실습 흐름**: 강사 시연 보기 → 함께 따라하기 → 혼자 변형해 보기

이제 진짜 인터랙티브 웹페이지를 만들 차례입니다. 버튼을 누르면 텍스트가 바뀌는 기능을 만들겠습니다.

아래 코드를 `index.html`에 직접 타이핑합니다:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>나의 첫 인터랙티브 페이지</title>
  </head>
  <body>
    <h1>나의 첫 웹페이지</h1>

    <!-- 버튼을 클릭하면 sayHello 함수가 실행됩니다 -->
    <button onclick="sayHello()">인사하기</button>

    <!-- 결과가 여기에 표시됩니다. 처음엔 비어있습니다. -->
    <p id="result"></p>

    <script>
      // sayHello 함수 정의
      function sayHello() {
        // id가 'result'인 요소를 찾아서 텍스트를 바꿉니다
        document.getElementById('result').textContent = '안녕하세요, 반갑습니다!';
      }
    </script>
  </body>
</html>
```

저장하고 브라우저에서 열면 "인사하기" 버튼이 보입니다. 버튼을 클릭하면 "안녕하세요, 반갑습니다!"라는 텍스트가 나타납니다.

**와! 작동합니다!**

### 코드 해부하기

각 줄이 무엇을 하는지 자세히 살펴봅시다.

**HTML 부분:**

```html
<button onclick="sayHello()">인사하기</button>
```

- `<button>`: 클릭 가능한 버튼을 만듭니다
- `onclick="sayHello()"`: 클릭되면(onclick) `sayHello()`라는 함수를 실행하라는 명령
- `인사하기`: 버튼에 표시되는 텍스트

```html
<p id="result"></p>
```

- `<p>`: 문단 태그
- `id="result"`: 이 요소의 고유 이름. JavaScript에서 찾을 때 사용합니다
- 시작 태그와 끝 태그 사이에 내용이 없어서 처음엔 아무것도 안 보입니다

**JavaScript 부분:**

```javascript
function sayHello() {
  document.getElementById('result').textContent = '안녕하세요, 반갑습니다!';
}
```

- `function`: "함수를 만든다"는 키워드. 요리 레시피와 같습니다.
- `sayHello()`: 이 함수의 이름. 나중에 이 이름으로 호출합니다.
- `{ ... }`: 중괄호 안이 이 함수가 할 일들입니다.
- `document.getElementById('result')`: HTML에서 id='result'인 요소를 찾습니다.
- `.textContent = '...'`: 그 요소의 텍스트를 바꿉니다.

### 함수(Function)란?

함수는 **이름 붙인 레시피**입니다.

```
함수 sayHello()의 레시피:
  1. id가 'result'인 요소를 찾는다
  2. 그 요소의 텍스트를 '안녕하세요, 반갑습니다!'로 바꾼다
```

버튼을 클릭할 때마다 이 레시피가 실행됩니다.

함수를 미리 정의해두는 이유는, 같은 동작을 여러 곳에서 반복하지 않기 위해서입니다. 라면 레시피를 한 번만 쓰고, 필요할 때마다 꺼내 보는 것과 같습니다.

### 여러 버튼 추가해보기

버튼을 더 만들어봅시다. 아래 코드로 `index.html`을 업데이트하세요:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>나의 첫 인터랙티브 페이지</title>
  </head>
  <body>
    <h1>나의 첫 웹페이지</h1>

    <button onclick="sayHello()">인사하기</button>
    <button onclick="sayBye()">작별인사</button>
    <button onclick="clearMessage()">지우기</button>

    <p id="result"></p>

    <script>
      function sayHello() {
        document.getElementById('result').textContent = '안녕하세요, 반갑습니다! 👋';
      }

      function sayBye() {
        document.getElementById('result').textContent = '안녕히 가세요! 다음에 또 봐요.';
      }

      function clearMessage() {
        // 빈 문자열로 바꾸면 내용이 사라집니다
        document.getElementById('result').textContent = '';
      }
    </script>
  </body>
</html>
```

세 가지 버튼이 각각 다른 동작을 합니다. 각 버튼은 자신에 해당하는 함수를 호출합니다.

## ⭐ 핵심: 4️⃣ 나만의 자기소개 페이지 만들기 [↑](#toc)

> 💡 **실습 흐름**: 강사 시연 보기 → 함께 따라하기 → 혼자 변형해 보기

이제 더 완성된 페이지를 만들겠습니다. 자기소개 페이지입니다.

### 전체 코드

새 파일 `about.html`을 만들고 아래 코드를 직접 타이핑합니다:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>자기소개 페이지</title>
  </head>
  <body>
    <!-- 기본 정보 섹션 -->
    <h1 id="name">김코딩</h1>
    <p id="intro">안녕하세요! 저는 JavaScript를 배우고 있습니다.</p>

    <hr><!-- 가로선 -->

    <!-- 버튼들 -->
    <button onclick="showHobby()">취미 보기</button>
    <button onclick="showFood()">좋아하는 음식 보기</button>
    <button onclick="changeBackground()">배경색 바꾸기</button>

    <!-- 정보가 표시될 영역 -->
    <p id="info"></p>

    <script>
      // 취미를 보여주는 함수
      function showHobby() {
        document.getElementById('info').textContent =
          '제 취미는 영화 보기, 산책하기, 요리입니다.';
      }

      // 좋아하는 음식을 보여주는 함수
      function showFood() {
        document.getElementById('info').textContent =
          '제가 좋아하는 음식은 삼겹살, 초밥, 파스타입니다.';
      }

      // 배경색을 바꾸는 함수
      function changeBackground() {
        // document.body는 <body> 전체를 가리킵니다
        // style.backgroundColor는 배경색을 설정합니다
        document.body.style.backgroundColor = 'lightyellow';
        document.getElementById('info').textContent =
          '배경색이 바뀌었어요!';
      }
    </script>
  </body>
</html>
```

### 코드 수정하기

위 코드에서 여러분의 정보로 바꿔보세요:

1. `김코딩` → 여러분의 이름
2. `안녕하세요! 저는 JavaScript를 배우고 있습니다.` → 여러분의 소개
3. `영화 보기, 산책하기, 요리` → 여러분의 취미
4. `삼겹살, 초밥, 파스타` → 여러분이 좋아하는 음식

저장하고 브라우저에서 열어보세요. 버튼들이 모두 동작하나요?

### 배경색 더 바꿔보기

`changeBackground` 함수에서 `'lightyellow'` 대신 다른 색깔 이름을 써보세요:

```javascript
// 색깔 이름 예시
document.body.style.backgroundColor = 'lightblue';   // 연한 파랑
document.body.style.backgroundColor = 'lightgreen';  // 연한 초록
document.body.style.backgroundColor = 'lightpink';   // 연한 분홍
document.body.style.backgroundColor = 'lavender';    // 라벤더
document.body.style.backgroundColor = 'white';       // 흰색 (원래대로)

// 16진수 코드로도 색을 지정할 수 있습니다
document.body.style.backgroundColor = '#FFD700';     // 금색
document.body.style.backgroundColor = '#87CEEB';     // 하늘색
```

버튼을 누를 때마다 다른 색으로 바뀌게 만들 수 있을까요? 지금은 어렵지만, Phase 1을 배우고 나면 쉽게 만들 수 있습니다.

### innerHTML vs textContent

방금 `.textContent`를 사용했는데, `.innerHTML`이라는 것도 있습니다.

```javascript
// textContent: 순수 텍스트만 넣을 수 있음
document.getElementById('info').textContent = '취미: <b>영화</b>';
// 결과: 취미: <b>영화</b>  (태그가 그대로 보임)

// innerHTML: HTML 태그도 함께 넣을 수 있음
document.getElementById('info').innerHTML = '취미: <b>영화</b>';
// 결과: 취미: 영화  (영화가 굵게 표시됨)
```

`<b>` 태그는 텍스트를 굵게(bold) 만듭니다.

지금 단계에서는 `textContent`를 주로 사용합니다. 더 안전하기 때문입니다. (나중에 배울 보안 개념과 관련이 있습니다)

## 5️⃣ AI에게 도움 받기 [↑](#toc)

### 언제 AI에게 물어볼까?

코딩 중에 이런 상황이 생깁니다:

- 태그 이름이 기억나지 않을 때
- 에러 메시지가 뭔 뜻인지 모를 때
- "이렇게 하는 게 맞나?" 궁금할 때
- 개념이 완전히 이해되지 않을 때

이럴 때 ChatGPT에게 물어보세요. **단, 코드를 만들어달라고는 하지 마세요.**

### 실전 질문 예시

**상황 1: 태그가 기억나지 않을 때**

```
나는 JavaScript 입문자야.
HTML에서 텍스트를 굵게 만드는 태그가 뭐야?
사용 예시도 보여줘.
```

**상황 2: onclick이 이해가 안 될 때**

```
나는 JavaScript를 처음 배우고 있어.
HTML에서 onclick 속성이 정확히 어떤 역할을 하는 거야?
버튼을 클릭하면 어떤 순서로 실행되는지 단계별로 설명해줘.
```

**상황 3: 에러가 났을 때**

```
JavaScript를 배우고 있는데 에러가 났어.
아래 에러 메시지가 무슨 뜻인지, 어떤 실수를 했을 때 이 에러가 나는지 설명해줘:

"Uncaught ReferenceError: sayHello is not defined"
```

**상황 4: 내가 쓴 코드 이해가 안 될 때**

```
나는 JavaScript 입문자야.
아래 코드가 어떻게 동작하는 건지 한 줄씩 설명해줘.
코드를 수정하거나 다시 만들어주지는 말고, 설명만 해줘.

function sayHello() {
  document.getElementById('result').textContent = '안녕하세요!';
}
```

마지막 예시가 중요합니다. "설명만 해줘"라고 명확히 요청하면 AI가 코드를 바꾸지 않고 설명만 합니다.

### AI와 대화할 때 팁

**팁 1: 내 수준을 알려주세요**

"나는 JavaScript 입문자야", "비전공자야", "방금 처음 배우기 시작했어" 같이 말하면 AI가 더 쉽게 설명합니다.

**팁 2: 이해가 안 되면 다시 물어보세요**

```
좀 더 쉽게 설명해줄 수 있어?
비유를 들어서 설명해줘.
예시를 하나 더 보여줘.
```

AI는 지치지 않습니다. 이해될 때까지 계속 물어보세요.

**팁 3: 대화를 이어가세요**

AI와의 대화는 단발성이 아닙니다. 한 주제에 대해 계속 파고들 수 있습니다.

```
사용자: getElementById랑 querySelector의 차이가 뭐야?
AI: (설명)
사용자: 그럼 언제 querySelector를 쓰는 게 더 좋아?
AI: (설명)
사용자: 그 예시를 더 보여줄 수 있어?
AI: (설명)
```

**Phase 0 AI 사용 규칙 다시 확인:**

> ✅ **허용**: 개념 설명, 에러 메시지 해석, 코드 이해 도움  
> ❌ **금지**: 코드 생성 요청, 완성된 코드 달라고 하기

> 여유가 있다면 도전해 보세요! 수업 후에 천천히 익혀도 좋습니다.

새 파일 `food.html`을 만들고 아래 조건을 만족하는 페이지를 만드세요:

### 💪 보너스 챌린지: 카운터 만들기

> 시간이 남는다면 도전! 빠르게 끝낸 분들을 위한 추가 과제입니다.

버튼을 누를 때마다 숫자가 1씩 올라가는 카운터를 만드세요.

새 파일 `counter.html`을 만들고 아래 조건을 만족하는 카운터를 만드세요:

**조건:**
1. 숫자 `0`이 화면에 표시됩니다
2. "+" 버튼을 누르면 숫자가 1씩 올라갑니다
3. "-" 버튼을 누르면 숫자가 1씩 내려갑니다 (0 미만으로는 안 내려갑니다)
4. "초기화" 버튼을 누르면 숫자가 다시 0이 됩니다

**핵심 개념: 변수(variable)**

숫자를 기억하려면 **변수**가 필요합니다. 변수는 값을 저장하는 상자입니다.

```javascript
// let은 "변수를 만든다"는 키워드
// count라는 이름의 상자를 만들고, 0을 넣어둡니다
let count = 0;

// 상자의 값을 바꿀 수 있습니다
count = 5;       // count가 5가 됩니다
count = count + 1; // count가 6이 됩니다 (현재 값 + 1)
```

**힌트:**

```html
<h1>카운터</h1>
<p id="counter">0</p>

<button onclick="increase()">+</button>
<button onclick="decrease()">-</button>
<button onclick="reset()">초기화</button>

<script>
  // 현재 숫자를 기억하는 변수
  let count = 0;

  function increase() {
    count = count + 1;  // 1을 더합니다
    // 화면에 새 숫자를 표시하세요 (여기에 코드 추가)
  }

  function decrease() {
    // 0 미만이 되지 않게 하려면 어떻게 할까요?
    // (힌트: if 조건문을 써봐요 — 다음 장에서 배웁니다)
    // 지금은 그냥 1을 빼보세요
    count = count - 1;
    // 화면에 새 숫자를 표시하세요
  }

  function reset() {
    count = 0;
    // 화면에 0을 표시하세요
  }
</script>
```

도전 과제에서 변수(`let count`)를 처음 사용합니다. 다음 장에서 변수를 본격적으로 배웁니다.

**0 미만 방지는 어떻게 할까요?**  
다음 장에서 배울 `if` 문을 써야 합니다. 지금은 일단 빠지는 것을 만들어보고, 다음 장을 배운 후에 0 미만 방지 기능을 추가해보세요.

## 7️⃣ 정리 [↑](#toc)

### 이 장에서 배운 것

| 개념 | 한 줄 정리 |
|------|----------|
| HTML | 웹페이지의 뼈대. 태그로 구조를 만듭니다 |
| `<h1>` ~ `<h6>` | 제목 태그. 숫자가 작을수록 크고 굵습니다 |
| `<p>` | 문단 태그. 일반 텍스트에 사용합니다 |
| `<button>` | 클릭 가능한 버튼을 만듭니다 |
| `id` 속성 | HTML 요소에 고유한 이름을 붙입니다 |
| `onclick` | 클릭 시 실행할 JavaScript 함수를 지정합니다 |
| `<script>` | HTML 안에 JavaScript 코드를 넣는 태그입니다 |
| `function` | 재사용 가능한 코드 묶음 (레시피)을 만듭니다 |
| `getElementById()` | id로 HTML 요소를 찾습니다 |
| `.textContent` | 요소의 텍스트 내용을 읽거나 바꿉니다 |
| `let` | 값을 저장하는 변수를 만듭니다 |

### 학습 체크리스트

- [ ] `index.html` 파일을 만들고 기본 HTML 구조를 작성했다
- [ ] `<script>` 태그를 추가하고 `console.log`를 실행했다
- [ ] 버튼을 클릭하면 텍스트가 바뀌는 페이지를 만들었다
- [ ] `about.html` 자기소개 페이지를 만들고 정보를 채웠다
- [ ] 기본 과제(음식 페이지)를 완성했다
- [ ] 도전 과제(카운터)를 시도해봤다
- [ ] ChatGPT에 이번 장 관련 질문을 해봤다

### 자주 하는 실수

**1. id 이름이 다름**
```html
<!-- HTML에서: -->
<p id="Result"></p>  <!-- 대문자 R -->

<!-- JavaScript에서: -->
document.getElementById('result')  <!-- 소문자 r -->
```
JavaScript는 대소문자를 구분합니다. `Result`와 `result`는 다릅니다.

**2. function 괄호를 빠뜨림**
```html
<!-- ❌ 잘못된 예: -->
<button onclick="sayHello">클릭</button>

<!-- ✅ 맞는 예: -->
<button onclick="sayHello()">클릭</button>
```
함수를 호출할 때는 반드시 `()`를 붙여야 합니다.

**3. script 태그 위치**
```html
<!-- ❌ 잘못된 예: script가 body보다 먼저 -->
<script>
  document.getElementById('result') // 아직 result가 없음!
</script>
<body>
  <p id="result"></p>
</body>

<!-- ✅ 맞는 예: body 안의 마지막에 script -->
<body>
  <p id="result"></p>
  <script>
    document.getElementById('result') // result가 이미 만들어진 후
  </script>
</body>
```

### 핵심 한 줄

> 웹페이지는 HTML(구조) + JavaScript(동작)로 만듭니다. 버튼 클릭 → 함수 실행 → 화면 변경이 기본 패턴입니다.
