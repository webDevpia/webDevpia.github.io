## ⭐ 핵심: 1️⃣ 객체란? — 명함 비유 [↑](#toc)

> 💡 **실습 흐름**: 강사 시연 보기 → 함께 따라하기 → 혼자 변형해 보기

### 명함을 생각해보세요

한 장의 명함에는 여러 정보가 있습니다:

```
+------------------------------+
| 이름:   김민준                |
| 직책:   프론트엔드 개발자     |
| 이메일: min@email.com        |
| 전화:   010-1234-5678        |
+------------------------------+
```

지금까지는 이 정보를 따로따로 저장했습니다:

```javascript
const name = "김민준";
const role = "프론트엔드 개발자";
const email = "min@email.com";
const phone = "010-1234-5678";
```

사람이 100명이면 변수가 400개! 관리하기 어렵습니다.

**객체(Object)**를 사용하면 관련 데이터를 한 곳에 묶을 수 있습니다.

---

### 객체 리터럴 문법

```javascript
const person = {
  name: "김민준",
  role: "프론트엔드 개발자",
  email: "min@email.com",
  phone: "010-1234-5678",
};
```

- `{` `}` 중괄호로 감쌉니다
- `키: 값` 쌍으로 데이터를 저장합니다
- 각 쌍은 쉼표로 구분합니다 (마지막 쉼표는 선택사항, 있어도 됨)

---

### 속성에 접근하기

**점 표기법 (dot notation)** — 가장 일반적

```javascript
const person = {
  name: "김민준",
  age: 28,
  city: "서울",
};

console.log(person.name); // 출력: 김민준
console.log(person.age);  // 출력: 28
console.log(person.city); // 출력: 서울
```

**대괄호 표기법 (bracket notation)** — 키가 변수일 때 유용

```javascript
const key = "name";
console.log(person[key]);    // 출력: 김민준
console.log(person["age"]); // 출력: 28
```

---

### 속성 추가하기, 변경하기, 삭제하기

```javascript
const student = {
  name: "이서연",
  score: 85,
};

// 속성 추가
student.grade = "B";
console.log(student); // 출력: { name: '이서연', score: 85, grade: 'B' }

// 속성 변경
student.score = 92;
console.log(student.score); // 출력: 92

// 속성 삭제
delete student.grade;
console.log(student); // 출력: { name: '이서연', score: 92 }
```

`const`로 선언해도 객체 **안의 값**은 바꿀 수 있습니다.  
`const`가 막는 건 객체 자체를 다른 것으로 교체하는 것입니다.

```javascript
const student = { name: "이서연" };
student.name = "김철수";     // 가능 — 속성 변경
student = { name: "박민수" }; // 에러! — 객체 자체를 교체
```

---

### 없는 속성에 접근하면?

```javascript
const person = { name: "홍길동" };
console.log(person.age);    // 출력: undefined
console.log(person.email);  // 출력: undefined
// 에러가 아님! undefined를 반환
```

## 2️⃣ 객체 활용하기 — 메서드와 실전 예제 [↑](#toc)

### 객체 안에 함수 — 메서드

객체의 속성 값으로 함수를 넣을 수 있습니다. 이를 **메서드(method)**라고 합니다.

```javascript
const calculator = {
  add: (a, b) => a + b,
  subtract: (a, b) => a - b,
  multiply: (a, b) => a * b,
};

console.log(calculator.add(5, 3));      // 출력: 8
console.log(calculator.subtract(10, 4)); // 출력: 6
console.log(calculator.multiply(3, 7));  // 출력: 21
```

---

### 학생 정보 관리 예제

```javascript
const student = {
  name: "박지훈",
  scores: [85, 92, 78, 96],
  getAverage() {
    const total = this.scores.reduce((sum, s) => sum + s, 0);
    return total / this.scores.length;
  },
  getGrade() {
    const avg = this.getAverage();
    if (avg >= 90) return "A";
    if (avg >= 80) return "B";
    if (avg >= 70) return "C";
    return "F";
  },
};

console.log(student.name);           // 출력: 박지훈
console.log(student.getAverage());   // 출력: 87.75
console.log(student.getGrade());     // 출력: B
```

`this`는 "이 객체 자신"을 가리킵니다. 지금은 패턴으로만 기억하세요.

---

### 객체 배열 — 여러 사람 관리

실제 업무에서 가장 자주 보이는 패턴입니다.

```javascript
const students = [
  { name: "김민준", score: 85 },
  { name: "이서연", score: 92 },
  { name: "박지훈", score: 78 },
  { name: "최유리", score: 96 },
  { name: "정현우", score: 61 },
];

// 반복문으로 출력
for (const student of students) {
  console.log(`${student.name}: ${student.score}점`);
}
// 출력:
// 김민준: 85점
// 이서연: 92점
// 박지훈: 78점
// 최유리: 96점
// 정현우: 61점
```

## 🚀 도전: 3️⃣ 구조분해 할당 — 꺼내기의 편리함 [↑](#toc)

> 여유가 있다면 도전해 보세요! 수업 후에 천천히 익혀도 좋습니다.

### 객체 구조분해 할당

객체에서 특정 속성을 꺼낼 때 반복해서 `object.property`를 쓰는 것이 번거롭습니다.

```javascript
// 기존 방법
const person = { name: "홍길동", age: 25, city: "서울" };
const name = person.name;
const age = person.age;
const city = person.city;
```

**구조분해 할당**으로 한 줄에:

```javascript
const person = { name: "홍길동", age: 25, city: "서울" };
const { name, age, city } = person;

console.log(name); // 출력: 홍길동
console.log(age);  // 출력: 25
console.log(city); // 출력: 서울
```

왼쪽의 `{ name, age, city }`가 객체의 키 이름과 일치해야 합니다.

---

### 다른 이름으로 꺼내기

```javascript
const { name: userName, age: userAge } = person;
console.log(userName); // 출력: 홍길동
console.log(userAge);  // 출력: 25
```

---

### 기본값 설정

```javascript
const config = { theme: "dark" };
const { theme, language = "ko" } = config; // language가 없으면 "ko" 사용

console.log(theme);    // 출력: dark
console.log(language); // 출력: ko
```

---

### 배열 구조분해 할당

배열도 순서대로 꺼낼 수 있습니다.

```javascript
const colors = ["빨강", "파랑", "초록", "노랑"];
const [first, second, , fourth] = colors; // 세 번째는 건너뜀

console.log(first);  // 출력: 빨강
console.log(second); // 출력: 파랑
console.log(fourth); // 출력: 노랑
```

---

### 함수 매개변수에서 구조분해

함수가 객체를 받을 때 매우 유용합니다.

```javascript
// 구조분해 없이
function printStudent(student) {
  console.log(`${student.name}: ${student.score}점`);
}

// 구조분해 있이 — 더 명확
function printStudent({ name, score }) {
  console.log(`${name}: ${score}점`);
}

const student = { name: "이서연", score: 92 };
printStudent(student); // 출력: 이서연: 92점
```

## 🚀 도전: 4️⃣ 배열 고급 메서드 — map, filter, find [↑](#toc)

> 여유가 있다면 도전해 보세요! 수업 후에 천천히 익혀도 좋습니다.

### 왜 배울까?

지금까지 배열을 다룰 때 `for` 반복문을 썼습니다.  
`map`, `filter`, `find`는 더 간결하고 선언적인 방법입니다.

---

### map — "변환기"

배열의 **모든 원소를 변환**해서 새 배열을 만듭니다.

```javascript
const scores = [85, 92, 78, 96, 61];

// for 반복문 방식
const doubled = [];
for (const score of scores) {
  doubled.push(score * 2);
}

// map 방식 — 훨씬 간결
const doubled2 = scores.map(score => score * 2);

console.log(doubled);  // 출력: [ 170, 184, 156, 192, 122 ]
console.log(doubled2); // 출력: [ 170, 184, 156, 192, 122 ]
```

원본 배열은 **변경되지 않습니다**. 새 배열을 반환합니다.

---

### map 실전 예제

```javascript
const students = [
  { name: "김민준", score: 85 },
  { name: "이서연", score: 92 },
  { name: "박지훈", score: 78 },
];

// 이름만 추출
const names = students.map(s => s.name);
console.log(names); // 출력: [ '김민준', '이서연', '박지훈' ]

// 점수에 10점 보너스 추가
const boosted = students.map(s => ({
  name: s.name,
  score: s.score + 10,
}));
console.log(boosted);
// 출력:
// [ { name: '김민준', score: 95 },
//   { name: '이서연', score: 102 },
//   { name: '박지훈', score: 88 } ]

// 등급 추가
const withGrades = students.map(s => ({
  ...s,
  grade: s.score >= 90 ? "A" : s.score >= 80 ? "B" : "C",
}));
console.log(withGrades);
// 출력:
// [ { name: '김민준', score: 85, grade: 'B' },
//   { name: '이서연', score: 92, grade: 'A' },
//   { name: '박지훈', score: 78, grade: 'C' } ]
```

---

### filter — "걸러내기"

조건에 맞는 원소만 모아 새 배열을 만듭니다.

```javascript
const scores = [85, 92, 78, 96, 61, 55, 74, 88];

// 70점 이상만 필터링
const passing = scores.filter(score => score >= 70);
console.log(passing); // 출력: [ 85, 92, 78, 96, 74, 88 ]

// 홀수만
const odds = scores.filter(n => n % 2 !== 0);
console.log(odds); // 출력: [ 85, 61, 55 ]
```

---

### filter 실전 예제

```javascript
const students = [
  { name: "김민준", score: 85, passed: true },
  { name: "이서연", score: 92, passed: true },
  { name: "박지훈", score: 55, passed: false },
  { name: "최유리", score: 96, passed: true },
  { name: "정현우", score: 61, passed: false },
];

// 합격자만
const passedStudents = students.filter(s => s.passed);
console.log(passedStudents.map(s => s.name));
// 출력: [ '김민준', '이서연', '최유리' ]

// 80점 이상
const topStudents = students.filter(s => s.score >= 80);
console.log(topStudents.length); // 출력: 3
```

---

### find — "첫 번째 찾기"

조건에 맞는 **첫 번째 원소 하나**를 반환합니다. 없으면 `undefined`.

```javascript
const students = [
  { name: "김민준", score: 85 },
  { name: "이서연", score: 92 },
  { name: "박지훈", score: 78 },
];

const topStudent = students.find(s => s.score >= 90);
console.log(topStudent); // 출력: { name: '이서연', score: 92 }

const notFound = students.find(s => s.score >= 100);
console.log(notFound);   // 출력: undefined
```

---

### map + filter 연결하기

메서드 체이닝으로 여러 단계를 연결할 수 있습니다.

```javascript
const students = [
  { name: "김민준", score: 85 },
  { name: "이서연", score: 92 },
  { name: "박지훈", score: 55 },
  { name: "최유리", score: 96 },
];

// 70점 이상인 학생의 이름만
const passingNames = students
  .filter(s => s.score >= 70)
  .map(s => s.name);

console.log(passingNames); // 출력: [ '김민준', '이서연', '최유리' ]
```

## 📖 더 알아보기: 5️⃣ 스프레드 연산자 — 복사와 합치기 [↑](#toc)

> 수업 후에 천천히 읽어보세요. 실무에서 매우 자주 사용되는 문법입니다.

### 스프레드 연산자란?

`...` (점 세 개)는 **스프레드(spread) 연산자**입니다.  
배열이나 객체를 "펼쳐서" 복사하거나 합칠 때 씁니다.

---

### 배열 복사

```javascript
const original = [1, 2, 3];
const copy = [...original];

copy.push(4);

console.log(original); // 출력: [ 1, 2, 3 ]  ← 원본 그대로
console.log(copy);     // 출력: [ 1, 2, 3, 4 ]
```

주의: `const copy = original;` 은 복사가 아닙니다!

```javascript
const arr1 = [1, 2, 3];
const arr2 = arr1;       // 복사가 아닌 참조 복사!
arr2.push(4);
console.log(arr1);       // 출력: [ 1, 2, 3, 4 ]  ← 원본도 바뀜!

const arr3 = [1, 2, 3];
const arr4 = [...arr3];  // 진짜 복사
arr4.push(4);
console.log(arr3);       // 출력: [ 1, 2, 3 ]  ← 원본 안 바뀜
```

---

### 배열 합치기

```javascript
const first = [1, 2, 3];
const second = [4, 5, 6];

const combined = [...first, ...second];
console.log(combined); // 출력: [ 1, 2, 3, 4, 5, 6 ]

// 중간에 원소 추가
const withExtra = [...first, 99, ...second];
console.log(withExtra); // 출력: [ 1, 2, 3, 99, 4, 5, 6 ]
```

---

### 객체 복사와 합치기

```javascript
const defaults = { theme: "dark", language: "ko", fontSize: 14 };
const userSettings = { theme: "light", fontSize: 16 };

// 기본값에 사용자 설정을 덮어씌우기
const finalSettings = { ...defaults, ...userSettings };
console.log(finalSettings);
// 출력: { theme: 'light', language: 'ko', fontSize: 16 }
// theme과 fontSize는 userSettings 값으로 덮어씌워짐
```

---

### 함수 인자로 배열 펼치기

```javascript
const numbers = [3, 1, 4, 1, 5, 9, 2, 6];

console.log(Math.max(...numbers)); // 출력: 9
console.log(Math.min(...numbers)); // 출력: 1
```

`Math.max()`는 배열이 아닌 개별 숫자를 받습니다.  
`...`로 배열을 펼쳐서 개별 인자로 전달합니다.

## 6️⃣ 실습 과제 [↑](#toc)

### 기본 과제 — 학생 정보 관리

파일 `students.js`를 만들고 다음 내용을 **직접** 작성하세요.

```javascript
const students = [
  { name: "김민준", grade: "1학년", score: 85 },
  { name: "이서연", grade: "2학년", score: 92 },
  { name: "박지훈", grade: "1학년", score: 55 },
  { name: "최유리", grade: "3학년", score: 96 },
  { name: "정현우", grade: "2학년", score: 61 },
];
```

조건:
1. 구조분해 할당으로 첫 번째 학생의 name, grade, score를 출력
2. `forEach`로 모든 학생의 이름과 점수를 출력
3. 전체 평균 점수를 계산해서 출력

예상 출력:
```
이름: 김민준, 학년: 1학년, 점수: 85
---
김민준: 85점
이서연: 92점
박지훈: 55점
최유리: 96점
정현우: 61점
---
전체 평균: 77.8점
```

---

### 도전 과제 — map + filter로 성적 분석

위의 `students` 배열을 사용해서 다음을 구현하세요.

조건:
1. `filter`로 80점 이상 학생만 걸러내기
2. 걸러낸 학생들의 이름만 `map`으로 추출
3. 결과를 한 줄로 연결해서 출력

추가 조건:
4. 전체 학생에게 `map`으로 등급(A/B/C/F)을 추가한 새 배열을 만들어 출력

예상 출력:
```
80점 이상: 김민준, 이서연, 최유리

전체 성적표:
이름: 김민준 | 점수: 85 | 등급: B
이름: 이서연 | 점수: 92 | 등급: A
이름: 박지훈 | 점수: 55 | 등급: F
이름: 최유리 | 점수: 96 | 등급: A
이름: 정현우 | 점수: 61 | 등급: F
```

힌트: 이름만 추출한 배열을 문자열로 합칠 때 `.join(", ")`을 사용하세요.

## 7️⃣ 정리 [↑](#toc)

### 이 장에서 배운 것

| 개념 | 핵심 |
|------|------|
| 객체 | `{ key: value }` — 관련 데이터를 하나로 묶음 |
| 점 표기법 | `object.key` — 속성 접근 |
| 대괄호 표기법 | `object["key"]` — 동적 속성 접근 |
| 구조분해 할당 | `const { name, age } = person` — 속성을 바로 변수로 꺼냄 |
| `map` | 배열을 변환해서 새 배열 반환. 원본 불변 |
| `filter` | 조건에 맞는 원소만 새 배열로 반환 |
| `find` | 조건에 맞는 첫 번째 원소 반환 |
| 스프레드 `...` | 배열/객체 복사, 합치기, 함수 인자로 펼치기 |

### 자주 하는 실수

```javascript
// 실수 1: 없는 속성 접근
const obj = { name: "홍길동" };
console.log(obj.age.toString()); // 에러! age가 undefined이므로

// 실수 2: const 객체 재할당 시도
const person = { name: "홍길동" };
person = { name: "이순신" }; // 에러! 재할당 불가

// 실수 3: map 결과를 저장 안 함
const scores = [1, 2, 3];
scores.map(n => n * 2); // 결과 버림!
const doubled = scores.map(n => n * 2); // 이렇게 해야 함

// 실수 4: filter가 원소 개수를 유지한다고 착각
// filter는 조건에 맞는 원소만 담은 새 배열 반환
const result = [1, 2, 3, 4, 5].filter(n => n > 3);
console.log(result); // 출력: [ 4, 5 ]  ← 길이가 줄어듦
```
