## 1️⃣ 비지도학습 복습 [↑](#toc)

지금까지 우리가 배운 모델들은 모두 **지도학습(Supervised Learning)**이었습니다.

| 구분 | 지도학습 | 비지도학습 |
|------|----------|------------|
| 정답(레이블) | 있음 ✅ | 없음 ❌ |
| 목표 | 예측 | 구조 발견 |
| 예시 | 스팸 분류, 집값 예측 | 고객 그룹, 이상 탐지 |

> **비유**: 지도학습은 "선생님이 정답지를 주고 가르치는 것"이고, 비지도학습은 "정답지 없이 스스로 규칙을 찾는 것"입니다.

**클러스터링(Clustering)**은 비지도학습의 대표 기법으로, **비슷한 데이터끼리 자동으로 묶어주는** 알고리즘입니다.

**어디에 쓸까요?**
- 고객을 구매 패턴별로 그룹화 → 맞춤 마케팅
- 뉴스 기사를 주제별로 자동 분류
- 유전자 데이터에서 유사 유전자 군집 발견
- 이상 거래 탐지 (혼자 떨어진 데이터 = 이상)

## 2️⃣ K-Means 직관 — 깃발과 팀 [↑](#toc)

> **K-Means = 무작위로 깃발을 꽂고, 가장 가까운 사람끼리 팀을 만든다. 그 다음 깃발을 팀 중앙으로 옮기고 반복한다.**

운동회를 상상해보세요.

1. 운동장에 학생들이 흩어져 있습니다.
2. 선생님이 눈 감고 무작위로 **깃발 3개**를 꽂습니다.
3. 각 학생은 **가장 가까운 깃발 쪽 팀**에 합류합니다.
4. 각 팀의 **중심(평균 위치)**을 계산해 깃발을 옮깁니다.
5. 3~4번을 **팀 구성이 바뀌지 않을 때까지 반복**합니다.

이것이 바로 **K-Means** 알고리즘입니다!

- **K** = 팀(클러스터)의 수
- **Means** = 팀의 평균 위치(중심점, Centroid)

```
초기 상태:     1번 반복:       2번 반복:       수렴:
● ● ×  ●     ●●  ×  ●       ●●  ×   ●      ●●  ×   ●
   ×          ●●   ●          ●●   ●●         ●●   ●●
● ×  ●       ×    ●●         ×    ●●          ×    ●●
(×=깃발)
```

## 3️⃣ K-Means 알고리즘 단계 [↑](#toc)

**Step 1: 초기화 (Initialization)**
- k개의 중심점을 랜덤으로 선택합니다.
- (실제로는 `k-means++` 방식으로 더 영리하게 초기화합니다)

**Step 2: 할당 (Assignment)**
- 모든 데이터 포인트를 가장 가까운 중심점의 클러스터에 할당합니다.
- 거리 계산: 유클리드 거리 (두 점 사이의 직선 거리)

**Step 3: 업데이트 (Update)**
- 각 클러스터의 새 중심점 = 해당 클러스터에 속한 모든 점들의 **평균**

**Step 4: 반복 (Repeat)**
- 중심점이 더 이상 바뀌지 않거나, 최대 반복 횟수에 도달할 때까지 2~3단계 반복

<details markdown="0">
<summary>📐 더 알아보기 — 거리 공식과 목적 함수</summary>

**유클리드 거리 (2D)**

두 점 A=(x₁, y₁)와 B=(x₂, y₂) 사이의 거리:

```
d(A, B) = √((x₂-x₁)² + (y₂-y₁)²)
```

**K-Means의 목적 함수 (WCSS)**

K-Means는 클러스터 내 분산의 합(Within-Cluster Sum of Squares)을 최소화합니다:

```
WCSS = Σ(클러스터별) Σ(점별) ||점 - 중심점||²
```

값이 작을수록 각 클러스터가 촘촘하게 모여있다는 의미입니다.

## 4️⃣ sklearn으로 K-Means 실행하기 [↑](#toc)

Google Colab을 열고 다음 코드를 실행해 보세요.

```python
# 필요한 라이브러리 불러오기
import numpy as np
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs

# 실습용 데이터 만들기 (인위적으로 3개의 클러스터)
X, y_true = make_blobs(n_samples=300, centers=3, cluster_std=0.6, random_state=42)

# K-Means 모델 생성 및 학습
kmeans = KMeans(n_clusters=3, random_state=42, n_init=10)
kmeans.fit(X)

# 각 데이터 포인트의 클러스터 번호
labels = kmeans.labels_
print("클러스터 레이블 (앞 10개):", labels[:10])

# 중심점 확인
centroids = kmeans.cluster_centers_
print("\n중심점 좌표:")
print(centroids)
```

실행 결과:
```
클러스터 레이블 (앞 10개): [1 2 2 0 1 2 1 1 0 2]

중심점 좌표:
[[-7.09  -8.34]
 [-0.09   5.41]
 [ 5.01  -0.51]]
```

```python
# 시각화
fig, axes = plt.subplots(1, 2, figsize=(12, 5))

# 왼쪽: 클러스터링 전
axes[0].scatter(X[:, 0], X[:, 1], c='gray', alpha=0.5)
axes[0].set_title('클러스터링 전 (레이블 없음)')

# 오른쪽: 클러스터링 후
colors = ['#FF6B6B', '#4ECDC4', '#45B7D1']
for i in range(3):
    mask = labels == i
    axes[1].scatter(X[mask, 0], X[mask, 1],
                    c=colors[i], alpha=0.7, label=f'클러스터 {i}')

axes[1].scatter(centroids[:, 0], centroids[:, 1],
                c='black', marker='*', s=300, label='중심점')
axes[1].set_title('K-Means 클러스터링 결과')
axes[1].legend()

plt.tight_layout()
plt.show()
```

## 5️⃣ 최적 k 찾기 — Elbow Method [↑](#toc)

> "k를 크게 할수록 WCSS는 항상 줄어듭니다. k=데이터 수이면 WCSS=0이니까요. 그렇다면 얼마가 적당할까요?"

**Elbow Method(팔꿈치 방법)**는 WCSS 그래프에서 **꺾이는 지점(팔꿈치)**을 최적 k로 선택합니다.

> **비유**: 계단을 올라갈 때 처음에는 한 걸음에 많이 올라가다가 어느 순간부터는 별 차이가 없는 것처럼, k가 늘어날 때 WCSS 감소량이 급격히 줄어드는 지점이 최적 k입니다.

```python
# Elbow Method 실행
wcss_list = []
k_range = range(1, 11)

for k in k_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    kmeans.fit(X)
    wcss_list.append(kmeans.inertia_)  # inertia_ = WCSS 값

# 그래프 그리기
plt.figure(figsize=(8, 5))
plt.plot(k_range, wcss_list, 'bo-', linewidth=2, markersize=8)
plt.xlabel('클러스터 수 (k)', fontsize=12)
plt.ylabel('WCSS (군집 내 분산 합)', fontsize=12)
plt.title('Elbow Method — 최적 k 찾기', fontsize=14)

# 팔꿈치 지점 표시
plt.axvline(x=3, color='red', linestyle='--', label='최적 k=3')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

print("각 k에 대한 WCSS:")
for k, wcss in zip(k_range, wcss_list):
    print(f"  k={k}: {wcss:.1f}")
```

실행 결과:
```
각 k에 대한 WCSS:
  k=1: 2852.3
  k=2: 1083.7
  k=3: 211.4   ← 여기서 크게 꺾임!
  k=4: 185.2
  k=5: 163.8
  ...
```

k=3에서 WCSS가 크게 줄어들고, 그 이후로는 감소폭이 작아집니다. 따라서 k=3이 최적입니다.

## 6️⃣ 실루엣 점수 [↑](#toc)

**실루엣 점수(Silhouette Score)**는 클러스터링 품질을 평가하는 또 다른 방법입니다.

각 데이터 포인트에 대해:
- **a**: 같은 클러스터 내 다른 점들과의 평균 거리 (작을수록 좋음)
- **b**: 가장 가까운 다른 클러스터 점들과의 평균 거리 (클수록 좋음)

> **직관**: "나는 내 팀 친구들과 얼마나 가깝고, 다른 팀과는 얼마나 멀리 있는가?"

**범위**: -1 ~ +1
- **1에 가까울수록**: 클러스터가 잘 분리됨 (좋음)
- **0에 가까울수록**: 클러스터 경계에 있음
- **음수**: 잘못 분류되었을 가능성

```python
from sklearn.metrics import silhouette_score
import warnings
warnings.filterwarnings('ignore')

# 여러 k에 대해 실루엣 점수 계산
silhouette_scores = []
k_range = range(2, 11)  # k=1은 계산 불가

for k in k_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    labels = kmeans.fit_predict(X)
    score = silhouette_score(X, labels)
    silhouette_scores.append(score)
    print(f"k={k}: 실루엣 점수 = {score:.4f}")

# 시각화
plt.figure(figsize=(8, 5))
plt.plot(k_range, silhouette_scores, 'go-', linewidth=2, markersize=8)
plt.xlabel('클러스터 수 (k)', fontsize=12)
plt.ylabel('실루엣 점수', fontsize=12)
plt.title('실루엣 점수로 최적 k 찾기', fontsize=14)
plt.grid(True, alpha=0.3)
plt.show()
```

실행 결과:
```
k=2: 실루엣 점수 = 0.6282
k=3: 실루엣 점수 = 0.7922  ← 가장 높음!
k=4: 실루엣 점수 = 0.7023
k=5: 실루엣 점수 = 0.6891
...
```

**두 방법 모두 k=3을 가리킵니다.** 실전에서는 Elbow Method와 실루엣 점수를 함께 사용하는 것이 좋습니다.

## 7️⃣ DBSCAN — 소음 속에서 무리 찾기 [↑](#toc)

K-Means의 한계: **k를 미리 정해야 합니다**. 또한 **원형이 아닌 클러스터**나 **이상치**에 취약합니다.

**DBSCAN(Density-Based Spatial Clustering of Applications with Noise)**은 이런 문제를 해결합니다.

> **비유**: 콘서트장에서 팬들이 몰려있는 곳이 클러스터입니다. 혼자 떨어진 사람은 '소음(노이즈)'으로 분류됩니다. k를 미리 정하지 않아도 됩니다.

**핵심 개념:**
- **eps**: 한 점의 이웃으로 볼 최대 거리 ("반경")
- **min_samples**: 클러스터를 이루기 위한 최소 이웃 수
- **핵심 포인트**: 반경 eps 내에 min_samples개 이상의 점이 있는 점
- **노이즈 포인트**: 어떤 클러스터에도 속하지 않는 점 (레이블 = -1)

```python
from sklearn.cluster import DBSCAN
from sklearn.datasets import make_moons

# 원형이 아닌 데이터 (초승달 모양)
X_moon, _ = make_moons(n_samples=300, noise=0.05, random_state=42)

# K-Means vs DBSCAN 비교
fig, axes = plt.subplots(1, 2, figsize=(12, 5))

# K-Means (실패 예시)
km = KMeans(n_clusters=2, random_state=42, n_init=10)
labels_km = km.fit_predict(X_moon)
axes[0].scatter(X_moon[:, 0], X_moon[:, 1], c=labels_km, cmap='Set1', alpha=0.7)
axes[0].set_title('K-Means (k=2) — 잘 안 됨')

# DBSCAN (성공)
db = DBSCAN(eps=0.2, min_samples=5)
labels_db = db.fit_predict(X_moon)
axes[1].scatter(X_moon[:, 0], X_moon[:, 1], c=labels_db, cmap='Set1', alpha=0.7)
axes[1].set_title('DBSCAN — 초승달 모양도 잘 분리!')

plt.tight_layout()
plt.show()

# 노이즈 포인트 수 확인
n_noise = np.sum(labels_db == -1)
n_clusters = len(set(labels_db)) - (1 if -1 in labels_db else 0)
print(f"발견된 클러스터 수: {n_clusters}")
print(f"노이즈 포인트 수: {n_noise}")
```

실행 결과:
```
발견된 클러스터 수: 2
노이즈 포인트 수: 0
```

| | K-Means | DBSCAN |
|--|---------|--------|
| k 지정 | 필요 | 불필요 |
| 클러스터 모양 | 원형 가정 | 자유로운 모양 |
| 이상치 처리 | 취약 | 노이즈로 분류 |
| 속도 | 빠름 | 느릴 수 있음 |

## 8️⃣ 실전: 고객 세분화 [↑](#toc)

Mall Customers 데이터셋을 사용해 실제 고객 세분화를 해봅시다.

```python
import pandas as pd
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score
import matplotlib.pyplot as plt
import seaborn as sns

# 데이터 불러오기 (Google Colab에서 실행)
url = "https://raw.githubusercontent.com/dsrscientist/dataset1/master/Mall_Customers.csv"
df = pd.read_csv(url)

print("데이터 shape:", df.shape)
print("\n처음 5행:")
df.head()
```

실행 결과:
```
데이터 shape: (200, 5)

처음 5행:
   CustomerID  Genre  Age  Annual Income (k$)  Spending Score (1-100)
0           1   Male   19                  15                      39
1           2   Male   21                  15                      81
2           3  Female  20                  16                       6
...
```

```python
# 분석에 사용할 피처 선택: 연소득 + 지출점수
X_mall = df[['Annual Income (k$)', 'Spending Score (1-100)']].values

# 스케일링
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X_mall)

# Elbow Method로 최적 k 찾기
wcss = []
for k in range(1, 11):
    km = KMeans(n_clusters=k, random_state=42, n_init=10)
    km.fit(X_scaled)
    wcss.append(km.inertia_)

plt.figure(figsize=(8, 4))
plt.plot(range(1, 11), wcss, 'bo-', linewidth=2)
plt.xlabel('클러스터 수 (k)')
plt.ylabel('WCSS')
plt.title('Mall Customers — Elbow Method')
plt.axvline(x=5, color='red', linestyle='--', label='최적 k=5')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

```python
# k=5로 최종 클러스터링
kmeans_final = KMeans(n_clusters=5, random_state=42, n_init=10)
df['Cluster'] = kmeans_final.fit_predict(X_scaled)

# 시각화
plt.figure(figsize=(10, 7))
colors = ['#FF6B6B', '#4ECDC4', '#45B7D1', '#96CEB4', '#FFEAA7']
cluster_names = ['검약형', '고소득 저지출', '중간형', '저소득 고지출', '고소득 고지출']

for i in range(5):
    mask = df['Cluster'] == i
    plt.scatter(df[mask]['Annual Income (k$)'],
                df[mask]['Spending Score (1-100)'],
                c=colors[i], s=80, alpha=0.8, label=f'클러스터 {i}: {cluster_names[i]}')

plt.xlabel('연간 소득 (k$)', fontsize=12)
plt.ylabel('지출 점수 (1-100)', fontsize=12)
plt.title('고객 세분화 결과', fontsize=14)
plt.legend(loc='upper left', fontsize=9)
plt.grid(True, alpha=0.3)
plt.show()
```

```python
# 클러스터별 통계
print("=== 클러스터별 평균 특성 ===")
cluster_stats = df.groupby('Cluster')[['Annual Income (k$)', 'Spending Score (1-100)', 'Age']].mean()
cluster_stats.columns = ['평균 소득', '평균 지출점수', '평균 나이']

print(df['Cluster'].value_counts().sort_index())
```

<a id="9"></a>
## 9️⃣ 정리 [↑](#toc)

| 개념 | 핵심 내용 | sklearn 코드 |
|------|-----------|-------------|
| K-Means | 깃발(중심점)을 반복 이동하며 클러스터 형성 | `KMeans(n_clusters=k)` |
| Elbow Method | WCSS 그래프의 꺾이는 지점 = 최적 k | `kmeans.inertia_` |
| 실루엣 점수 | 클러스터 응집도/분리도 평가 (-1~1) | `silhouette_score(X, labels)` |
| DBSCAN | 밀도 기반, k 불필요, 노이즈 처리 가능 | `DBSCAN(eps=0.2, min_samples=5)` |
| 고객 세분화 | 연소득+지출점수로 고객 그룹 발견 | — |

**K-Means 주요 파라미터:**
- `n_clusters`: 클러스터 수 (가장 중요!)
- `random_state`: 재현성 확보
- `n_init`: 초기화 반복 횟수 (기본 10, 높을수록 안정적)
- `max_iter`: 최대 반복 횟수 (기본 300)

**언제 어떤 알고리즘을?**
- 클러스터 수를 알고, 원형 모양 예상 → **K-Means**
- 모양을 모르고, 이상치가 있을 것 같다 → **DBSCAN**
