### 오늘의 키워드 (칠판에 미리 적어 둡니다)

```

전이학습(Transfer Learning) — 이미 학습된 모델의 지식을 가져와서 우리 문제에 적용
Feature Extraction — 가져온 모델을 통째로 얼리고(freeze), 분류기만 새로 학습
Fine-tuning — 일부 레이어를 녹여서(unfreeze) 미세 조정
```

**deepcopy 설명:**

> "`best_state = deepcopy(model.state_dict())` — 이게 왜 deepcopy인가요? 모델 가중치는 텐서(참조 객체)이기 때문에, 단순 대입(=)을 하면 원본이 바뀔 때 같이 바뀝니다. deepcopy는 완전히 별도 복사본을 만들어서, 나중에 모델이 더 학습되어 나빠져도 best 시점의 가중치를 보존합니다."

## §8. 전이학습 개념 설명 (10분) ★ 가장 중요한 이론

> "CNN도 마찬가지입니다. ImageNet이라는 거대한 데이터셋(120만 장, 1000개 클래스)으로 학습된 ResNet18 모델이 있어요. 이 모델은 이미 '눈', '코', '귀', '털 질감' 같은 시각적 특징을 알고 있습니다."

```
ResNet18 (ImageNet으로 학습됨)
┌─────────────────────────────────────┐

│ 깊은층: 얼굴 구조, 몸체 형태         │ ← ImageNet 특화
```

## §9. 실험 3 — 전이학습 Feature Extraction (7분) ★ 핵심 실험

**실행 전 설명:**

> "이제 진짜로 해봅시다. ResNet18의 사전학습 가중치를 불러오고, 모든 레이어를 freeze하고, 마지막 분류기만 교체합니다."

**코드 주요 포인트:**

```python
for param in transfer_model.parameters():
    param.requires_grad = False       # "배운 거 건드리지 마!"
transfer_model.fc = nn.Linear(..., 1) # "마지막만 새로 달아!"
```

> "`requires_grad = False`가 freeze입니다. 이 파라미터는 학습할 때 업데이트하지 않겠다는 뜻이에요."

**실행 후 — 충격의 결과:**

> "trainable params: 513개. Baseline CNN은 __만 개였죠? 그런데 test accuracy는...?"

> "**파라미터가 수백 배 적은데 성능은 더 좋다!** 이게 전이학습의 위력입니다."

**기록:** `전이학습 효과: +__%p` (칠판에 추가)

**칠판 요약 (이 시점에서):**

```
모델               test acc   학습 파라미터
──────────────────────────────────────────
Baseline CNN       ___%       ~300K
증강 CNN           ___%       ~300K        (+__%p)
전이학습 ResNet18   ___%       513          (+__%p) ★
```

## §10. 실험 4 — Fine-tuning (5분, 선택)

**실행 전 설명:**

> "마지막 실험은 선택입니다. Feature Extraction에서 한 발 더 나가서, 깊은 층(layer4)도 살짝 풀어서 미세 조정합니다."

> "**주의**: 이 실험에서는 두 가지가 동시에 바뀝니다 — layer4 unfreeze + 증강 적용. 그래서 순수하게 'unfreeze 효과만'은 아닙니다. 실전에서 흔히 쓰는 조합을 보여주는 겁니다."

**코드 주요 포인트:**

```python
# 차등 학습률!
optimizer = optim.Adam([
    {'params': finetune_model.layer4.parameters(), 'lr': 1e-4},  # 낮게
    {'params': finetune_model.fc.parameters(), 'lr': 1e-3}       # 높게
])
```

> "왜 학습률을 다르게 하나요? layer4는 이미 좋은 가중치를 가지고 있어서, 크게 바꾸면 학습한 지식이 날아갑니다. 살짝만 조정하는 거예요. fc는 새로 만든 거니까 빨리 배워도 됩니다."

## §13. 마무리 (5분)

**칠판 최종 정리:**

```
오늘 배운 것:
1. 같은 모델 + 증강 = 일반화 성능 향상 (과적합 감소)
2. 전이학습 = 적은 파라미터로 높은 성능 (ImageNet 지식 재활용)
3. Fine-tuning = 전이학습의 확장 (일부 레이어를 미세 조정)
4. 숫자만 보지 말고 오분류 이미지를 분석하라
5. 학습된 모델 → Streamlit 앱 → Cloud 배포 = 누구나 사용 가능
```

|------|------|
