# RAG Retrieval 심화
{: .no_toc }

{: .fs-6 .fw-300 }

---

{: .warning }
> **심화 과정**: 이 과정은 검색 증강 생성(RAG)의 **고급 기법**을 다룹니다. BM25, RRF, HyDE, 리랭킹 등 정보 검색(IR) 이론이 포함되어 있어 **중급 이상** 수준입니다.

## 선수 학습

이 과정을 시작하기 전에 다음을 먼저 수강하세요:
- [LangChain](/llm/langchain) 과정의 **10~13번** (RAG 개요, 임베딩, 벡터 검색)
- Python 기본, Pinecone 계정

## 학습 경로

| 순서 | 주제 | 난이도 | 핵심 내용 |
|------|------|--------|----------|
| 1 | 인덱싱 | ★★☆ | Pinecone + LangChain으로 문서 벡터화 |
| 2 | BM25 vs Dense | ★★★ | 키워드 검색과 의미 검색 비교 실험 |
| 3 | RRF | ★★★ | 하이브리드 검색 (BM25 + Dense 결합) |
| 4 | HyDE | ★★★★ | LLM으로 가상 문서 생성 후 검색 |
| 5 | Cohere 리랭킹 | ★★★ | 외부 리랭킹 서비스로 검색 품질 개선 |
| 6 | 컨텍스트 압축 | ★★★ | LLM으로 문서 요약 후 검색 |
| 7-1 | 메타데이터 인덱싱 | ★★★ | 메타데이터를 활용한 정밀 검색 |
| 7-2 | SelfQueryRetriever | ★★★★ | LLM이 자동으로 메타데이터 필터 생성 |
| 8 | 멀티홉 검색 | ★★★★ | 2단계 순차 검색으로 복합 질문 처리 |

## LangChain RAG 기본과의 관계

| LangChain 10~13 (기본) | 이 과정 (심화) |
|------------------------|--------------|
| RAG 개념 이해 | 검색 기법 비교 실험 |
| 벡터DB 기본 사용 | 하이브리드 검색 (BM25+Dense) |
| 단순 similarity_search | HyDE, 리랭킹, 압축 등 고급 기법 |
| 메타데이터 기본 필터 | SelfQueryRetriever, 멀티홉 |

> **비전공자 가이드**: LangChain의 RAG 기본(10~13)과 프로젝트(14~15)만으로도 충분히 RAG를 활용할 수 있습니다. 이 심화 과정은 **검색 품질을 더 높이고 싶을 때** 참고하세요.
