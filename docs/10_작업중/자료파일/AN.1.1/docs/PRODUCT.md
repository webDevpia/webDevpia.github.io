# Todo Manager - 제품 요구사항 명세 (통합 실습 완성본)

## 제품 개요

**Todo Manager**는 Python으로 구현된 명령줄 기반 할일 관리 도구입니다.

---

## 구현된 기능

| 기능 | 설명 | 상태 |
|------|------|------|
| 할일 추가 | 제목을 입력하여 새 할일 항목을 생성한다 | ✅ |
| 전체 조회 | 등록된 모든 할일을 목록으로 표시한다 | ✅ |
| 완료 처리 | ID를 지정하여 할일을 완료 상태로 변경한다 | ✅ |
| 삭제 | ID를 지정하여 할일을 목록에서 제거한다 | ✅ |
| 상태별 필터링 | 상태(pending/done)로 할일 목록을 필터링한다 | [ ] |

---

## 비기능 요구사항

- Python 3.11 이상에서 동작한다.
- 외부 라이브러리 없이 표준 라이브러리만 사용한다 (실습 기본 버전).
- 단위 테스트 커버리지 80% 이상을 유지한다.
- 모든 공개 메서드에 한국어 docstring을 작성한다.

---

## 제약 사항

- 할일 제목은 빈 문자열일 수 없다.
- 존재하지 않는 ID에 대한 조작 시 `ValueError` 를 발생시킨다.
- ID는 삭제 후에도 재사용하지 않는다.
