<a id="part1"></a>

## 1️⃣ 프로젝트 개요 [↑](#toc)

Day 1과 Day 2에서 배운 모든 개념을 통합하여 **실용적인 여행 플래너**를 완성합니다.

### 프로젝트 아키텍처

```
[사용자 입력]
  여행지, 기간, 예산, 여행 스타일
         ↓
[Flow: TravelPlannerFlow]
         ↓
  ┌──────────────────────────────────┐
  │  1단계: 여행지 리서치 Crew         │
  │  → 에이전트: 여행지 리서치 전문가   │
  │  → 도구: SerperDevTool (검색)     │
  └──────────────────────────────────┘
         ↓
  ┌──────────────────────────────────┐
  │  2단계: 예산 분석 Crew             │
  │  → 에이전트: 예산 분석가           │
  │  → 도구: Calculator              │
  └──────────────────────────────────┘
         ↓
  ┌──────────────────────────────────┐
  │  3단계: 계획 작성 Crew             │
  │  → 에이전트: 여행 계획 작가        │
  │  → 구조화 출력: TravelPlan         │
  │  → Guardrail: 예산/관광지 검증    │
  └──────────────────────────────────┘
         ↓
[최종 출력: TravelPlan Pydantic 객체]
```

### 사용되는 개념 목록

| 개념 | 출처 | 역할 |
|------|------|------|
| Agent / Task / Crew | 01~03장 | 기본 구성 요소 |
| 도구(Tools) | 04장 | 검색, 계산 |
| Process.sequential | 05장 | 에이전트 순차 실행 |
| Memory | 08장 | 대화 맥락 유지 |
| Flow | 09장 | 단계별 워크플로 연결 |
| 구조화 출력 | 11장 | TravelPlan Pydantic 모델 |
| Guardrail | 11장 | 출력 품질 검증 |
| Planning | 11장 | 실행 전 계획 수립 |

## 3️⃣ Flow 구성 [↑](#toc)

### 상태 모델과 출력 모델

```python
from pydantic import BaseModel, Field
from typing import List, Optional

# ─── Flow 상태 모델 ──────────────────────────────────────────
class TravelFlowState(BaseModel):
    # 입력
    destination: str = ""
    duration: str = ""
    budget: int = 0
    style: str = ""

    # 단계별 결과
    research_result: str = ""
    budget_analysis: str = ""

    # 최종 출력
    final_plan: Optional[dict] = None

# ─── 구조화 출력 모델 ─────────────────────────────────────────
class TravelPlan(BaseModel):
    destination: str = Field(description="여행지")
    duration: str = Field(description="여행 기간")
    total_budget: int = Field(description="총 예산 (원)")
    budget_breakdown: dict = Field(
        description="예산 항목별 분류",
        default_factory=dict
    )
    highlights: List[str] = Field(description="여행지 주요 특징 3가지")
    attractions: List[str] = Field(description="추천 관광지 (최소 3곳)")
    restaurants: List[str] = Field(description="추천 맛집 (최소 2곳)")
    itinerary: List[str] = Field(description="일자별 상세 일정")
    transportation: str = Field(description="이동 수단 및 교통 팁")
    accommodation: str = Field(description="추천 숙박 유형 및 지역")
    tips: List[str] = Field(description="실용적인 여행 팁")
```

### Flow 흐름 설계

```
@start: set_inputs()
    여행지, 기간, 예산, 스타일 초기화
         ↓
@listen(set_inputs): research_phase()
    여행지 리서치 Crew 실행
         ↓
@listen(research_phase): budget_phase()
    예산 분석 Crew 실행
         ↓
@listen(budget_phase): planning_phase()
    계획 작성 Crew 실행 + 구조화 출력 + Guardrail
         ↓
    최종 TravelPlan 반환
```

## 4️⃣ 전체 코드 [↑](#toc)

아래 코드를 `12_travel_planner.py`로 저장하고 바로 실행할 수 있습니다.

```python
# 파일명: 12_travel_planner.py
# ─── 모든 import ───────────────────────────────────────────────
from crewai import Agent, Task, Crew, Process
from crewai.flow.flow import Flow, listen, start
from crewai.task import TaskOutput
from crewai_tools import SerperDevTool
from pydantic import BaseModel, Field
from typing import List, Optional
from dotenv import load_dotenv
import os

load_dotenv()

# ─────────────────────────────────────────────────────────────
# 1. 모델 정의
# ─────────────────────────────────────────────────────────────

class TravelFlowState(BaseModel):
    destination: str = ""
    duration: str = ""
    budget: int = 0
    style: str = ""
    research_result: str = ""
    budget_analysis: str = ""

class TravelPlan(BaseModel):
    destination: str = Field(description="여행지")
    duration: str = Field(description="여행 기간")
    total_budget: int = Field(description="총 예산 (원)")
    budget_breakdown: dict = Field(
        description="예산 항목별 분류",
        default_factory=dict
    )
    highlights: List[str] = Field(description="여행지 주요 특징")
    attractions: List[str] = Field(description="추천 관광지 최소 3곳")
    restaurants: List[str] = Field(description="추천 맛집 최소 2곳")
    itinerary: List[str] = Field(description="일자별 일정")
    transportation: str = Field(description="교통 안내")
    accommodation: str = Field(description="숙박 안내")
    tips: List[str] = Field(description="실용적인 팁")

# ─────────────────────────────────────────────────────────────
# 2. Guardrail
# ─────────────────────────────────────────────────────────────

def validate_plan(output: TaskOutput) -> tuple[bool, str]:
    """여행 계획 품질 검증 Guardrail"""
    try:
        plan = output.pydantic
    except Exception as e:
        return False, f"형식 오류: {e}"

    if plan.total_budget <= 0:
        return False, "총 예산이 0보다 커야 합니다."

    if len(plan.attractions) < 3:
        return False, f"관광지를 최소 3곳 이상 포함해야 합니다. (현재: {len(plan.attractions)}곳)"

    if len(plan.restaurants) < 2:
        return False, f"맛집을 최소 2곳 이상 포함해야 합니다. (현재: {len(plan.restaurants)}곳)"

    if len(plan.itinerary) == 0:
        return False, "일자별 일정이 비어 있습니다."

    return True, ""

# ─────────────────────────────────────────────────────────────
# 3. Flow 정의
# ─────────────────────────────────────────────────────────────

class TravelPlannerFlow(Flow[TravelFlowState]):

    @start()
    def set_inputs(self):
        """Flow 시작: 입력값 확인"""
        print(f"\n[여행 플래너 시작]")
        print(f"여행지: {self.state.destination}")
        print(f"기간: {self.state.duration}")
        print(f"예산: {self.state.budget:,}원")
        print(f"스타일: {self.state.style}\n")
        return self.state.destination

    @listen(set_inputs)
    def research_phase(self):
        """1단계: 여행지 리서치"""
        print("[1단계] 여행지 리서치 시작...")

        # 도구 설정
        search_tool = SerperDevTool()

        # 에이전트
        researcher = Agent(
            role="여행지 리서치 전문가",
            goal="주어진 여행지의 핵심 정보를 수집한다",
            backstory=(
                "세계 각지를 직접 방문한 여행 블로거로, "
                "신뢰할 수 있는 최신 여행 정보를 제공합니다."
            ),
            tools=[search_tool],
            verbose=False   # Flow 실행 중 노이즈 최소화
        )

        # 태스크
        task = Task(
            description=(
                f"'{self.state.destination}' 여행에 대해 다음 정보를 조사하세요:\n"
                f"1. 주요 관광 명소 (최소 5곳)\n"
                f"2. 현지 유명 맛집 (최소 3곳)\n"
                f"3. 이동 수단 및 교통 팁\n"
                f"4. 숙박 추천 지역\n"
                f"5. 여행 기간 '{self.state.duration}'에 맞는 일정 팁\n"
                f"6. 여행 스타일 '{self.state.style}'에 적합한 추천사항"
            ),
            expected_output=(
                "관광 명소, 맛집, 교통, 숙박, 일정 팁이 포함된 "
                "상세 리서치 결과 (500자 이상)"
            ),
            agent=researcher
        )

        crew = Crew(
            agents=[researcher],
            tasks=[task],
            verbose=False
        )

        result = crew.kickoff()
        self.state.research_result = str(result)
        print("[1단계] 리서치 완료")
        return self.state.research_result

    @listen(research_phase)
    def budget_phase(self):
        """2단계: 예산 분석"""
        print("[2단계] 예산 분석 시작...")

        # 에이전트
        analyst = Agent(
            role="여행 예산 분석가",
            goal="여행 예산을 항목별로 최적 배분한다",
            backstory=(
                "재무 컨설턴트 출신의 여행 예산 전문가입니다. "
                "현실적인 예산 계획으로 여행자의 만족도를 높입니다."
            ),
            verbose=False
        )

        # 태스크
        task = Task(
            description=(
                f"다음 여행 조건에 맞는 예산 분석을 수행하세요:\n\n"
                f"- 여행지: {self.state.destination}\n"
                f"- 기간: {self.state.duration}\n"
                f"- 총 예산: {self.state.budget:,}원\n"
                f"- 여행 스타일: {self.state.style}\n\n"
                f"[리서치 참고 자료]\n{self.state.research_result}\n\n"
                f"숙박비, 교통비, 식비, 관광/체험비, 기타 비용으로 예산을 분배하고 "
                f"각 항목의 예상 금액과 절약 팁을 제시하세요."
            ),
            expected_output=(
                "예산 항목별 분배 (숙박/교통/식비/관광/기타), "
                "각 항목 예상 금액, 비용 절약 팁"
            ),
            agent=analyst
        )

        crew = Crew(
            agents=[analyst],
            tasks=[task],
            verbose=False
        )

        result = crew.kickoff()
        self.state.budget_analysis = str(result)
        print("[2단계] 예산 분석 완료")
        return self.state.budget_analysis

    @listen(budget_phase)
    def planning_phase(self):
        """3단계: 최종 여행 계획 작성 (구조화 출력 + Guardrail)"""
        print("[3단계] 여행 계획 작성 시작...")

        # 에이전트
        planner = Agent(
            role="여행 계획 작가",
            goal="모든 정보를 종합하여 완성도 높은 여행 계획을 작성한다",
            backstory=(
                "10년 경력의 여행 작가로, 수십 권의 가이드북을 집필했습니다. "
                "복잡한 정보를 독자 친화적인 일정으로 정리하는 전문가입니다."
            ),
            verbose=True
        )

        # 태스크 (구조화 출력 + Guardrail)
        task = Task(
            description=(
                f"다음 자료를 바탕으로 완성된 여행 계획을 작성하세요:\n\n"
                f"[여행 조건]\n"
                f"- 여행지: {self.state.destination}\n"
                f"- 기간: {self.state.duration}\n"
                f"- 총 예산: {self.state.budget:,}원\n"
                f"- 스타일: {self.state.style}\n\n"
                f"[리서치 결과]\n{self.state.research_result}\n\n"
                f"[예산 분석]\n{self.state.budget_analysis}\n\n"
                f"관광지 3곳 이상, 맛집 2곳 이상, 일자별 상세 일정을 반드시 포함하세요."
            ),
            expected_output=(
                "모든 필드가 채워진 완성된 여행 계획 "
                "(관광지 3곳+, 맛집 2곳+, 일자별 일정 포함)"
            ),
            output_pydantic=TravelPlan,   # 구조화 출력
            guardrail=validate_plan,       # 품질 검증
            agent=planner
        )

        crew = Crew(
            agents=[planner],
            tasks=[task],
            planning=True,   # 실행 전 계획 수립
            verbose=True
        )

        crew.kickoff()
        plan: TravelPlan = task.output.pydantic
        self.state.final_plan = plan.model_dump()
        print("[3단계] 여행 계획 완성!")
        return plan

# ─────────────────────────────────────────────────────────────
# 4. 실행 및 결과 출력
# ─────────────────────────────────────────────────────────────

def print_travel_plan(plan: TravelPlan):
    """여행 계획 출력 포맷"""
    print("\n" + "=" * 60)
    print(f"여행 플래너 결과")
    print("=" * 60)
    print(f"여행지: {plan.destination}")
    print(f"기간: {plan.duration}")
    print(f"총 예산: {plan.total_budget:,}원")

    print("\n[예산 항목별 분류]")
    for item, amount in plan.budget_breakdown.items():
        if isinstance(amount, (int, float)):
            print(f"  {item}: {int(amount):,}원")
        else:
            print(f"  {item}: {amount}")

    print("\n[여행지 특징]")
    for h in plan.highlights:
        print(f"  ✓ {h}")

    print("\n[추천 관광지]")
    for i, att in enumerate(plan.attractions, 1):
        print(f"  {i}. {att}")

    print("\n[추천 맛집]")
    for i, rest in enumerate(plan.restaurants, 1):
        print(f"  {i}. {rest}")

    print("\n[일자별 일정]")
    for day in plan.itinerary:
        print(f"  {day}")

    print(f"\n[교통 안내]\n  {plan.transportation}")
    print(f"\n[숙박 안내]\n  {plan.accommodation}")

    print("\n[여행 팁]")
    for tip in plan.tips:
        print(f"  • {tip}")
    print("=" * 60)

if __name__ == "__main__":
    # Flow 실행
    flow = TravelPlannerFlow()
    result = flow.kickoff(inputs={
        "destination": "교토",
        "duration": "3박4일",
        "budget": 1200000,
        "style": "역사와 문화 중심의 전통 여행"
    })

    # 결과 출력
    if isinstance(result, TravelPlan):
        print_travel_plan(result)
    else:
        print("\n[최종 결과]")
        print(result)
```

### 핵심 요약

- Flow로 3단계(리서치 → 예산 분석 → 계획 작성)를 명확히 분리
- 각 단계에서 독립적인 Crew를 실행하여 역할 분리
- `self.state`로 단계 간 데이터를 안전하게 전달
- 구조화 출력으로 최종 결과를 Python 객체로 처리
- Guardrail로 출력 품질을 자동 보장
