## 1. 같은 AI, 다른 결과 [↑](#toc)

### Before vs After — 프롬프트가 결과를 결정한다

똑같은 AI에게 똑같은 주제를 물어도, **어떻게 묻느냐**에 따라 결과가 완전히 달라집니다.

| | 모호한 프롬프트 | 잘 구조화된 프롬프트 |
|---|---|---|
| **입력** | "파이썬 리스트 알려줘" | "파이썬을 처음 배우는 학생입니다. 리스트와 딕셔너리의 차이를 실생활 비유로 설명하고, 각각 언제 쓰는지 코드 예시와 함께 표로 정리해줘." |
| **결과** | 일반적이고 두루뭉술한 설명 | 비유 + 용도별 코드 예시 + 비교표 |

> **ChatGPT에서 바로 해보기**: 위 두 프롬프트를 ChatGPT에 각각 입력하고 결과를 비교해보세요.

### 프롬프트 엔지니어링이란?

**AI에게 원하는 결과를 정확히 얻어내기 위해 입력(프롬프트)을 설계하는 기술**입니다.

> **비유**: AI는 매우 유능한 신입 직원입니다. 능력은 뛰어나지만, **지시가 모호하면 제멋대로** 일합니다. 명확한 지시를 주면 기대 이상의 결과를 냅니다.

### 프로그래밍/딥러닝 학습에서의 활용

| 상황 | 활용 예시 |
|------|----------|
| **개념 이해** | "CNN과 RNN의 차이를 비유로 설명해줘" |
| **에러 해결** | "이 에러 메시지의 원인과 해결법을 알려줘" |
| **코드 리뷰** | "이 코드의 문제점을 찾고 개선된 버전을 보여줘" |
| **프로젝트 설계** | "YOLO로 실시간 객체 탐지 프로젝트의 단계를 정리해줘" |
| **시험 준비** | "딥러닝 핵심 개념 10개를 퀴즈 형식으로 만들어줘" |
| **문서 작성** | "README.md를 작성해줘. 설치 방법과 사용법을 포함해서" |

## 2. LLM이란? [↑](#toc)

### 한 줄 정의

> LLM(Large Language Model)은 **"다음에 올 단어를 예측"**하는 것을 반복하여 문장을 생성하는 AI 모델입니다.

```
입력: "오늘 날씨가"
AI 예측: "좋다" (확률 35%) / "맑다" (30%) / "춥다" (20%) / ...
→ "좋다"를 선택
→ "오늘 날씨가 좋다"에서 다시 다음 단어 예측...
```

### 2026년 주요 모델

| 모델 | 제공사 | 입력 비용 (1M토큰) | 특징 | 교육 실습용 |
|------|--------|:---:|------|:---:|
| **GPT-4.1-nano** | OpenAI | **$0.10** | 최저가, 빠름, 교육에 최적 | ✅ 권장 |
| GPT-4.1-mini | OpenAI | 중간 | 코딩 특화, 명령어 따르기 개선 | ✅ |
| GPT-5.4-nano | OpenAI | $0.20 | 최신 플래그십의 경량 버전 | - |
| Claude Sonnet 4.6 | Anthropic | $3.00 | 1M 토큰, 코드 분석 강점 | - |
| Gemini 3 Flash | Google | 저가 | 멀티모달, 고속 | - |

> 이 과정에서는 **GPT-4.1-nano**를 사용합니다. OpenAI에서 가장 저렴하면서($0.10/1M 입력 토큰) 교육 실습에 충분한 성능을 제공합니다.

## 3. 환경 설정 [↑](#toc)

### 방법 1: ChatGPT 웹 (코드 없이 실습)

1. [chat.openai.com](https://chat.openai.com) 접속
2. 계정 생성/로그인
3. 채팅창에 프롬프트 입력 → 바로 결과 확인


### 방법 2: OpenAI API (코드로 자동화)

Google Colab에서 실행합니다.

**API 키 발급:**
1. [platform.openai.com](https://platform.openai.com) 접속 → 로그인
2. API Keys 메뉴 → "Create new secret key"
3. 생성된 키를 복사 (한 번만 표시됨!)

**Colab에서 API 키 등록:**
1. Colab 왼쪽 사이드바의 `🔑` (Secrets) 아이콘 클릭
2. Name: `OPENAI_API_KEY`, Value: 복사한 키 붙여넣기
3. "노트북 액세스" 토글 켜기

**첫 번째 API 호출:**

```python
!pip install openai -q

import os
from openai import OpenAI
from google.colab import userdata

# Colab Secrets에서 API 키 가져오기
os.environ['OPENAI_API_KEY'] = userdata.get('OPENAI_API_KEY')

# OpenAI 클라이언트 생성
client = OpenAI()

# 첫 번째 API 호출
response = client.chat.completions.create(
    model="gpt-4.1-nano",
    messages=[{"role": "user", "content": "프롬프트 엔지니어링이 뭔지 한 문장으로 설명해줘"}]
)

# 결과 출력
print(response.choices[0].message.content)
```

## 4. 주요 파라미터 체험 [↑](#toc)

AI의 응답 스타일을 조절하는 "다이얼"이 있습니다.

| 파라미터 | 범위 | 역할 | 비유 |
|---------|------|------|------|
| `temperature` | 0~2 | 창의성 조절 | 0=교과서, 1=수필, 2=시 |
| `max_tokens` | 1~ | 최대 응답 길이 | 원고지 매수 제한 |
| `top_p` | 0~1 | 단어 선택 범위 | 0.1=상위 10%만, 1.0=전부 고려 |

### 실습: temperature 비교

```python
prompt = "파이썬에서 for문과 while문의 차이를 한 문장으로 설명해줘."

for temp in [0.0, 0.7, 1.5]:
    response = client.chat.completions.create(
        model="gpt-4.1-nano",
        messages=[{"role": "user", "content": prompt}],
        temperature=temp,
        max_tokens=100
    )
    print(f"\n[temperature={temp}]")
    print(response.choices[0].message.content)
```

> **관찰 포인트**:
> - `temperature=0`: 매번 **같은** 결과 (결정적)
> - `temperature=0.7`: 자연스럽고 약간의 변화
> - `temperature=1.5`: 매번 **다른** 결과 (창의적이지만 불안정)

> **ChatGPT에서 해보기**: ChatGPT 웹에서도 같은 질문을 여러 번 해보세요. 매번 조금씩 다른 답변이 나오는 이유가 temperature 때문입니다.

> **확인**: 계약서나 법률 문서를 작성할 때는 temperature를 높게 설정해야 할까요, 낮게 설정해야 할까요? 왜 그런가요?
