## 비전공자를 위한 학습 경로

```
Level 1 — LLM 기초
  LangChain 01~05: 설정 → 개요 → 캐싱 → 템플릿 → 파서
  "AI에게 질문하고 구조화된 답을 받을 수 있다"

Level 2 — LLM 활용
  LangChain 07~09: LCEL → 도구 → 프로젝트
  "외부 API를 연결하고 파이프라인을 조합할 수 있다"

Level 3 — RAG (검색증강생성)
  LangChain 10~15: RAG 개요 → 벡터검색 → 소믈리에/주택청약 프로젝트
  "회사 문서를 넣으면 답해주는 AI를 만들 수 있다"

Level 4 — AI 에이전트
  LangGraph 01~05: 그래프 개념 → 챗봇 → 도구 → 메모리
  "스스로 판단하고 도구를 쓰는 AI 에이전트를 만들 수 있다"

Level 5 — 심화 (선택)
  LangGraph 06~09: HITL, 커스텀 상태, 멀티 에이전트
  RAG Retrieval: BM25, RRF, HyDE, 리랭킹 (검색 품질 최적화)
```
