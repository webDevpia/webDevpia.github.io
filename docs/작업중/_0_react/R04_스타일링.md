# 03장. 스타일링 — Tailwind CSS

> "Tailwind = 옷에 이름표 붙이기 — 'bg-blue-500'이라고 쓰면 파란 배경이 됩니다."

옷을 사면 세탁 방법이 적힌 이름표가 붙어 있습니다. Tailwind는 이것과 비슷합니다. `className`에 원하는 속성의 이름을 쓰면 그대로 적용됩니다. CSS 파일을 따로 열 필요가 없습니다.

---

## 학습 목표

- Tailwind CSS의 유틸리티 클래스 철학을 이해한다
- 색상, 여백, 크기, 레이아웃 관련 기본 클래스를 활용할 수 있다
- 반응형 디자인을 위한 브레이크포인트를 사용할 수 있다
- hover, focus 등 상태 변형 클래스를 적용할 수 있다
- 아름다운 프로필 카드를 Tailwind만으로 만들 수 있다

## 1️⃣ 왜 Tailwind CSS인가? [↑](#toc)

### 문제: 스타일링 방법이 너무 많습니다

React에서 스타일을 적용하는 방법은 여러 가지가 있습니다.

**방법 1: 인라인 스타일**

```jsx
// 자바스크립트 객체로 작성 — camelCase 필수
<div style={{ backgroundColor: "blue", fontSize: "16px", padding: "16px" }}>
  내용
</div>
```

단점: 자동완성이 없고, hover/media query를 쓸 수 없습니다.

**방법 2: 별도 CSS 파일**

```css
/* Card.css */
.card {
  background-color: white;
  border-radius: 12px;
  padding: 24px;
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

.card:hover {
  box-shadow: 0 10px 20px rgba(0, 0, 0, 0.15);
}
```

```jsx
import "./Card.css";

function Card() {
  return <div className="card">내용</div>;
}
```

단점: 파일을 두 개 관리해야 하고, 클래스명이 전역 공간에서 충돌할 수 있습니다.

**방법 3: CSS Modules**

```css
/* Card.module.css */
.card { ... }
```

```jsx
import styles from "./Card.module.css";

function Card() {
  return <div className={styles.card}>내용</div>;
}
```

단점: 설정이 복잡하고, 클래스명이 자동으로 난독화됩니다.

**방법 4: Tailwind CSS (이 강의의 선택)**

```jsx
function Card() {
  return (
    <div className="bg-white rounded-xl p-6 shadow-md hover:shadow-xl transition-shadow">
      내용
    </div>
  );
}
```

### Tailwind의 장점

| 항목 | 설명 |
|------|------|
| **함께 위치** | 컴포넌트와 스타일이 같은 파일에 있어 맥락을 파악하기 쉽습니다 |
| **충돌 없음** | 전역 클래스명 충돌 걱정이 없습니다 |
| **일관성** | 디자인 시스템(색상, 간격, 크기)이 내장되어 있습니다 |
| **자동완성** | VS Code 확장(Tailwind IntelliSense)으로 편하게 작성합니다 |
| **빌드 최적화** | 사용된 클래스만 포함해 최종 CSS 파일이 작습니다 |

## 2️⃣ Tailwind 기본 클래스 [↑](#toc)

### 색상

Tailwind는 색상 이름과 숫자(50~950)로 색상을 표현합니다. 숫자가 클수록 어둡습니다.

```jsx
{/* 배경색 */}
<div className="bg-blue-500">파란 배경</div>
<div className="bg-gray-100">연한 회색 배경</div>
<div className="bg-white">흰 배경</div>

{/* 글자색 */}
<p className="text-blue-600">파란 글자</p>
<p className="text-gray-500">회색 글자</p>
<p className="text-white">흰 글자</p>

{/* 테두리색 */}
<div className="border border-gray-200">회색 테두리</div>
```

자주 쓰는 색상: `slate`, `gray`, `red`, `orange`, `yellow`, `green`, `blue`, `indigo`, `purple`, `pink`

### 간격 (Spacing)

Tailwind의 간격 단위는 `4 = 1rem = 16px`입니다.

```jsx
{/* 패딩 — 안쪽 여백 */}
<div className="p-4">상하좌우 모두 16px</div>
<div className="px-6 py-3">좌우 24px, 상하 12px</div>
<div className="pt-8">위쪽만 32px</div>

{/* 마진 — 바깥 여백 */}
<div className="m-4">상하좌우 모두 16px</div>
<div className="mx-auto">좌우 자동 (가운데 정렬)</div>
<div className="mt-6 mb-4">위 24px, 아래 16px</div>

{/* gap — Flex/Grid 자식 간격 */}
<div className="flex gap-4">
  <div>A</div>
  <div>B</div>
</div>
```

### 크기 (Sizing)

```jsx
{/* 너비 */}
<div className="w-full">부모 너비 100%</div>
<div className="w-64">256px (= 64 × 4)</div>
<div className="w-1/2">50%</div>
<div className="max-w-md">최대 448px</div>
<div className="max-w-screen-lg">최대 1024px</div>

{/* 높이 */}
<div className="h-screen">뷰포트 높이 100%</div>
<div className="h-48">192px</div>
<div className="min-h-screen">최소 뷰포트 높이</div>
```

### 타이포그래피

```jsx
{/* 폰트 크기 */}
<p className="text-xs">12px</p>
<p className="text-sm">14px</p>
<p className="text-base">16px (기본)</p>
<p className="text-lg">18px</p>
<p className="text-xl">20px</p>
<p className="text-2xl">24px</p>
<p className="text-4xl">36px</p>

{/* 굵기 */}
<p className="font-normal">400</p>
<p className="font-medium">500</p>
<p className="font-semibold">600</p>
<p className="font-bold">700</p>

{/* 정렬 */}
<p className="text-left">왼쪽</p>
<p className="text-center">가운데</p>
<p className="text-right">오른쪽</p>

{/* 줄 높이 */}
<p className="leading-relaxed">여유 있는 줄간격</p>
<p className="leading-tight">촘촘한 줄간격</p>
```

### 테두리와 그림자

```jsx
{/* 테두리 */}
<div className="border">1px 테두리</div>
<div className="border-2 border-blue-500">2px 파란 테두리</div>
<div className="border-none">테두리 없음</div>

{/* 둥근 모서리 */}
<div className="rounded">4px 둥글게</div>
<div className="rounded-lg">8px</div>
<div className="rounded-xl">12px</div>
<div className="rounded-2xl">16px</div>
<div className="rounded-full">완전히 둥글게 (원)</div>

{/* 그림자 */}
<div className="shadow-sm">작은 그림자</div>
<div className="shadow">기본 그림자</div>
<div className="shadow-md">중간 그림자</div>
<div className="shadow-lg">큰 그림자</div>
<div className="shadow-xl">더 큰 그림자</div>
<div className="shadow-none">그림자 없음</div>
```

## 3️⃣ 레이아웃 — Flexbox와 Grid [↑](#toc)

### Flexbox

```jsx
{/* 기본 flex */}
<div className="flex">
  <div>A</div>
  <div>B</div>
  <div>C</div>
</div>

{/* 세로 방향 */}
<div className="flex flex-col">
  <div>위</div>
  <div>아래</div>
</div>

{/* 주축 정렬 (justify-content) */}
<div className="flex justify-start">왼쪽</div>
<div className="flex justify-center">가운데</div>
<div className="flex justify-end">오른쪽</div>
<div className="flex justify-between">양 끝</div>
<div className="flex justify-around">균등 간격</div>

{/* 교차축 정렬 (align-items) */}
<div className="flex items-start">위</div>
<div className="flex items-center">수직 가운데</div>
<div className="flex items-end">아래</div>

{/* 줄 바꿈 */}
<div className="flex flex-wrap">
  {/* 넘치면 다음 줄로 */}
</div>

{/* flex-1: 남은 공간 채우기 */}
<div className="flex">
  <div className="w-64">고정 너비</div>
  <div className="flex-1">남은 공간 전부</div>
</div>
```

### 실전 패턴: 헤더 레이아웃

```jsx
function Header() {
  return (
    <header className="flex items-center justify-between px-6 py-4 bg-white shadow-sm">
      <div className="flex items-center gap-3">
        <div className="w-8 h-8 bg-blue-500 rounded-full" />
        <span className="font-bold text-gray-800">날씨 앱</span>
      </div>
      <nav className="flex items-center gap-6">
        <a href="/" className="text-gray-600 hover:text-blue-600">홈</a>
        <a href="/about" className="text-gray-600 hover:text-blue-600">소개</a>
        <button className="bg-blue-500 text-white px-4 py-2 rounded-lg text-sm">
          로그인
        </button>
      </nav>
    </header>
  );
}
```

### CSS Grid

```jsx
{/* 3열 그리드 */}
<div className="grid grid-cols-3 gap-4">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>

{/* 12열 그리드 시스템 */}
<div className="grid grid-cols-12 gap-4">
  <div className="col-span-8">주 콘텐츠</div>  {/* 8열 차지 */}
  <div className="col-span-4">사이드바</div>   {/* 4열 차지 */}
</div>
```

## 4️⃣ 반응형 디자인 [↑](#toc)

### 브레이크포인트

Tailwind는 **모바일 우선(mobile-first)** 방식입니다. 클래스 앞에 접두사를 붙여 특정 화면 크기 이상에서만 적용됩니다.

| 접두사 | 최소 너비 | 대상 기기 |
|--------|-----------|-----------|
| (없음) | 0px | 모든 화면 (기본) |
| `sm:` | 640px | 큰 스마트폰 |
| `md:` | 768px | 태블릿 |
| `lg:` | 1024px | 노트북 |
| `xl:` | 1280px | 데스크탑 |
| `2xl:` | 1536px | 넓은 화면 |

### 사용 예시

```jsx
{/* 모바일: 1열, 태블릿: 2열, 데스크탑: 3열 */}
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-6">
  <Card />
  <Card />
  <Card />
</div>

{/* 모바일: 작은 글씨, 데스크탑: 큰 글씨 */}
<h1 className="text-2xl md:text-4xl lg:text-5xl font-bold">
  날씨 앱
</h1>

{/* 모바일에서 숨기고, 태블릿 이상에서 표시 */}
<nav className="hidden md:flex gap-4">
  <a href="/">홈</a>
  <a href="/about">소개</a>
</nav>

{/* 모바일 메뉴 버튼 — 태블릿 이상에서 숨김 */}
<button className="md:hidden">
  ☰ 메뉴
</button>
```

## 5️⃣ 호버와 상태 변형 [↑](#toc)

### hover: 마우스를 올렸을 때

```jsx
{/* 배경색 변경 */}
<button className="bg-blue-500 hover:bg-blue-600 text-white px-4 py-2 rounded-lg">
  호버해보세요
</button>

{/* 그림자 변경 */}
<div className="shadow hover:shadow-xl transition-shadow">
  카드
</div>

{/* 글자색 변경 */}
<a href="#" className="text-gray-600 hover:text-blue-600">
  링크
</a>

{/* 크기 변경 (scale) */}
<img className="hover:scale-105 transition-transform" src="..." alt="..." />
```

### transition: 부드러운 전환 효과

```jsx
{/* 모든 속성 전환 */}
<button className="bg-blue-500 hover:bg-blue-700 transition-all duration-300">
  버튼
</button>

{/* 특정 속성만 전환 */}
<div className="shadow hover:shadow-xl transition-shadow duration-200">
  카드
</div>

{/* 크기 전환 */}
<div className="scale-100 hover:scale-105 transition-transform">
  확대되는 박스
</div>
```

### focus: 포커스 상태 (접근성에 중요)

```jsx
<input
  type="text"
  className="border border-gray-300 rounded-lg px-3 py-2
             focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-transparent"
  placeholder="이름을 입력하세요"
/>
```

### active: 클릭할 때

```jsx
<button className="bg-blue-500 hover:bg-blue-600 active:bg-blue-700 text-white px-4 py-2 rounded-lg">
  클릭해보세요
</button>
```

### disabled: 비활성화 상태

```jsx
<button
  disabled
  className="bg-gray-300 text-gray-500 cursor-not-allowed px-4 py-2 rounded-lg
             disabled:opacity-50"
>
  비활성화
</button>
```

### 완성 코드

```jsx
// src/components/ProfileCard.jsx
function ProfileCard() {
  const profile = {
    name: "홍길동",
    role: "프론트엔드 개발자",
    bio: "React와 TypeScript를 좋아합니다. 사용자 경험을 개선하는 것에 관심이 많습니다.",
    skills: ["React", "JavaScript", "Tailwind CSS", "Git"],
    avatarColor: "from-blue-400 to-indigo-600",
    github: "https://github.com",
    email: "user@example.com",
    followers: 128,
    following: 64,
    repos: 42,
  };

  return (
    <div className="bg-white rounded-2xl shadow-lg overflow-hidden max-w-sm w-full
                    hover:shadow-2xl transition-shadow duration-300">

      {/* 배너 */}
      <div className={`bg-gradient-to-r ${profile.avatarColor} h-24`} />

      {/* 아바타 — 배너와 겹치게 */}
      <div className="px-6 pb-6">
        <div className="-mt-12 mb-4">
          <div className={`w-20 h-20 rounded-full bg-gradient-to-br ${profile.avatarColor}
                          flex items-center justify-center text-white text-3xl font-bold
                          border-4 border-white shadow-md`}>
            {profile.name[0]}
          </div>
        </div>

        {/* 이름과 직업 */}
        <h1 className="text-xl font-bold text-gray-900">{profile.name}</h1>
        <p className="text-blue-600 text-sm mt-0.5">{profile.role}</p>

        {/* 자기소개 */}
        <p className="text-gray-600 text-sm mt-3 leading-relaxed">
          {profile.bio}
        </p>

        {/* 통계 */}
        <div className="flex justify-around mt-4 py-3 border-y border-gray-100">
          <div className="text-center">
            <p className="font-bold text-gray-900">{profile.repos}</p>
            <p className="text-xs text-gray-500">저장소</p>
          </div>
          <div className="text-center">
            <p className="font-bold text-gray-900">{profile.followers}</p>
            <p className="text-xs text-gray-500">팔로워</p>
          </div>
          <div className="text-center">
            <p className="font-bold text-gray-900">{profile.following}</p>
            <p className="text-xs text-gray-500">팔로잉</p>
          </div>
        </div>

        {/* 기술 스택 */}
        <div className="mt-4 flex flex-wrap gap-2">
          {profile.skills.map(skill => (
            <span
              key={skill}
              className="bg-blue-50 text-blue-700 text-xs font-medium px-3 py-1 rounded-full
                         border border-blue-100 hover:bg-blue-100 transition-colors cursor-default"
            >
              {skill}
            </span>
          ))}
        </div>

        {/* 버튼 */}
        <div className="mt-5 flex gap-3">
          <a
            href={profile.github}
            className="flex-1 bg-gray-900 hover:bg-gray-700 text-white text-sm font-medium
                       py-2 rounded-lg text-center transition-colors"
            target="_blank"
            rel="noopener noreferrer"
          >
            GitHub
          </a>
          <a
            href={`mailto:${profile.email}`}
            className="flex-1 border border-gray-300 hover:border-blue-400 hover:text-blue-600
                       text-gray-700 text-sm font-medium py-2 rounded-lg text-center transition-colors"
          >
            이메일
          </a>
        </div>
      </div>
    </div>
  );
}

export default ProfileCard;
```

## 실습 과제

### 기본 과제

카드를 자신의 정보로 커스터마이징하세요.
- `avatarColor`를 바꿔 다른 그라디언트를 적용해보세요
- skills 배열을 자신의 기술로 교체하세요
- 통계 숫자를 원하는 값으로 바꿔보세요

### 도전 과제

다크 모드 버전을 만들어보세요. `bg-white` → `bg-gray-900`, `text-gray-900` → `text-white` 패턴으로 변경하고, 버튼을 눌러 다크/라이트를 토글하는 기능은 다음 장(useState)에서 배울 것입니다.

또는 카드를 3개 나란히 배치하는 그리드를 만들어보세요.

```jsx
// App.jsx
<div className="min-h-screen bg-gradient-to-br from-slate-100 to-blue-50 p-8">
  <div className="grid grid-cols-1 md:grid-cols-3 gap-6 max-w-5xl mx-auto">
    <ProfileCard />
    <ProfileCard />
    <ProfileCard />
  </div>
</div>
```

**예쁜 컴포넌트를 만들었습니다.** 하지만 아직 한 가지 문제가 남아 있습니다. 카드 3개가 모두 홍길동이라는 같은 사람 정보를 보여줍니다. 실제 앱에서는 각 카드가 다른 사람의 데이터를 받아서 표시해야 합니다. 다음 장에서 **Props**를 배우면 이 문제를 해결할 수 있습니다. 부모 컴포넌트가 자식 컴포넌트에게 원하는 데이터를 넘겨주는 방법입니다.
