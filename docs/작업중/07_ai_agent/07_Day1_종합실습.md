## 1️⃣ 프로젝트: 뉴스 리서치 & 요약 에이전트 [↑](#toc)

### 프로젝트 개요

오늘 배운 모든 것을 활용하여 **AI 관련 최신 뉴스를 찾고, 요약하고, 한국어로 번역**하는 3인 에이전트 팀을 만들어봅니다.

```
Agent 1: 뉴스 검색 전문가
  - 도구: SerperDevTool (웹 검색)
  - 역할: 최신 AI 뉴스 3건 수집

         ↓ context

Agent 2: 뉴스 요약 작가
  - 도구: 없음 (LLM 능력만 사용)
  - 역할: 수집된 뉴스를 독자 친화적으로 요약

         ↓ context

Agent 3: 한국어 번역 & 현지화 전문가
  - 도구: 없음
  - 역할: 영어 뉴스를 한국 독자에게 맞게 번역
```

### 사용 기술 총정리

| 기술 | 적용 위치 |
|------|---------|
| Agent (03장) | 뉴스 검색 전문가, 요약 작가, 번역가 |
| Task (03장) | 검색 Task, 요약 Task, 번역 Task |
| Tool (05장) | SerperDevTool — 실시간 뉴스 검색 |
| context (04장) | Task 간 결과 전달 |
| Sequential (06장) | 검색 → 요약 → 번역 순서 실행 |

## 2️⃣ 단계별 구현 [↑](#toc)

### Step 1: 환경 설정

```python
from dotenv import load_dotenv
load_dotenv()

from crewai import Agent, Task, Crew, Process, LLM
from crewai_tools import SerperDevTool

llm = LLM(model="gpt-4o-mini")
```

### Step 2: 도구 및 에이전트 생성

```python
# 검색 도구 (Serper API 키 필요)
search_tool = SerperDevTool()

# Agent 1: 뉴스 검색 전문가
news_searcher = Agent(
    role="AI 뉴스 검색 전문가",
    goal="최신 AI 에이전트 관련 뉴스를 3건 수집하고 핵심 내용을 정리한다",
    backstory=(
        "당신은 AI/테크 전문 뉴스 큐레이터입니다. "
        "하루 수백 건의 뉴스 중 가장 중요한 소식을 선별하는 것이 특기입니다."
    ),
    llm=llm,
    tools=[search_tool],
    verbose=True
)

# Agent 2: 뉴스 요약 작가
news_writer = Agent(
    role="뉴스 요약 전문 작가",
    goal="복잡한 AI 뉴스를 비전공자도 이해할 수 있게 핵심만 뽑아 요약한다",
    backstory=(
        "당신은 테크 뉴스레터 편집자입니다. "
        "어려운 기술 뉴스를 3분 안에 읽을 수 있게 요약하는 것이 특기입니다."
    ),
    llm=llm,
    verbose=True
)

# Agent 3: 한국어 번역 & 현지화 전문가
korean_translator = Agent(
    role="IT 전문 번역 & 현지화 전문가",
    goal="영어 AI 뉴스를 한국 독자가 공감할 수 있는 자연스러운 한국어로 번역한다",
    backstory=(
        "당신은 10년 경력의 IT 분야 전문 번역가입니다. "
        "직역이 아닌 의역으로 한국 독자에게 맞는 자연스러운 표현을 구사합니다."
    ),
    llm=llm,
    verbose=True
)
```

### Step 3: Task 설계

```python
# Task 1: 최신 AI 뉴스 검색
search_task = Task(
    description=(
        "오늘 날짜 기준 최신 'AI agent' 또는 'AI 에이전트' 관련 뉴스를 검색하세요.\n"
        "다음 조건을 만족하는 뉴스 3건을 선별하세요:\n"
        "- 최근 1주일 이내 발행된 뉴스\n"
        "- 실제 AI 에이전트 활용 사례 또는 신기술 발표 관련\n"
        "각 뉴스마다 제목, 출처, 날짜, 핵심 내용(3문장 이내)을 정리하세요."
    ),
    expected_output=(
        "3건의 최신 AI 에이전트 관련 뉴스 목록. "
        "각 항목: 제목, 출처, 날짜, 핵심 내용 포함. 마크다운 형식."
    ),
    agent=news_searcher
)

# Task 2: 뉴스 요약 (search_task 결과 참고)
summary_task = Task(
    description=(
        "수집된 3건의 뉴스를 독자 친화적인 뉴스레터 형식으로 요약하세요.\n"
        "요구사항:\n"
        "- 각 뉴스마다 '한 줄 요약' 포함\n"
        "- 왜 이 뉴스가 중요한지 설명 (독자에게 미치는 영향)\n"
        "- 전체 분량: 300~400자\n"
        "- 비전공자도 이해할 수 있는 쉬운 언어 사용"
    ),
    expected_output=(
        "뉴스레터 형식의 AI 에이전트 뉴스 요약. "
        "3건 각각의 한 줄 요약과 중요성 설명, 300~400자 분량."
    ),
    agent=news_writer,
    context=[search_task]
)

# Task 3: 한국어 번역 & 현지화 (summary_task 결과 참고)
translation_task = Task(
    description=(
        "요약된 뉴스레터를 한국 독자에게 최적화된 한국어로 완성하세요.\n"
        "요구사항:\n"
        "- 영어 용어는 한국어로 번역하되, 필요한 경우 영어 원문도 병기\n"
        "- 한국 독자가 공감할 수 있는 맥락 추가\n"
        "- 뉴스레터 제목과 발행일(오늘 날짜) 포함\n"
        "- 자연스럽고 읽기 편한 한국어 문체"
    ),
    expected_output=(
        "발행 준비 완료된 한국어 AI 뉴스레터. "
        "제목, 날짜, 3건의 뉴스 요약을 포함한 완성본."
    ),
    agent=korean_translator,
    context=[summary_task]
)
```

### Step 4: Crew 실행

```python
# Crew 구성 및 실행
crew = Crew(
    agents=[news_searcher, news_writer, korean_translator],
    tasks=[search_task, summary_task, translation_task],
    process=Process.sequential,
    verbose=True
)

result = crew.kickoff()

print("\n" + "=" * 60)
print("AI 뉴스레터 발행 완료!")
print("=" * 60)
print(result)
```

    process=Process.sequential,
    verbose=True
)

print("AI 뉴스레터 생성 시작...")

print("\n" + "=" * 60)
print("AI 뉴스레터 발행 완료!")
