## 1️⃣ Python 환경 설정 [↑](#toc)

### 주방 도구 비유

> 요리를 시작하기 전에 주방 도구를 준비해야 하듯이,
> 코딩도 개발 환경을 먼저 갖춰야 합니다.
> 가상환경(Virtual Environment)은 **각 프로젝트만의 전용 주방** — 다른 프로젝트 패키지와 충돌하지 않습니다.

### 방법 A: uv 사용 (권장)

`uv`는 Python 패키지 관리자로, pip보다 훨씬 빠르고 편리합니다. (Rust로 작성되어 10~100배 빠름)

```bash
# uv 설치 (macOS/Linux)
curl -LsSf https://astral.sh/uv/install.sh | sh

# uv 설치 (Windows PowerShell)
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

# 프로젝트 생성 및 패키지 설치
uv init ai-agent-workshop
cd ai-agent-workshop
uv add crewai crewai-tools python-dotenv
```

```
# 실행 결과
Using Python 3.12.3
Initialized project `ai-agent-workshop`
Resolved 87 packages in 1.23s
Installed 87 packages in 3.45s
 + crewai==0.100.1
 + crewai-tools==0.38.1
 + python-dotenv==1.0.1
```

### 방법 B: pip + venv 사용 (기존 방식)

```bash
# 가상환경 생성
python -m venv .venv

# 활성화 (macOS/Linux)
source .venv/bin/activate

# 활성화 (Windows)
.venv\Scripts\activate

# 패키지 설치
pip install crewai crewai-tools python-dotenv
```

### conda 사용자

기존에 Anaconda/Miniconda를 사용 중이라면:

```bash
# conda 환경 생성
conda create -n ai-agent python=3.12
conda activate ai-agent

# 패키지 설치
pip install crewai crewai-tools python-dotenv
```

## 2️⃣ API 키 발급과 관리 [↑](#toc)

### OpenAI API 키 발급 절차

1. [platform.openai.com](https://platform.openai.com) 접속
2. 계정 생성 또는 로그인
3. 우측 상단 프로필 → **"API keys"** 클릭
4. **"+ Create new secret key"** 클릭
5. 이름 입력 후 키 생성 → **반드시 복사해두기** (창을 닫으면 다시 볼 수 없음!)

### 비용 안내

| 모델 | 입력 토큰 (1M) | 출력 토큰 (1M) | 실습 예상 비용 |
|------|-------------|-------------|------------|
| **gpt-4o-mini** (권장) | $0.15 | $0.60 | ~$0.50 (2일) |
| gpt-4o | $2.50 | $10.00 | ~$5.00 (2일) |

> 실습에서는 **gpt-4o-mini**를 사용합니다. 빠르고 저렴하면서도 충분한 성능을 제공합니다.

### .env 파일 생성

API 키를 코드에 직접 쓰면 안 됩니다. `.env` 파일에 저장하고 코드에서 불러옵니다.

```bash
# 프로젝트 폴더에 .env 파일 생성
touch .env
```

```bash
# .env 파일 내용
OPENAI_API_KEY=sk-proj-여기에_발급받은_키를_붙여넣기
```

### ⚠️ 중요: .gitignore 설정

```bash
# .gitignore 파일에 .env 추가
echo ".env" >> .gitignore
```

> API 키를 GitHub에 올리면 몇 분 안에 봇이 발견해서 악용합니다.
> **절대로** `.env` 파일을 커밋하지 마세요!

### python-dotenv 사용법

```python
from dotenv import load_dotenv
import os

load_dotenv()  # .env 파일을 읽어서 환경변수로 등록

# 확인
api_key = os.getenv("OPENAI_API_KEY")
print(api_key[:10], "...")  # 앞 10자리만 출력
```

```
# 실행 결과
sk-proj-Ab ...
```

## 3️⃣ CrewAI 설치 확인 [↑](#toc)

설치가 제대로 됐는지 확인합니다.

```python
# 파일: check_install.py
import crewai
import crewai_tools

print(f"CrewAI 버전: {crewai.__version__}")
print(f"crewai-tools 버전: {crewai_tools.__version__}")
print("설치 완료!")
```

```
# 실행 결과
CrewAI 버전: 0.100.1
crewai-tools 버전: 0.38.1
설치 완료!
```

### 자주 발생하는 설치 오류

| 오류 | 원인 | 해결 |
|------|------|------|
| `ModuleNotFoundError: No module named 'crewai'` | 가상환경 비활성화 | `source .venv/bin/activate` 실행 |
| `ERROR: Could not find a version that satisfies the requirement` | Python 버전 낮음 | Python 3.10 이상 필요 |
| `ImportError: cannot import name 'Agent'` | 구버전 CrewAI | `pip install --upgrade crewai` |

## 4️⃣ 첫 LLM 호출 [↑](#toc)

### "Hello, AI!" 실습

CrewAI를 통해 처음으로 LLM과 대화해봅니다.

```python
# 파일: hello_llm.py
from dotenv import load_dotenv
load_dotenv()  # .env에서 OPENAI_API_KEY 로드

from crewai import LLM

# LLM 객체 생성
llm = LLM(
    model="gpt-4o-mini",    # 사용할 모델
    temperature=0.7,         # 창의성 조절 (0: 결정적, 1: 창의적)
    max_tokens=500           # 최대 응답 길이
)

# LLM 호출
response = llm.call("안녕하세요! 자기소개 해주세요. 한국어로 짧게 답해주세요.")
print(response)
```

```
# 실행 결과
안녕하세요! 저는 OpenAI가 개발한 GPT-4o-mini입니다.
대화, 질문 답변, 글쓰기, 분석 등 다양한 언어 관련 작업을 도와드릴 수 있습니다.
무엇을 도와드릴까요?
```

### temperature 파라미터 이해

```python
# temperature 비교 실습
from crewai import LLM

prompt = "사과를 한 단어로 묘사해주세요."

llm_cold = LLM(model="gpt-4o-mini", temperature=0.0)
llm_hot = LLM(model="gpt-4o-mini", temperature=1.0)

print("temperature=0.0 (결정적):", llm_cold.call(prompt))
print("temperature=1.0 (창의적):", llm_hot.call(prompt))
```

```
# 실행 결과 예시
temperature=0.0 (결정적): 빨간
temperature=1.0 (창의적): 상큼한
# (temperature=0.0은 항상 같은 답, temperature=1.0은 실행마다 다를 수 있음)
```

## 5️⃣ 정리 [↑](#toc)

### 환경 체크리스트

설치가 완료됐는지 아래 체크리스트를 확인하세요.

- [ ] Python 3.10 이상 설치됨 (`python --version`)
- [ ] 가상환경 생성 및 활성화됨
- [ ] `crewai`, `crewai-tools`, `python-dotenv` 설치됨
- [ ] `.env` 파일에 `OPENAI_API_KEY` 입력됨
- [ ] `.gitignore`에 `.env` 추가됨
- [ ] `check_install.py` 실행해서 버전 확인됨
- [ ] `hello_llm.py` 실행해서 LLM 응답 받음

### 폴더 구조

```
ai-agent-workshop/
├── .env                 ← API 키 (절대 커밋 금지!)
├── .gitignore           ← .env 포함
├── check_install.py
├── hello_llm.py
└── ...                  ← 앞으로 만들 파일들
```

### 핵심 정리

| 개념 | 내용 |
|------|------|
| **가상환경** | 프로젝트별 독립 Python 환경 |
| **python-dotenv** | `.env` 파일에서 환경변수 로드 |
| **OPENAI_API_KEY** | OpenAI API 사용을 위한 인증 키 |
| **temperature** | LLM 출력의 창의성/일관성 조절 (0~1) |
| **gpt-4o-mini** | 빠르고 저렴한 실습용 모델 |
