# Todo Manager 프로젝트 - Copilot 전역 지침

## 프로젝트 개요

Todo Manager는 Python으로 구현된 명령줄 할일 관리 도구입니다.

## 기술 스택

- Python 3.11+
- pytest (테스트)
- uv (패키지 관리)

## 핵심 원칙

- TDD 흐름(Red → Green → Refactor)을 따른다.
- 모든 공개 메서드에 한국어 docstring을 작성한다.
- 단일 책임 원칙을 따른다.

## 디렉터리 구조

- `src/todo/` : 소스 코드
- `tests/` : 테스트 코드
- `docs/` : 프로젝트 문서 (PRODUCT.md, ARCHITECTURE.md, CONTRIBUTING.md)
