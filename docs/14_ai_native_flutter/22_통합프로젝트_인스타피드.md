## 1️⃣ 프로젝트 소개 [↑](#toc)

### 무엇을 만드나요?

인스타그램 스타일의 사진 피드 앱입니다.  
JSONPlaceholder API로 사진 데이터를 가져오고, picsum.photos로 실제 이미지를 표시합니다.

### 주요 기능

1. **피드 화면**: GridView 3열 사진 그리드
2. **상세 화면**: 큰 이미지 + 제목 + 좋아요 버튼
3. **프로필 헤더**: 아바타 + 사용자명 + 포스트 수
4. **좋아요**: 토글 + SharedPreferences 저장
5. **하단 네비게이션**: 홈(피드) / 좋아요 목록 / 프로필

### 사용할 API

| API | 엔드포인트 | 용도 |
|-----|-----------|------|
| JSONPlaceholder | `/photos` | 사진 목록 (id, title, thumbnailUrl) |
| picsum.photos | `/id/{id}/300/300` | 실제 이미지 (id 기반 랜덤) |

JSONPlaceholder 응답 예시:

```json
[
  {
    "albumId": 1,
    "id": 1,
    "title": "accusamus beatae ad facilis cum similique qui sunt",
    "url": "https://placehold.co/600/92c952",
    "thumbnailUrl": "https://placehold.co/150/92c952"
  }
]
```

### 이전 프로젝트와의 차이점

| 항목 | 날씨 앱 | 할일 앱 | 인스타 피드 |
|------|--------|--------|-----------|
| 화면 수 | 1개 | 1개 | 4개 + 하단 탭 |
| 레이아웃 | 세로 스크롤 | 리스트 | GridView |
| 이미지 | 아이콘 URL | 없음 | 실제 이미지 |
| 화면 전환 | 없음 | 없음 | Hero 애니메이션 |
| 독립성 | 강사 시연 | 강사 시연 | 혼자 구현 |

## 2️⃣ ⭐ Step 1: 요구사항 작성 [↑](#toc)

직접 요구사항을 완성해보세요. 아래는 시작점입니다.

### 기능 요구사항 템플릿

```
인스타 피드 앱 요구사항
======================

[피드 화면 - 홈 탭]
- 앱 시작 시 JSONPlaceholder에서 사진 50개 로드
- GridView 3열로 사진 그리드 표시
- 각 그리드 셀: picsum.photos 이미지 표시
- 로딩 중: CircularProgressIndicator
- 에러 시: 에러 메시지 + 다시 시도 버튼
- 사진 탭 시 상세 화면으로 이동 (Hero 애니메이션)

[상세 화면]
- 큰 이미지 (화면 너비의 100%)
- 사진 제목 텍스트
- 좋아요 버튼 (하트 아이콘, 토글)
- 뒤로 가기 버튼

[좋아요 기능]
- 상세 화면에서 하트 버튼으로 좋아요 토글
- 좋아요 상태를 SharedPreferences에 저장
- 앱 재시작 후에도 좋아요 상태 유지
- 좋아요 수 표시 (실제로는 랜덤 값 + 내 좋아요 반영)

[좋아요 목록 화면 - 하트 탭]
- 좋아요 한 사진 목록 표시
- 2열 GridView로 표시
- 좋아요가 없으면 빈 상태 메시지

[프로필 화면 - 사람 탭]
- CircleAvatar (picsum.photos/seed/profile/100/100)
- 사용자명 (예: flutter_learner)
- 포스트 수 / 팔로워 / 팔로잉 (더미 데이터)
- 내 좋아요 목록을 그리드로 표시

[하단 네비게이션]
- 홈(피드) / 좋아요 / 프로필 3개 탭
- 현재 탭 아이콘 강조

[추가 고려사항]
- 사진 로딩 중 placeholder (회색 배경 또는 shimmer)
- 이미지 캐싱 (cached_network_image 패키지 선택적)
```

### ApiService 테스트 예시

```dart
// test/services/api_service_test.dart
import 'package:flutter_test/flutter_test.dart';
import 'package:insta_feed/services/api_service.dart';

void main() {
  group('ApiService', () {
    late ApiService service;

    setUp(() => service = ApiService());

    test('buildPhotosUrl()이 올바른 URL을 반환한다', () {
      final url = service.buildPhotosUrl(limit: 50);
      expect(url, contains('jsonplaceholder.typicode.com'));
      expect(url, contains('photos'));
      expect(url, contains('_limit=50'));
    });

    test('parsePhotos()가 JSON 리스트를 Photo 리스트로 변환한다', () {
      final jsonList = [
        {
          'albumId': 1,
          'id': 1,
          'title': '테스트 사진',
          'url': 'https://example.com/600.jpg',
          'thumbnailUrl': 'https://example.com/150.jpg',
        }
      ];
      final photos = service.parsePhotos(jsonList);
      expect(photos.length, equals(1));
      expect(photos.first.title, equals('테스트 사진'));
    });
  });
}
```

### FavoritesService 테스트 예시

```dart
// test/services/favorites_service_test.dart
import 'package:flutter_test/flutter_test.dart';
import 'package:insta_feed/services/favorites_service.dart';

void main() {
  group('FavoritesService', () {
    late FavoritesService service;

    setUp(() => service = FavoritesService());

    test('초기 상태에서 isFavorite()는 false다', () {
      expect(service.isFavorite(1), isFalse);
    });

    test('addFavorite() 후 isFavorite()는 true다', () {
      service.addFavorite(1);
      expect(service.isFavorite(1), isTrue);
    });

    test('removeFavorite() 후 isFavorite()는 false다', () {
      service.addFavorite(1);
      service.removeFavorite(1);
      expect(service.isFavorite(1), isFalse);
    });

    test('toggle() 후 상태가 반전된다', () {
      expect(service.isFavorite(1), isFalse);
      service.toggle(1);
      expect(service.isFavorite(1), isTrue);
      service.toggle(1);
      expect(service.isFavorite(1), isFalse);
    });

    test('favoriteIds가 좋아요 ID 목록을 반환한다', () {
      service.addFavorite(1);
      service.addFavorite(3);
      expect(service.favoriteIds, containsAll([1, 3]));
    });
  });
}
```

### 5-1. GridView.builder 기본 패턴

```dart
GridView.builder(
  gridDelegate: const SliverGridDelegateWithFixedCrossAxisCount(
    crossAxisCount: 3,        // 열 수
    crossAxisSpacing: 2,      // 가로 간격
    mainAxisSpacing: 2,       // 세로 간격
    childAspectRatio: 1,      // 가로:세로 비율 (1 = 정사각형)
  ),
  itemCount: photos.length,
  itemBuilder: (context, index) {
    final photo = photos[index];
    return GestureDetector(
      onTap: () => Navigator.push(
        context,
        MaterialPageRoute(
          builder: (_) => DetailScreen(photo: photo),
        ),
      ),
      child: Image.network(
        photo.thumbUrl,
        fit: BoxFit.cover,
        loadingBuilder: (context, child, loadingProgress) {
          if (loadingProgress == null) return child;
          return Container(color: Colors.grey.shade200);
        },
      ),
    );
  },
)
```

### 5-4. 프로필 헤더 (CircleAvatar)

```dart
class ProfileHeader extends StatelessWidget {
  final int postCount;
  const ProfileHeader({super.key, required this.postCount});

  @override
  Widget build(BuildContext context) {
    return Padding(
      padding: const EdgeInsets.all(16),
      child: Row(
        children: [
          const CircleAvatar(
            radius: 40,
            backgroundImage: NetworkImage(
              'https://picsum.photos/seed/profile/200/200',
            ),
          ),
          const SizedBox(width: 24),
          Expanded(
            child: Row(
              mainAxisAlignment: MainAxisAlignment.spaceAround,
              children: [
                _StatItem(value: postCount, label: '포스트'),
                const _StatItem(value: 128, label: '팔로워'),
                const _StatItem(value: 64, label: '팔로잉'),
              ],
            ),
          ),
        ],
      ),
    );
  }
}

class _StatItem extends StatelessWidget {
  final int value;
  final String label;
  const _StatItem({required this.value, required this.label});

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        Text(
          '$value',
          style: const TextStyle(fontSize: 20, fontWeight: FontWeight.bold),
        ),
        Text(label, style: const TextStyle(color: Colors.grey)),
      ],
    );
  }
}
```

### 좋아요 수 표시

각 사진마다 좋아요 수를 표시합니다. 실제 API가 없으므로 사진 ID를 시드로 한 랜덤 값을 사용합니다:

```dart
int getLikeCount(int photoId, bool isFavorite) {
  final base = (photoId * 7 + 13) % 500 + 10; // 의사 랜덤
  return isFavorite ? base + 1 : base;
}
```

### 프로젝트 비교

| 항목 | BMI 계산기 | 날씨 앱 | 할일 앱 | 인스타 피드 |
|------|----------|--------|--------|-----------|
| 핵심 기술 | 순수 함수 + 테스트 | API + AI-Native | CRUD + 로컬 저장 | 복잡한 UI + 독립 실행 |

### AI-Native 개발의 핵심 원칙

> "무엇을 만들까? → 요구사항 → AI 협업 → 테스트 검증"

이 사이클이 이제 여러분의 것이 되었습니다.

### 앞으로의 여정

**이 과정에서 배운 것:**
- Dart 문법 기초 ~ 심화 (비동기, 클래스, 제네릭)
- Flutter 위젯 (StatelessWidget, StatefulWidget, 레이아웃)
- 상태 관리 (setState, Provider)
- API 연동 (http, FutureBuilder)
- AI-Native 워크플로우 (Custom Instructions, Prompt Files, TDD)
- 테스트 (단위 테스트, 위젯 테스트)
- 로컬 저장 (SharedPreferences)

**다음에 배울 것들:**
- Firebase (실시간 데이터베이스, 인증)
