## 1️⃣ ⭐ **핵심** — Row와 Column [↑](#toc)

### 책장 비유

Flutter의 레이아웃은 **책장**과 같습니다.

- **Column** = 세로로 책을 꽂는 책장 (위에서 아래로)
- **Row** = 가로로 책을 꽂는 책장 (왼쪽에서 오른쪽으로)

```
Column               Row
┌──────┐          ┌──┬──┬──┐
│  위젯1  │          │위1│위2│위3│
├──────┤          └──┴──┴──┘
│  위젯2  │
├──────┤
│  위젯3  │
└──────┘
```

### Column 기본 사용

```dart
import 'package:flutter/material.dart';

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('레이아웃 연습')),
        body: Column(
          children: [
            Container(color: Colors.red, height: 80, width: double.infinity),
            Container(color: Colors.green, height: 80, width: double.infinity),
            Container(color: Colors.blue, height: 80, width: double.infinity),
          ],
        ),
      ),
    );
  }
}
```

> `children` 배열 안에 위젯을 넣으면 순서대로 위에서 아래로 배치됩니다.

### Row 기본 사용

```dart
body: Row(
  children: [
    Container(color: Colors.red, height: 80, width: 80),
    Container(color: Colors.green, height: 80, width: 80),
    Container(color: Colors.blue, height: 80, width: 80),
  ],
),
```

> `children` 배열 안 위젯이 왼쪽에서 오른쪽으로 배치됩니다.

### 중첩(Nesting)

실제 앱 화면은 Row와 Column을 **중첩**해서 만듭니다.

```dart
body: Column(
  children: [
    // 첫 번째 행: 아이콘 3개
    Row(
      children: [
        Icon(Icons.home),
        Icon(Icons.search),
        Icon(Icons.person),
      ],
    ),
    // 두 번째 행: 텍스트
    const Text('안녕하세요!'),
  ],
),
```

## 2️⃣ ⭐ **핵심** — MainAxisAlignment [↑](#toc)

**주축(Main Axis)**이란 배치 방향과 같은 축입니다.
- Column의 주축 = 세로 (위↓아래)
- Row의 주축 = 가로 (왼←오른)

### 6가지 정렬 값

```dart
// start: 시작 방향으로 몰기 (기본값)
Column(
  mainAxisAlignment: MainAxisAlignment.start,
  children: [...],
)

// center: 가운데 정렬
Column(
  mainAxisAlignment: MainAxisAlignment.center,
  children: [...],
)

// end: 끝 방향으로 몰기
Column(
  mainAxisAlignment: MainAxisAlignment.end,
  children: [...],
)

// spaceBetween: 첫/마지막은 끝에, 나머지는 균등 간격
Column(
  mainAxisAlignment: MainAxisAlignment.spaceBetween,
  children: [...],
)

// spaceAround: 각 위젯 주변에 균등 공간
Column(
  mainAxisAlignment: MainAxisAlignment.spaceAround,
  children: [...],
)

// spaceEvenly: 모든 공간(양끝 포함)을 균등하게
Column(
  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
  children: [...],
)
```

### 시각적 비교

```
spaceBetween:   spaceAround:    spaceEvenly:
┌─────────┐     ┌─────────┐     ┌─────────┐
│ [위젯1]  │     │  [위젯1] │     │  [위젯1] │
│         │     │         │     │         │
│         │     │  [위젯2] │     │  [위젯2] │
│ [위젯2]  │     │         │     │         │
│         │     │  [위젯3] │     │  [위젯3] │
│ [위젯3]  │     │         │     │         │
└─────────┘     └─────────┘     └─────────┘
```

### 실제 예시 — 화면 중앙에 배치

```dart
body: Column(
  mainAxisAlignment: MainAxisAlignment.center,
  children: [
    const Icon(Icons.flutter_dash, size: 100),
    const Text('Flutter 시작하기', style: TextStyle(fontSize: 24)),
    const SizedBox(height: 16),
    ElevatedButton(
      onPressed: () {},
      child: const Text('시작'),
    ),
  ],
),
```

## 3️⃣ CrossAxisAlignment — 교차축 정렬 [↑](#toc)

**교차축(Cross Axis)**은 주축과 수직인 축입니다.
- Column의 교차축 = 가로
- Row의 교차축 = 세로

```dart
Column(
  crossAxisAlignment: CrossAxisAlignment.start,   // 왼쪽 정렬
  // crossAxisAlignment: CrossAxisAlignment.center, // 가운데 정렬 (기본값)
  // crossAxisAlignment: CrossAxisAlignment.end,    // 오른쪽 정렬
  // crossAxisAlignment: CrossAxisAlignment.stretch, // 가득 채우기
  children: [
    Container(color: Colors.red, height: 40, width: 100),
    Container(color: Colors.green, height: 40, width: 150),
    Container(color: Colors.blue, height: 40, width: 80),
  ],
),
```

### stretch — 가득 채우기

`CrossAxisAlignment.stretch`는 자식 위젯을 교차축 방향으로 **가득 채웁니다**.

```dart
Column(
  crossAxisAlignment: CrossAxisAlignment.stretch,
  children: [
    Container(color: Colors.red, height: 60),    // width 없어도 전체 너비
    Container(color: Colors.green, height: 60),
  ],
),
```

## 4️⃣ ⭐ **핵심** — Expanded와 Flexible [↑](#toc)

### 풍선 비유

Row나 Column 안에서 남은 공간을 차지하려면 **Expanded**를 사용합니다.

> 친구들과 긴 소파에 앉는다고 상상하세요.
> 어떤 친구는 "내 자리만"(고정 크기), 어떤 친구는 "남은 자리는 내가 다"(Expanded)라고 합니다.

```dart
Row(
  children: [
    Container(color: Colors.red, width: 80, height: 80),    // 고정 80px
    Expanded(                                                 // 나머지 전체
      child: Container(color: Colors.green, height: 80),
    ),
    Container(color: Colors.blue, width: 80, height: 80),   // 고정 80px
  ],
),
```

### flex 비율로 나누기

```dart
Row(
  children: [
    Expanded(
      flex: 1,                                              // 전체의 1/3
      child: Container(color: Colors.red, height: 80),
    ),
    Expanded(
      flex: 2,                                              // 전체의 2/3
      child: Container(color: Colors.green, height: 80),
    ),
  ],
),
```

### Expanded vs Flexible 차이

| | Expanded | Flexible |
|---|---|---|
| 남은 공간 | 반드시 모두 차지 | 필요한 만큼만 차지 |
| 기본 fit | FlexFit.tight | FlexFit.loose |
| 사용 빈도 | 자주 사용 | 가끔 사용 |

```dart
Row(
  children: [
    Flexible(
      child: Text('짧은 텍스트'),       // 텍스트 길이만큼만 차지
    ),
    Flexible(
      child: Text('이것은 매우 긴 텍스트인데 Flexible이면 줄바꿈됩니다'),
    ),
  ],
),
```

## 5️⃣ SizedBox — 고정 크기 공간 [↑](#toc)

`SizedBox`는 **고정된 크기**의 빈 공간을 만들거나, 위젯의 크기를 강제할 때 사용합니다.

```dart
Column(
  children: [
    const Text('첫 번째 텍스트'),
    const SizedBox(height: 16),           // 16px 세로 여백
    const Text('두 번째 텍스트'),
    const SizedBox(height: 8),
    const Text('세 번째 텍스트'),
  ],
),
```

### 위젯 크기 강제하기

```dart
SizedBox(
  width: 200,
  height: 50,
  child: ElevatedButton(
    onPressed: () {},
    child: const Text('버튼'),
  ),
),
```

### Row에서 가로 여백

```dart
Row(
  children: [
    const Icon(Icons.star),
    const SizedBox(width: 8),             // 8px 가로 여백
    const Text('즐겨찾기'),
  ],
),
```

## 6️⃣ Padding과 EdgeInsets — 안쪽 여백 [↑](#toc)

`Padding` 위젯은 자식 위젯 주변에 **안쪽 여백**을 줍니다.

### EdgeInsets 4가지 방법

```dart
// 1. 전체 동일 여백
Padding(
  padding: const EdgeInsets.all(16.0),
  child: Text('모든 방향 16px'),
),

// 2. 각 방향 따로 지정
Padding(
  padding: const EdgeInsets.only(left: 16, top: 8, right: 16, bottom: 8),
  child: Text('방향별 다른 여백'),
),

// 3. 가로/세로 대칭
Padding(
  padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 8),
  child: Text('가로 16, 세로 8'),
),

// 4. 상하좌우 직접 지정 (left, top, right, bottom 순서)
Padding(
  padding: const EdgeInsets.fromLTRB(16, 8, 16, 32),
  child: Text('fromLTRB 여백'),
),
```

### Container의 padding

`Container` 위젯은 `padding` 속성을 직접 가집니다.

```dart
Container(
  padding: const EdgeInsets.all(16),
  color: Colors.amber,
  child: const Text('Container 내부 여백'),
),
```

## 7️⃣ 📖 **더 알아보기** — Stack [↑](#toc)

### 포토샵 레이어 비유

`Stack`은 **포토샵의 레이어**처럼 위젯을 겹쳐서 배치합니다.

> Row와 Column이 "옆에, 아래에" 배치라면,
> Stack은 "위에, 위에" 배치입니다.

```dart
Stack(
  children: [
    // 맨 아래 레이어 — 배경 이미지
    Container(
      width: 300,
      height: 200,
      color: Colors.blue,
    ),
    // 중간 레이어 — 반투명 오버레이
    Container(
      width: 300,
      height: 200,
      color: Colors.black.withOpacity(0.3),
    ),
    // 맨 위 레이어 — 텍스트
    const Center(
      child: Text(
        '위에 표시',
        style: TextStyle(color: Colors.white, fontSize: 24),
      ),
    ),
  ],
),
```

### Positioned — 위치 지정

`Stack` 안에서 `Positioned`를 사용하면 특정 위치에 배치할 수 있습니다.

```dart
Stack(
  children: [
    Container(width: 300, height: 300, color: Colors.grey[200]),
    Positioned(
      top: 10,
      right: 10,
      child: Container(
        width: 30,
        height: 30,
        decoration: const BoxDecoration(
          color: Colors.red,
          shape: BoxShape.circle,
        ),
        child: const Center(
          child: Text('3', style: TextStyle(color: Colors.white)),
        ),
      ),
    ),
  ],
),
```

> 이 패턴은 앱 아이콘의 **배지(badge)** — 알림 숫자 빨간 동그라미를 만들 때 자주 씁니다.

## 8️⃣ 💪 **보너스 챌린지** — 실습: 프로필 카드 만들기 [↑](#toc)

지금까지 배운 레이아웃 위젯을 모두 사용해서 프로필 카드를 만들어 봅시다.

### 완성 목표

```
┌────────────────────────────┐
│  ┌──────┐  홍길동            │
│  │ 사진  │  Flutter 개발자   │
│  └──────┘  ★★★★☆           │
│                            │
│  📧 hong@example.com       │
│  📱 010-1234-5678          │
│                            │
│  [  메시지 보내기  ]  [팔로우]  │
└────────────────────────────┘
```

### 단계별 구현

**1단계: 기본 구조 만들기**

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(
      home: ProfileCard(),
    );
  }
}

class ProfileCard extends StatelessWidget {
  const ProfileCard({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      backgroundColor: Colors.grey[100],
      appBar: AppBar(
        title: const Text('프로필'),
        backgroundColor: Colors.white,
        foregroundColor: Colors.black,
      ),
      body: Center(
        child: _buildCard(),
      ),
    );
  }

  Widget _buildCard() {
    return Container(
      width: 320,
      margin: const EdgeInsets.all(16),
      padding: const EdgeInsets.all(20),
      decoration: BoxDecoration(
        color: Colors.white,
        borderRadius: BorderRadius.circular(16),
        boxShadow: [
          BoxShadow(
            color: Colors.black.withOpacity(0.1),
            blurRadius: 10,
            offset: const Offset(0, 4),
          ),
        ],
      ),
      child: Column(
        mainAxisSize: MainAxisSize.min,
        children: [
          _buildHeader(),
          const SizedBox(height: 16),
          const Divider(),
          const SizedBox(height: 16),
          _buildContactInfo(),
          const SizedBox(height: 20),
          _buildButtons(),
        ],
      ),
    );
  }

  Widget _buildHeader() {
    return Row(
      children: [
        // 프로필 사진
        Container(
          width: 80,
          height: 80,
          decoration: BoxDecoration(
            shape: BoxShape.circle,
            color: Colors.blue[100],
          ),
          child: const Icon(Icons.person, size: 50, color: Colors.blue),
        ),
        const SizedBox(width: 16),
        // 이름, 직업, 별점
        Expanded(
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              const Text(
                '홍길동',
                style: TextStyle(fontSize: 22, fontWeight: FontWeight.bold),
              ),
              const SizedBox(height: 4),
              Text(
                'Flutter 개발자',
                style: TextStyle(fontSize: 14, color: Colors.grey[600]),
              ),
              const SizedBox(height: 6),
              Row(
                children: List.generate(5, (index) {
                  return Icon(
                    index < 4 ? Icons.star : Icons.star_border,
                    color: Colors.amber,
                    size: 18,
                  );
                }),
              ),
            ],
          ),
        ),
      ],
    );
  }

  Widget _buildContactInfo() {
    return Column(
      children: [
        _buildInfoRow(Icons.email, 'hong@example.com'),
        const SizedBox(height: 8),
        _buildInfoRow(Icons.phone, '010-1234-5678'),
      ],
    );
  }

  Widget _buildInfoRow(IconData icon, String text) {
    return Row(
      children: [
        Icon(icon, size: 18, color: Colors.grey[600]),
        const SizedBox(width: 8),
        Text(text, style: TextStyle(color: Colors.grey[700])),
      ],
    );
  }

  Widget _buildButtons() {
    return Row(
      children: [
        Expanded(
          flex: 2,
          child: OutlinedButton(
            onPressed: () {},
            child: const Text('메시지 보내기'),
          ),
        ),
        const SizedBox(width: 8),
        Expanded(
          flex: 1,
          child: ElevatedButton(
            onPressed: () {},
            child: const Text('팔로우'),
          ),
        ),
      ],
    );
  }
}
```

## 9️⃣ 정리 [↑](#toc)

| 위젯 | 역할 | 핵심 속성 |
|------|------|-----------|
| `Column` | 세로 배치 | `mainAxisAlignment`, `crossAxisAlignment` |
| `Row` | 가로 배치 | `mainAxisAlignment`, `crossAxisAlignment` |
| `Expanded` | 남은 공간 채우기 | `flex` |
| `Flexible` | 필요한 만큼 차지 | `flex`, `fit` |
| `SizedBox` | 고정 크기/여백 | `width`, `height` |
| `Padding` | 안쪽 여백 | `EdgeInsets` |
| `Stack` | 겹쳐 배치 | `alignment` |
| `Positioned` | Stack 내 위치 지정 | `top`, `left`, `right`, `bottom` |

### 핵심 규칙 3가지

1. **Row/Column은 항상 중첩**해서 복잡한 레이아웃을 만든다
2. **Expanded**는 남은 공간을 "빨아들인다"
3. **SizedBox**는 Row/Column 안에서 간격을 줄 때 가장 깔끔하다
