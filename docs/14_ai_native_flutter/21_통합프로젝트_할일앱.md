## 1️⃣ 프로젝트 소개 [↑](#toc)

### 무엇을 만드나요?

할일 목록을 추가하고, 완료 체크하고, 삭제할 수 있는 할일 관리 앱입니다.  
앱을 종료하고 다시 켜도 데이터가 유지됩니다.

### 주요 기능

1. **할일 추가**: 텍스트 입력 후 추가 버튼 또는 엔터
2. **완료 토글**: 체크박스로 완료/미완료 전환
3. **할일 삭제**: 스와이프로 삭제 (Dismissible 위젯)
4. **필터**: 전체 / 완료 / 미완료 탭으로 필터링
5. **로컬 저장**: SharedPreferences로 앱 재시작 후에도 유지
6. **(선택) Provider로 상태 관리**

### 날씨 앱과의 차이점

| 날씨 앱 | 할일 앱 |
|--------|--------|
| API 호출 (외부 데이터) | 로컬 저장 (내부 데이터) |
| 읽기(Read)만 | CRUD 전체 |
| FutureBuilder | setState + ListView |
| 단일 화면 | 필터 탭 있는 단일 화면 |

## 2️⃣ ⭐ Step 1: 요구사항 작성 [↑](#toc)

### 기능 요구사항

```
할일 관리 앱 요구사항
====================

[할일 추가]
- 텍스트 필드에 내용 입력 후 추가 버튼 또는 Enter 키로 추가
- 빈 텍스트는 추가 불가 (안내 메시지 표시)
- 추가 후 텍스트 필드 초기화 및 포커스 유지

[할일 목록]
- 추가된 순서대로 표시 (최신이 위에)
- 각 항목: 체크박스 + 내용 텍스트 + 삭제 아이콘
- 완료된 항목: 텍스트에 취소선 + 회색 처리

[완료 토글]
- 체크박스 탭 시 완료/미완료 상태 전환
- 상태 변경은 즉시 저장

[할일 삭제]
- 항목을 좌우로 스와이프하면 삭제 (Dismissible)
- 삭제 후 Undo 스낵바 표시 (3초 내 되돌리기 가능)

[필터]
- 하단 또는 상단에 탭: 전체 / 완료 / 미완료
- 탭 선택에 따라 해당 항목만 표시
- 탭에 해당 개수 뱃지 표시

[로컬 저장]
- 모든 변경 사항을 SharedPreferences에 즉시 저장
- 앱 재시작 후 마지막 상태 복원
- JSON 형식으로 직렬화/역직렬화

[(선택) Provider]
- TodoNotifier로 상태 관리
- Consumer로 필터별 목록 표시
```

```dart
      expect(toggled.id, equals(todo.id));
      expect(toggled.title, equals(todo.title));
```

### 4-2. TodoItem 구현

```dart
// lib/models/todo_item.dart
class TodoItem {
  final String id;
  final String title;
  final bool isDone;
  final DateTime createdAt;

  TodoItem({
    required this.id,
    required this.title,
    this.isDone = false,
    DateTime? createdAt,
  }) : createdAt = createdAt ?? DateTime.now();

  Map<String, dynamic> toJson() => {
        'id': id,
        'title': title,
        'isDone': isDone,
        'createdAt': createdAt.toIso8601String(),
      };

  factory TodoItem.fromJson(Map<String, dynamic> json) {
    return TodoItem(
      id: json['id'] as String,
      title: json['title'] as String,
      isDone: json['isDone'] as bool,
      createdAt: DateTime.parse(json['createdAt'] as String),
    );
  }

  TodoItem copyWith({
    String? id,
    String? title,
    bool? isDone,
    DateTime? createdAt,
  }) {
    return TodoItem(
      id: id ?? this.id,
      title: title ?? this.title,
      isDone: isDone ?? this.isDone,
      createdAt: createdAt ?? this.createdAt,
    );
  }
}
```

### 4-4. TodoService 구현

```dart
// lib/services/todo_service.dart
import '../models/todo_item.dart';

enum FilterType { all, active, completed }

class TodoService {
  final List<TodoItem> _todos = [];

  List<TodoItem> get todos => List.unmodifiable(_todos);

  void addTodo(String title) {
    if (title.trim().isEmpty) return;
    _todos.insert(
      0,
      TodoItem(
        id: DateTime.now().millisecondsSinceEpoch.toString(),
        title: title.trim(),
      ),
    );
  }

  void toggleTodo(String id) {
    final index = _todos.indexWhere((t) => t.id == id);
    if (index == -1) return;
    _todos[index] = _todos[index].copyWith(isDone: !_todos[index].isDone);
  }

  void deleteTodo(String id) {
    _todos.removeWhere((t) => t.id == id);
  }

  void restoreTodo(TodoItem todo) {
    _todos.insert(0, todo);
  }

  void loadAll(List<TodoItem> items) {
    _todos.clear();
    _todos.addAll(items);
  }

  List<TodoItem> getFiltered(FilterType filter) {
    switch (filter) {
      case FilterType.all:
        return List.unmodifiable(_todos);
      case FilterType.active:
        return _todos.where((t) => !t.isDone).toList();
      case FilterType.completed:
        return _todos.where((t) => t.isDone).toList();
    }
  }

  int countByFilter(FilterType filter) => getFiltered(filter).length;
}
```

### 4-5. StorageService 구현

```dart
// lib/services/storage_service.dart
import 'package:shared_preferences/shared_preferences.dart';
import 'dart:convert';
import '../models/todo_item.dart';

class StorageService {
  static const String _key = 'todos';

  Future<List<TodoItem>> loadTodos() async {
    final prefs = await SharedPreferences.getInstance();
    final jsonString = prefs.getString(_key);
    if (jsonString == null) return [];

    final jsonList = jsonDecode(jsonString) as List<dynamic>;
    return jsonList
        .map((json) => TodoItem.fromJson(json as Map<String, dynamic>))
        .toList();
  }

  Future<void> saveTodos(List<TodoItem> todos) async {
    final prefs = await SharedPreferences.getInstance();
    final jsonString = jsonEncode(todos.map((t) => t.toJson()).toList());
    await prefs.setString(_key, jsonString);
  }
}
```

## 5️⃣ ⭐ Step 4: UI 구현 [↑](#toc)

### 5-1. 메인 화면 (TodoScreen)

```dart
// lib/screens/todo_screen.dart
import 'package:flutter/material.dart';
import '../models/todo_item.dart';
import '../services/todo_service.dart';
import '../services/storage_service.dart';

class TodoScreen extends StatefulWidget {
  const TodoScreen({super.key});

  @override
  State<TodoScreen> createState() => _TodoScreenState();
}

class _TodoScreenState extends State<TodoScreen> {
  final TodoService _todoService = TodoService();
  final StorageService _storageService = StorageService();
  final TextEditingController _controller = TextEditingController();
  FilterType _currentFilter = FilterType.all;
  bool _isLoading = true;

  @override
  void initState() {
    super.initState();
    _loadTodos();
  }

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }

  Future<void> _loadTodos() async {
    final todos = await _storageService.loadTodos();
    _todoService.loadAll(todos);
    if (mounted) setState(() => _isLoading = false);
  }

  Future<void> _save() async {
    await _storageService.saveTodos(_todoService.todos.toList());
  }

  void _addTodo() {
    final title = _controller.text;
    if (title.trim().isEmpty) {
      ScaffoldMessenger.of(context).showSnackBar(
        const SnackBar(content: Text('할일을 입력해주세요')),
      );
      return;
    }
    setState(() => _todoService.addTodo(title));
    _controller.clear();
    _save();
  }

  void _toggleTodo(String id) {
    setState(() => _todoService.toggleTodo(id));
    _save();
  }

  void _deleteTodo(TodoItem todo) {
    setState(() => _todoService.deleteTodo(todo.id));
    _save();

    ScaffoldMessenger.of(context).showSnackBar(
      SnackBar(
        content: Text('"${todo.title}"을 삭제했습니다'),
        action: SnackBarAction(
          label: '되돌리기',
          onPressed: () {
            setState(() => _todoService.restoreTodo(todo));
            _save();
          },
        ),
        duration: const Duration(seconds: 3),
      ),
    );
  }

  @override
  Widget build(BuildContext context) {
    if (_isLoading) {
      return const Scaffold(body: Center(child: CircularProgressIndicator()));
    }

    final filtered = _todoService.getFiltered(_currentFilter);

    return Scaffold(
      appBar: AppBar(
        title: const Text('할일 관리'),
        backgroundColor: Theme.of(context).colorScheme.inversePrimary,
      ),
      body: Column(
        children: [
          // 입력 영역
          Padding(
            padding: const EdgeInsets.all(16),
            child: Row(
              children: [
                Expanded(
                  child: TextField(
                    controller: _controller,
                    decoration: InputDecoration(
                      hintText: '할일을 입력하세요',
                      border: OutlineInputBorder(
                        borderRadius: BorderRadius.circular(12),
                      ),
                      contentPadding: const EdgeInsets.symmetric(
                          horizontal: 16, vertical: 12),
                    ),
                    onSubmitted: (_) => _addTodo(),
                    textInputAction: TextInputAction.done,
                  ),
                ),
                const SizedBox(width: 8),
                FilledButton(
                  onPressed: _addTodo,
                  child: const Icon(Icons.add),
                ),
              ],
            ),
          ),

          // 필터 탭
          Padding(
            padding: const EdgeInsets.symmetric(horizontal: 16),
            child: Row(
              children: FilterType.values.map((filter) {
                final count = _todoService.countByFilter(filter);
                final label = switch (filter) {
                  FilterType.all => '전체',
                  FilterType.active => '미완료',
                  FilterType.completed => '완료',
                };
                final isSelected = _currentFilter == filter;
                return Expanded(
                  child: Padding(
                    padding: const EdgeInsets.symmetric(horizontal: 4),
                    child: FilterChip(
                      label: Text('$label ($count)'),
                      selected: isSelected,
                      onSelected: (_) =>
                          setState(() => _currentFilter = filter),
                    ),
                  ),
                );
              }).toList(),
            ),
          ),

          const SizedBox(height: 8),

          // 할일 목록
          Expanded(
            child: filtered.isEmpty
                ? Center(
                    child: Text(
                      _currentFilter == FilterType.all
                          ? '할일을 추가해보세요!'
                          : '해당하는 할일이 없습니다',
                      style: const TextStyle(color: Colors.grey, fontSize: 16),
                    ),
                  )
                : ListView.builder(
                    itemCount: filtered.length,
                    padding: const EdgeInsets.symmetric(horizontal: 8),
                    itemBuilder: (context, index) {
                      final todo = filtered[index];
                      return _TodoTile(
                        todo: todo,
                        onToggle: () => _toggleTodo(todo.id),
                        onDismiss: () => _deleteTodo(todo),
                      );
                    },
                  ),
          ),
        ],
      ),
    );
  }
}

class _TodoTile extends StatelessWidget {
  final TodoItem todo;
  final VoidCallback onToggle;
  final VoidCallback onDismiss;

  const _TodoTile({
    required this.todo,
    required this.onToggle,
    required this.onDismiss,
  });

  @override
  Widget build(BuildContext context) {
    return Dismissible(
      key: Key(todo.id),
      direction: DismissDirection.endToStart,
      onDismissed: (_) => onDismiss(),
      background: Container(
        alignment: Alignment.centerRight,
        padding: const EdgeInsets.only(right: 16),
        color: Colors.red,
        child: const Icon(Icons.delete, color: Colors.white),
      ),
      child: Card(
        margin: const EdgeInsets.symmetric(vertical: 4, horizontal: 4),
        child: ListTile(
          leading: Checkbox(
            value: todo.isDone,
            onChanged: (_) => onToggle(),
          ),
          title: Text(
            todo.title,
            style: TextStyle(
              decoration:
                  todo.isDone ? TextDecoration.lineThrough : null,
              color: todo.isDone ? Colors.grey : null,
            ),
          ),
          trailing: IconButton(
            icon: const Icon(Icons.delete_outline, color: Colors.red),
            onPressed: onDismiss,
          ),
        ),
      ),
    );
  }
}
```

### TodoNotifier로 교체

```dart
// lib/notifiers/todo_notifier.dart
import 'package:flutter/foundation.dart';
import '../models/todo_item.dart';
import '../services/todo_service.dart';
import '../services/storage_service.dart';

class TodoNotifier extends ChangeNotifier {
  final TodoService _todoService = TodoService();
  final StorageService _storageService = StorageService();
  FilterType _filter = FilterType.all;
  bool _isLoading = true;

  bool get isLoading => _isLoading;
  FilterType get filter => _filter;
  List<TodoItem> get filtered => _todoService.getFiltered(_filter);
  int countByFilter(FilterType f) => _todoService.countByFilter(f);

  Future<void> load() async {
    final todos = await _storageService.loadTodos();
    _todoService.loadAll(todos);
    _isLoading = false;
    notifyListeners();
  }

  Future<void> addTodo(String title) async {
    _todoService.addTodo(title);
    await _storageService.saveTodos(_todoService.todos.toList());
    notifyListeners();
  }

  Future<void> toggleTodo(String id) async {
    _todoService.toggleTodo(id);
    await _storageService.saveTodos(_todoService.todos.toList());
    notifyListeners();
  }

  Future<void> deleteTodo(String id) async {
    _todoService.deleteTodo(id);
    await _storageService.saveTodos(_todoService.todos.toList());
    notifyListeners();
  }

  Future<void> restoreTodo(TodoItem todo) async {
    _todoService.restoreTodo(todo);
    await _storageService.saveTodos(_todoService.todos.toList());
    notifyListeners();
  }

  void setFilter(FilterType filter) {
    _filter = filter;
    notifyListeners();
  }
}
```

### Provider 적용 시 main.dart

```dart
// lib/main.dart (Provider 버전)
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';
import 'notifiers/todo_notifier.dart';
import 'screens/todo_screen.dart';

void main() {
  runApp(
    ChangeNotifierProvider(
      create: (_) => TodoNotifier()..load(),
      child: const MyApp(),
    ),
  );
}
```

## 8️⃣ 마무리 [↑](#toc)

### 날씨 앱 vs 할일 앱 비교

| 항목 | 날씨 앱 | 할일 앱 |
|------|--------|--------|
| 데이터 출처 | 외부 API | 로컬 저장 |
| 데이터 조작 | Read only | CRUD |
| 저장 방식 | SharedPreferences (텍스트) | SharedPreferences (JSON) |
| 상태 패턴 | FutureBuilder | setState (+ 선택적 Provider) |
| 핵심 위젯 | FutureBuilder, Image.network | Dismissible, FilterChip |

### AI-Native 워크플로우 복습

```
요구사항 작성 → 파일 구조 설계 → Instructions 세팅
→ 테스트 먼저 (사람) → 구현 (AI) → 검증 (테스트 실행)
→ UI 구현 → 직접 확인
```
