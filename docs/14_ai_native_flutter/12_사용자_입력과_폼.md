# 11장. 사용자 입력과 폼
{: .no_toc }

> **Phase 2** · 예상 시간: 90분

> 💡 **실습 흐름**: 강사 시연 보기 → 함께 따라하기 → 혼자 변형해 보기

> **이 장의 AI 사용 규칙 — 생성 + 검증 모드**
> AI(Copilot)가 코드를 생성할 수 있습니다.
> 단, 반드시 생성된 코드를 **읽고, 이해하고, 테스트로 검증**해야 합니다.

## 학습 목표

- TextField로 사용자 텍스트 입력을 받을 수 있다
- TextEditingController로 입력값을 가져오고 관리할 수 있다
- Form과 validator로 입력값을 유효성 검사할 수 있다
- dispose()로 컨트롤러를 올바르게 정리할 수 있다

## 1️⃣ ⭐ **핵심** — TextField [↑](#toc)

### 우편함 비유

> TextField는 앱의 **우편함**입니다.
> 사용자가 메시지(텍스트)를 넣으면, 앱이 그 메시지를 꺼내 처리합니다.

### 기본 TextField

```dart
const TextField()
```

이것만으로도 텍스트 입력창이 생깁니다. 하지만 입력값을 가져오려면 Controller가 필요합니다.

### onChanged — 실시간 입력 감지

```dart
class _SearchState extends State<SearchPage> {
  String _searchText = '';

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        TextField(
          onChanged: (value) {
            setState(() {
              _searchText = value;    // 타이핑할 때마다 실행
            });
          },
          decoration: const InputDecoration(
            hintText: '검색어 입력',
          ),
        ),
        Text('입력 중: $_searchText'),
      ],
    );
  }
}
```

### onSubmitted — 완료(Enter) 시 처리

```dart
TextField(
  onSubmitted: (value) {
    // 사용자가 Enter를 누르거나 키보드의 완료 버튼 클릭 시
    print('제출된 값: $value');
  },
),
```

## 2️⃣ ⭐ **핵심** — TextEditingController [↑](#toc)

### 원격 제어기 비유

> TextEditingController는 TextField의 **원격 제어기**입니다.
> 원격으로 텍스트를 읽고, 바꾸고, 지울 수 있습니다.

```dart
class _FormState extends State<FormPage> {
  // 1. 컨트롤러 생성
  final _nameController = TextEditingController();

  @override
  void dispose() {
    // 3. 반드시 정리! (메모리 누수 방지)
    _nameController.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        // 2. TextField에 연결
        TextField(
          controller: _nameController,
          decoration: const InputDecoration(labelText: '이름'),
        ),
        ElevatedButton(
          onPressed: () {
            // 입력값 읽기
            final name = _nameController.text;
            print('입력된 이름: $name');
          },
          child: const Text('확인'),
        ),
        ElevatedButton(
          onPressed: () {
            // 입력값 지우기
            _nameController.clear();
          },
          child: const Text('초기화'),
        ),
      ],
    );
  }
}
```

### Controller의 3가지 용도

```dart
// 1. 값 읽기
String value = _controller.text;

// 2. 값 쓰기 (프로그램에서 강제 입력)
_controller.text = '초기값';

// 3. 값 지우기
_controller.clear();
```

## 3️⃣ 키보드 타입 설정 [↑](#toc)

`keyboardType`으로 상황에 맞는 키보드를 보여줄 수 있습니다.

```dart
// 기본 텍스트 키보드
TextField(
  keyboardType: TextInputType.text,
)

// 숫자 키보드 (전화번호, 나이 등)
TextField(
  keyboardType: TextInputType.number,
)

// 이메일 키보드 (@, .com 버튼 포함)
TextField(
  keyboardType: TextInputType.emailAddress,
)

// 전화번호 키보드
TextField(
  keyboardType: TextInputType.phone,
)

// 비밀번호 (입력값 숨기기)
TextField(
  obscureText: true,    // 입력값을 *** 로 표시
)

// 여러 줄 입력 (메모, 댓글 등)
TextField(
  maxLines: 5,          // 최대 5줄
  minLines: 3,          // 최소 3줄 높이
  keyboardType: TextInputType.multiline,
  textInputAction: TextInputAction.newline,
)
```

### textInputAction — 키보드 완료 버튼 모양

```dart
// 다음 필드로 이동
TextField(textInputAction: TextInputAction.next)

// 완료
TextField(textInputAction: TextInputAction.done)

// 검색
TextField(textInputAction: TextInputAction.search)

// 전송
TextField(textInputAction: TextInputAction.send)
```

## 4️⃣ ⭐ **핵심** — Form과 GlobalKey [↑](#toc)

### Form이 필요한 이유

여러 개의 TextField를 묶어서 **한 번에 유효성 검사**하려면 `Form` 위젯이 필요합니다.

> Form은 여러 입력 필드를 묶는 **폼 서류**입니다.
> GlobalKey는 그 서류에 붙은 **고유 ID 번호**입니다.
> 코드에서 ID로 서류를 찾아 검사할 수 있습니다.

### Form 기본 구조

```dart
class _SignupState extends State<SignupPage> {
  // Form을 제어하기 위한 키
  final _formKey = GlobalKey<FormState>();

  final _emailController = TextEditingController();
  final _passwordController = TextEditingController();

  @override
  void dispose() {
    _emailController.dispose();
    _passwordController.dispose();
    super.dispose();
  }

  void _submit() {
    // 모든 validator를 실행하고 결과 반환
    if (_formKey.currentState!.validate()) {
      // 모든 검사 통과 → 폼 처리
      print('이메일: ${_emailController.text}');
      print('비밀번호: ${_passwordController.text}');
    }
  }

  @override
  Widget build(BuildContext context) {
    return Form(
      key: _formKey,       // Key 연결
      child: Column(
        children: [
          TextFormField(          // TextField가 아닌 TextFormField!
            controller: _emailController,
            decoration: const InputDecoration(labelText: '이메일'),
            validator: (value) {
              if (value == null || value.isEmpty) {
                return '이메일을 입력해주세요';
              }
              return null;       // null = 검사 통과
            },
          ),
          TextFormField(
            controller: _passwordController,
            obscureText: true,
            decoration: const InputDecoration(labelText: '비밀번호'),
            validator: (value) {
              if (value == null || value.length < 6) {
                return '비밀번호는 6자 이상이어야 합니다';
              }
              return null;
            },
          ),
          ElevatedButton(
            onPressed: _submit,
            child: const Text('가입하기'),
          ),
        ],
      ),
    );
  }
}
```

### TextField vs TextFormField

| | TextField | TextFormField |
|---|---|---|
| Form과 통합 | 불가 | 가능 |
| validator | 없음 | 있음 |
| 사용 상황 | 단일 입력 | Form 안에 여러 입력 |

## 5️⃣ ⭐ **핵심** — validator 작성 [↑](#toc)

`validator`는 **검사 함수**입니다.
- 검사 통과 → `null` 반환
- 검사 실패 → 에러 메시지(String) 반환

### 빈 입력 검사

```dart
validator: (value) {
  if (value == null || value.isEmpty) {
    return '필수 입력 항목입니다';
  }
  return null;
},
```

### 길이 제한

```dart
validator: (value) {
  if (value == null || value.isEmpty) {
    return '닉네임을 입력해주세요';
  }
  if (value.length < 2) {
    return '닉네임은 2자 이상이어야 합니다';
  }
  if (value.length > 20) {
    return '닉네임은 20자 이하여야 합니다';
  }
  return null;
},
```

### 이메일 형식 검사

```dart
validator: (value) {
  if (value == null || value.isEmpty) {
    return '이메일을 입력해주세요';
  }
  // 정규식으로 이메일 형식 확인
  final emailRegex = RegExp(r'^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$');
  if (!emailRegex.hasMatch(value)) {
    return '올바른 이메일 형식이 아닙니다';
  }
  return null;
},
```

### 비밀번호 확인 (두 필드 일치 검사)

```dart
// 비밀번호 확인 필드의 validator
validator: (value) {
  if (value != _passwordController.text) {
    return '비밀번호가 일치하지 않습니다';
  }
  return null;
},
```

## 6️⃣ InputDecoration — TextField 꾸미기 [↑](#toc)

`InputDecoration`으로 TextField를 예쁘게 꾸밀 수 있습니다.

```dart
TextFormField(
  decoration: InputDecoration(
    // 라벨 (위로 올라가는 힌트)
    labelText: '이메일',
    labelStyle: const TextStyle(color: Colors.blue),

    // 힌트 (입력 전 흐릿하게 표시)
    hintText: 'example@email.com',

    // 도움말 텍스트 (필드 아래 항상 표시)
    helperText: '가입 시 사용할 이메일을 입력하세요',

    // 앞쪽 아이콘
    prefixIcon: const Icon(Icons.email),

    // 뒤쪽 아이콘 (예: 지우기 버튼)
    suffixIcon: IconButton(
      icon: const Icon(Icons.clear),
      onPressed: () => _emailController.clear(),
    ),

    // 테두리 스타일
    border: OutlineInputBorder(
      borderRadius: BorderRadius.circular(12),
    ),

    // 포커스 시 테두리 색상
    focusedBorder: OutlineInputBorder(
      borderRadius: BorderRadius.circular(12),
      borderSide: const BorderSide(color: Colors.blue, width: 2),
    ),

    // 에러 시 테두리
    errorBorder: OutlineInputBorder(
      borderRadius: BorderRadius.circular(12),
      borderSide: const BorderSide(color: Colors.red),
    ),

    // 배경색
    filled: true,
    fillColor: Colors.grey[50],
  ),
),
```

### 자주 쓰는 테두리 스타일

```dart
// 1. 기본 밑줄 (기본값)
border: const UnderlineInputBorder()

// 2. 사각형 테두리
border: const OutlineInputBorder()

// 3. 둥근 모서리 테두리
border: OutlineInputBorder(
  borderRadius: BorderRadius.circular(30),
)

// 4. 테두리 없음
border: InputBorder.none
```

## 7️⃣ ⭐ **핵심** — dispose() [↑](#toc)

### 메모리 누수 비유

> 수도꼭지를 잠그지 않으면 물이 계속 흘러 낭비됩니다.
> `dispose()`를 호출하지 않으면 메모리가 계속 새어나갑니다.

### dispose() 필수 대상

```dart
class _MyFormState extends State<MyForm> {
  // 아래 객체들은 반드시 dispose() 필요
  final _controller = TextEditingController();
  final _focusNode = FocusNode();
  late AnimationController _animController;

  @override
  void dispose() {
    _controller.dispose();      // TextEditingController
    _focusNode.dispose();       // FocusNode
    _animController.dispose();  // AnimationController
    super.dispose();            // 반드시 마지막에 super.dispose() 호출
  }

  // ...
}
```

### 안전한 dispose 패턴

```dart
@override
void dispose() {
  // 1. 모든 컨트롤러 dispose
  _nameController.dispose();
  _emailController.dispose();
  _passwordController.dispose();

  // 2. 반드시 super.dispose() 마지막에
  super.dispose();
}
```

> AI가 생성한 코드에서 Controller나 FocusNode를 사용하면서 dispose()가 없다면 반드시 추가하세요.

## 8️⃣ 💪 **보너스 챌린지** — 실습: 회원가입 폼 만들기 [↑](#toc)

지금까지 배운 모든 것을 사용해 회원가입 폼을 만들어 봅시다.

### Copilot에게 요청하기

다음 프롬프트로 Copilot에게 기초 코드를 요청하고, **체크리스트로 검증**하세요:

```
Flutter StatefulWidget으로 회원가입 폼을 만들어줘.

필드:
1. 이름 (2자 이상)
2. 이메일 (이메일 형식 검사)
3. 비밀번호 (6자 이상, obscureText)
4. 비밀번호 확인 (비밀번호와 일치 검사)

조건:
- Form + GlobalKey 사용
- TextFormField + validator
- InputDecoration으로 각 필드에 적절한 아이콘 추가
- 제출 버튼: 모든 검사 통과 시 SnackBar로 "가입 완료!" 메시지
- dispose() 필수 포함
- const 생성자 사용
```

### 완성 코드 (AI 생성 후 검증 완료 버전)

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(
      title: '회원가입',
      home: SignupPage(),
    );
  }
}

class SignupPage extends StatefulWidget {
  const SignupPage({super.key});

  @override
  State<SignupPage> createState() => _SignupPageState();
}

class _SignupPageState extends State<SignupPage> {
  final _formKey = GlobalKey<FormState>();
  final _nameController = TextEditingController();
  final _emailController = TextEditingController();
  final _passwordController = TextEditingController();
  final _confirmController = TextEditingController();

  bool _obscurePassword = true;
  bool _obscureConfirm = true;

  @override
  void dispose() {
    _nameController.dispose();
    _emailController.dispose();
    _passwordController.dispose();
    _confirmController.dispose();
    super.dispose();
  }

  void _submit() {
    if (_formKey.currentState!.validate()) {
      ScaffoldMessenger.of(context).showSnackBar(
        const SnackBar(
          content: Text('가입 완료!'),
          backgroundColor: Colors.green,
        ),
      );
    }
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('회원가입')),
      body: SingleChildScrollView(
        padding: const EdgeInsets.all(24),
        child: Form(
          key: _formKey,
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.stretch,
            children: [
              const Text(
                '계정 만들기',
                style: TextStyle(fontSize: 28, fontWeight: FontWeight.bold),
              ),
              const SizedBox(height: 32),

              // 이름 필드
              TextFormField(
                controller: _nameController,
                decoration: const InputDecoration(
                  labelText: '이름',
                  prefixIcon: Icon(Icons.person),
                  border: OutlineInputBorder(),
                ),
                validator: (value) {
                  if (value == null || value.isEmpty) {
                    return '이름을 입력해주세요';
                  }
                  if (value.length < 2) {
                    return '이름은 2자 이상이어야 합니다';
                  }
                  return null;
                },
              ),
              const SizedBox(height: 16),

              // 이메일 필드
              TextFormField(
                controller: _emailController,
                keyboardType: TextInputType.emailAddress,
                decoration: const InputDecoration(
                  labelText: '이메일',
                  prefixIcon: Icon(Icons.email),
                  border: OutlineInputBorder(),
                ),
                validator: (value) {
                  if (value == null || value.isEmpty) {
                    return '이메일을 입력해주세요';
                  }
                  final emailRegex =
                      RegExp(r'^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$');
                  if (!emailRegex.hasMatch(value)) {
                    return '올바른 이메일 형식이 아닙니다';
                  }
                  return null;
                },
              ),
              const SizedBox(height: 16),

              // 비밀번호 필드
              TextFormField(
                controller: _passwordController,
                obscureText: _obscurePassword,
                decoration: InputDecoration(
                  labelText: '비밀번호',
                  prefixIcon: const Icon(Icons.lock),
                  border: const OutlineInputBorder(),
                  suffixIcon: IconButton(
                    icon: Icon(_obscurePassword
                        ? Icons.visibility
                        : Icons.visibility_off),
                    onPressed: () {
                      setState(() {
                        _obscurePassword = !_obscurePassword;
                      });
                    },
                  ),
                ),
                validator: (value) {
                  if (value == null || value.isEmpty) {
                    return '비밀번호를 입력해주세요';
                  }
                  if (value.length < 6) {
                    return '비밀번호는 6자 이상이어야 합니다';
                  }
                  return null;
                },
              ),
              const SizedBox(height: 16),

              // 비밀번호 확인 필드
              TextFormField(
                controller: _confirmController,
                obscureText: _obscureConfirm,
                decoration: InputDecoration(
                  labelText: '비밀번호 확인',
                  prefixIcon: const Icon(Icons.lock_outline),
                  border: const OutlineInputBorder(),
                  suffixIcon: IconButton(
                    icon: Icon(_obscureConfirm
                        ? Icons.visibility
                        : Icons.visibility_off),
                    onPressed: () {
                      setState(() {
                        _obscureConfirm = !_obscureConfirm;
                      });
                    },
                  ),
                ),
                validator: (value) {
                  if (value != _passwordController.text) {
                    return '비밀번호가 일치하지 않습니다';
                  }
                  return null;
                },
              ),
              const SizedBox(height: 32),

              // 가입 버튼
              ElevatedButton(
                onPressed: _submit,
                style: ElevatedButton.styleFrom(
                  padding: const EdgeInsets.symmetric(vertical: 16),
                ),
                child: const Text('가입하기', style: TextStyle(fontSize: 16)),
              ),
            ],
          ),
        ),
      ),
    );
  }
}
```

### AI 코드 검증 체크리스트 적용

```
□ const 사용 여부 확인
  → const InputDecoration, const OutlineInputBorder 적절히 사용됨

□ null safety 확인
  → validator에서 null 체크 완료

□ dispose() 확인
  → 4개 컨트롤러 모두 dispose 완료

□ 이해 확인
  → 각 validator가 무엇을 검사하는지 설명할 수 있는가?
```

## 9️⃣ 정리 [↑](#toc)

### 핵심 위젯/클래스 요약

| | 역할 | 핵심 속성 |
|---|---|---|
| `TextField` | 텍스트 입력 | `controller`, `onChanged`, `keyboardType` |
| `TextFormField` | Form과 통합 입력 | `validator`, `controller` |
| `TextEditingController` | 입력값 제어 | `.text`, `.clear()` |
| `Form` | 여러 입력 묶기 | `key` |
| `GlobalKey<FormState>` | Form 참조 | `.currentState!.validate()` |
| `InputDecoration` | TextField 꾸미기 | `labelText`, `prefixIcon`, `border` |

### 반드시 기억할 3가지

```dart
// 1. Controller는 항상 dispose()
@override
void dispose() {
  _controller.dispose();
  super.dispose();
}

// 2. Form 제출은 validate() 후
if (_formKey.currentState!.validate()) { ... }

// 3. validator 반환값
validator: (value) {
  if (조건) return '에러 메시지';  // 실패
  return null;                     // 통과
}
```
