# Statistics for ML
{: .no_toc }

머신러닝·딥러닝을 **제대로 이해하기 위한** 통계 기초 과정입니다.
{: .fs-6 .fw-300 }

---

## 이 과정은 누구를 위한 것인가요?

- **수학적 지식이 거의 없는** ML/DL 입문자
- "그래서 손실함수가 왜 이렇게 생겼지?"가 궁금한 사람
- 회귀·분류 모델의 **출력을 해석할 수 있어야** 하는 사람
- 비전공자 출신으로 데이터 분석/AI에 뛰어들고 있는 사람

> 💡 본 과정은 **ML 과정의 선수과정**입니다. 수식 유도보다 **개념의 의미**와 **ML/DL에서의 역할**에 집중합니다.

---

## 학습 목표

이 과정을 마치면 다음을 할 수 있습니다.

1. **확률·분포** 개념을 비유로 설명할 수 있다
2. **베이지안 사고**의 직관을 이해하고 스팸 필터 같은 예에 적용할 수 있다
3. **회귀·로지스틱 회귀**의 출력(계수·p-value·R²·시그모이드)을 해석할 수 있다
4. ML/DL의 **손실함수가 통계의 MLE에서 나왔다**는 것을 안다
5. **엔트로피와 Cross-Entropy**가 분류 모델의 학습과 어떻게 연결되는지 안다

## 과정 구성 (10개 모듈)

| 일차 | # | 모듈 | 핵심 개념 | ML 연결 |
|:----:|:-:|------|----------|--------|
| **1일차** | 01 | [확률 기초](/statistics/probability) | 조건부확률, 덧셈/곱셈정리, 이산확률분포 | 모델 출력 = 확률 |
| | 02 | [베이지안 사고](/statistics/bayesian) | 베이즈 정리, 사전·우도·사후 | Naive Bayes, 스팸필터 |
| | 03 | [확률분포와 정규분포](/statistics/distribution) | 이산/연속 분포, 정규분포, 표준화 | 데이터 전처리(StandardScaler) |
| | 04 | [기술통계와 상관](/statistics/descriptive) | 평균·분산·표준편차, 공분산·상관계수 | EDA, 피처 선택 |
| | 05 | [회귀분석 기초](/statistics/regression) | 선형회귀, 최소제곱법, R² | ML의 핵심 모델, MSE 손실 |
| **2일차** | 06 | [로지스틱 회귀](/statistics/logistic) | 시그모이드, 결정경계 | **단일 뉴런 = 로지스틱 회귀** |
| | 07 | [MLE와 손실함수](/statistics/mle) | 우도, 최대우도추정 | **DL 손실함수의 근간** |
| | 08 | [가설검정과 정규화](/statistics/hypothesis) | p-value, t검정, 다중공선성 → Ridge/Lasso | 모델 평가, 피처 선택 |
| | 09 | [ANOVA와 카이제곱](/statistics/anova) | ANOVA, 카이제곱 독립성 | 피처 선택 (SelectKBest) |
| | 10 | [통계에서 ML로](/statistics/bridge) | 편향-분산, 교차검증, 엔트로피·Cross-Entropy | **통계 → ML 파이프라인 조감** |

