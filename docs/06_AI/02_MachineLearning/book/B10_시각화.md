## 1️⃣ Matplotlib 구조 [↑](#toc)

> **figure (도화지) → axes (그래프 영역)** 두 객체로 이루어짐.

### 객체지향 vs pyplot

```python
# pyplot 스타일 (간단·즉흥)
plt.plot(x, y)
plt.title('title')

# 객체지향 스타일 (권장)
fig, ax = plt.subplots(figsize=(8, 5))
ax.plot(x, y)
ax.set_title('title')
ax.set_xlabel('x')
ax.set_ylabel('y')
```

> 💡 **객체지향이 표준**. 서브플롯·재사용에 유리.

## 2️⃣ 기본 차트 6종 [↑](#toc)

| 차트 | API | 언제 쓰나 |
|---|---|---|
| Line | `ax.plot(x, y)` | 시간 추이 |
| Bar | `ax.bar(x, height)` | 범주 비교 |
| Histogram | `ax.hist(data, bins=20)` | 분포 |
| Scatter | `ax.scatter(x, y)` | 두 변수 관계 |
| Box | `ax.boxplot(data)` | 분포 + 이상치 |
| Pie | `ax.pie(sizes, labels=labels)` | 비율 (비추, bar로 대체 권장) |

```python
fig, axes = plt.subplots(2, 3, figsize=(15, 8))
axes[0, 0].plot([1,2,3,4], [1,4,9,16])      ; axes[0, 0].set_title('Line')
axes[0, 1].bar(['A','B','C'], [3,7,2])      ; axes[0, 1].set_title('Bar')
axes[0, 2].hist(np.random.randn(1000))      ; axes[0, 2].set_title('Hist')
axes[1, 0].scatter(np.random.randn(100), np.random.randn(100)); axes[1, 0].set_title('Scatter')
axes[1, 1].boxplot(np.random.randn(100))    ; axes[1, 1].set_title('Box')
axes[1, 2].pie([3,2,1], labels=['A','B','C']); axes[1, 2].set_title('Pie')
fig.tight_layout()
```

## 3️⃣ 서브플롯 [↑](#toc)

```python
fig, axes = plt.subplots(nrows=2, ncols=3, figsize=(15, 8),
                         sharex=True, sharey=True)
```

| 옵션 | 의미 |
|---|---|
| `sharex`, `sharey` | 축 공유 |
| `figsize=(가로, 세로)` | 인치 단위 |
| `gridspec_kw={'hspace':0.4, 'wspace':0.3}` | 간격 |

## 4️⃣ Seaborn 통계 시각화 [↑](#toc)

> Matplotlib 위에 얹은 고수준 API. **DataFrame 친화적**.

```python
import seaborn as sns
sns.set_theme(style='whitegrid')

sns.countplot(data=df, x='Sex', hue='Survived')
sns.histplot(data=df, x='Age', hue='Survived', kde=True, bins=30)
sns.boxplot(data=df, x='Pclass', y='Fare')
sns.violinplot(data=df, x='Pclass', y='Age')
sns.barplot(data=df, x='Pclass', y='Survived')   # 평균 + 신뢰구간 자동
```

> 💡 **hue 인자**로 한 축 추가 → 두 변수 동시 시각화. EDA의 핵심.

## 5️⃣ Heatmap & PairPlot [↑](#toc)

### 상관관계 Heatmap

```python
corr = df.select_dtypes('number').corr()
sns.heatmap(corr, annot=True, fmt='.2f', cmap='coolwarm', center=0)
```

> 💡 **EDA 표준 1번 차트**. 강한 상관 (|0.7| 이상) 피처 식별.

### PairPlot

```python
sns.pairplot(df, hue='Survived', vars=['Age', 'Fare', 'Pclass'])
```

모든 수치형 변수 쌍의 산점도 + 대각선에 분포.

> ⚠️ **수치 변수가 많으면 느리고 화면이 가득**. 5개 이내로 축소.

## 7️⃣ 정리와 체크리스트 [↑](#toc)

### 한 문장 요약

> **Matplotlib 객체지향 + Seaborn 고수준 API.** 한 줄로 EDA 차트 그려보고, 발표·보고서엔 색·라벨·축을 가다듬는다.

### 자기 점검

- [ ] figure 와 axes 차이를 안다
- [ ] 객체지향 스타일로 차트 그리기
- [ ] 6종 기본 차트의 적절한 사용처
- [ ] subplots 의 sharex/sharey 옵션
- [ ] Seaborn 의 `hue` 활용
- [ ] heatmap 의 `annot`, `fmt`, `cmap` 옵션
- [ ] Colab 한글 폰트 설정 코드

### 흔한 오해

| 오해 | 사실 |
|---|---|
| "plt.plot 한 줄이면 끝" | 발표용엔 객체지향으로 라벨·제목·범례 가다듬기 필수 |
| "pie 차트가 비율 보여주기 좋음" | 인간 눈이 각도 비교에 약함 — `barh` 권장 |
| "heatmap 의 색은 자동" | `vmin`, `vmax`, `center` 안 맞추면 왜곡 |
| "Colab 한글 폰트 한 번 설치하면 평생 OK" | 새 세션마다 다시 설치 + rcParams 설정 필요 |
