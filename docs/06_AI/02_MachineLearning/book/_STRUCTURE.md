## 0. 다중 페르소나 검토 요약

| 페르소나 | 핵심 의견 |
|---|---|
| **현업 강사** | Colab 옆에 두고 진행하므로 슬라이드/노트는 개념·도식·결과해석 중심. 코드는 미니샘플만, 풀코드는 Colab. |
| **학생** | 책+Colab+노트 3개 동시 보기 부담 → 헤더에 "📖 책 X장 / 💻 X.X_colab.ipynb" 앵커 필수. |
| **콘텐츠 작가** | PDF 본문(OCR)로 표현·예시 보강. 책 고유 비유 살리되 M-시리즈의 직관 우선 흐름 유지. |
| **교육공학자** | 단위 표준: 학습목표 → 도입 → 본론 → 실습 → 확장 → 정리/체크리스트. |
| **데블스 애드버킷** | M11(KNN/SVM)은 책에 거의 없음. 부록으로 두어야 충돌 없음. |
