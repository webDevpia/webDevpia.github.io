| 시간 | 내용 | 방법 |

## 예상 트러블슈팅

| 문제 | 원인 | 해결 |
|------|------|------|
| "인스턴스가 안 보여요" | 리전이 서울이 아님 | 우측 상단 리전 → ap-northeast-2 |
| "연결 시간 초과" | 보안 그룹 SSH(22) 미오픈 | 보안 그룹 → 인바운드 규칙 편집 |
| "Nginx 페이지 안 보여요" | HTTP(80) 미오픈 또는 https 접속 | 보안 그룹 확인 + http:// 확인 |
| "MySQL 비밀번호 오류" | 비밀번호 정책 미충족 | `Cloud2026!` (대소문자+숫자+특수문자) |
| "Permission denied" | sudo 빠뜨림 | 명령어 앞에 `sudo` 추가 |
| "카드가 없어요" | 계정 생성 불가 | 강사 화면 공유로 따라보기 |
